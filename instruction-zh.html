安装

在任何基于 Chromium 的浏览器（Chrome、Edge、Brave、Arc、Opera 等）上安装实时翻译扩展。

1 打开实时翻译网站

前往 https://translate.ezitech.io 在您的浏览器中。

2 单击“安装”按钮

单击 添加到 Chrome 按钮。这将下载扩展文件。

3 在浏览器中启用开发者模式

打开浏览器的扩展程序管理页面：

  • 镀铬/勇敢/电弧： 类型 在地址栏中输入 chrome://extensions ，然后按 Enter。
  • 边缘： 类型 在地址栏中输入 edge://extensions ，然后按 Enter。

切换 开发者模式 页面右上角的 ON。

4 安装扩展文件

拖放下载的 .zip 文件到扩展页面。

完成！

您现在应该在浏览器工具栏中看到实时翻译器图标。扩展已安装并准备好进行配置。

获取 Soniox API 密钥

Soniox 提供实时语音识别和翻译引擎。您需要 Soniox API 密钥才能使用实时翻译器。 必填

重要

Soniox API 密钥是 必填 使扩展能够正常工作。如果没有它，扩展程序就无法转录或翻译任何音频。

1 前往 Soniox 网站

打开 https://console.soniox.com/signup 在您的浏览器中。使用您的电子邮件或 Google 登录创建帐户。

2 获取您的 API 密钥

登录后，导航至 Soniox 仪表板的 API 部分。您也可以直接前往 Soniox 入门指南 了解分步说明。

复制您的 API 密钥 - 您将在后续步骤中需要它。

3 添加计费（API 使用所需）

向您的 Soniox 帐户添加付款方式。您可以使用信用卡、PayPal 或其他付款方式。 Soniox API 是即用即付的 — 您只需为使用的内容付费。

注意

您需要向您的帐户存入至少 1 美元才能使用 API。如果您愿意，您可以存入更多资金。

1 打开实时翻译设置

您可以通过以下任一方式打开设置：

  • 单击 齿轮图标 (⚙) 位于翻译器面板的右上角。
  • 右键单击工具栏中的实时翻译图标 → “选项”.
  • 转到浏览器的扩展页面→找到实时翻译→ “详细信息” → “扩展选项”.

在设置页面中，单击 “索尼克斯” 在左侧边栏中。将您的 Soniox API 密钥粘贴到输入字段中。密钥会自动保存。

点击 “OpenAI” 在左侧边栏中。将您的 OpenAI API 密钥粘贴到输入字段中。密钥会自动保存。

您的 API 密钥已存储 仅在您的设备本地 使用 Chrome 的安全扩展存储。它们永远不会发送到 Live Translator 的服务器 - 它们仅用于直接与 Soniox 和 OpenAI 通信。

开始翻译会话

了解如何在会议或视频中开始翻译。

1 打开支持的网站

前往任意 支持的平台 （例如 Google Meet、Zoom、YouTube 等）并加入会议或播放视频。

2 打开实时翻译面板

单击 实时翻译图标 在浏览器工具栏中。翻译面板将出现在页面上。您还可以单击 浮动图标 页面上显示的 。

3 选择您的语言

使用面板顶部的两种语言下拉菜单：

  • “我的语言” ——语言 你 说话（例如越南语）。
  • “他们的语言” ——语言 其他 说出您想要翻译的内容（例如英语）。
4 按开始按钮

单击 开始 按钮（播放图标）。

您的浏览器可能会请求捕获音频的权限 - 单击 “允许” and select the tab you want to capture.

5 See translations in real time

当人们说话时，你会看到 原文 和 翻译文本 出现在成绩单区域。 Lines are automatically grouped by speaker.

自动滚动

记录会自动滚动以显示最新文本。如果您向上滚动以阅读较早的文本，自动滚动会暂停。单击 “滚动到最新” 按钮跳回实时文本。

控件：暂停、恢复、结束

了解翻译会话控制。

按钮它的作用
开始 (▶)Begins a new translation session.清除任何以前的记录。
暂停 (⏸)Temporarily stops translation.成绩单被保留。音频捕获继续，但文本暂停。
简历 (▶)暂停后继续翻译。新文本出现在先前内容之后。
结束 (⏹)完全停止会话。文字记录将保留在屏幕上，直到您开始新的会话。结束音频捕获，关闭麦克风和旁白。

更改语言

您可以随时更改翻译语言。

1 单击语言下拉列表

单击 “我的语言” 或 面板顶部的 “他们的语言” 下拉菜单。

2 选择新语言

列表支持 60 多种语言 按字母顺序排序。滚动或键入以搜索您的语言。单击以选择它。

3 翻译自动重新开始

如果会话处于活动状态，更改语言将自动使用新语言对重新启动翻译。您的语言选择将被保存以供下次使用。

打开你的麦克风

启用您的麦克风，以便您自己的演讲也会与会议/视频音频一起翻译。

1 单击麦克风按钮

在控制栏中，单击 麦克风图标。当麦克风打开时，该图标将变为活动状态（突出显示）。

2 允许麦克风访问

如果这是第一次，您的浏览器将请求麦克风权限。点击 “允许”.

提示

麦克风正在 默认关闭 当您打开面板时。仅当您希望转录和翻译您自己的演讲时才打开它。当您按下 结束，麦克风自动关闭。

更改字体大小

调整文字大小以方便阅读。

1 单击字体大小按钮

在控制栏中，单击 字体大小按钮 （Aa 图标）。将出现一个具有三种尺寸的菜单：

  • 小 ——紧凑的视图，适合更多文本
  • 中 —默认平衡尺寸
  • 大 ——更容易从远处阅读

单击您的首选尺寸。更改会立即应用并保存以供下次使用。

发言者姓名

为发言者指定真实姓名，以便您可以轻松识别谁在文字记录中说了些什么。 免费

1 打开“更多”菜单

在控制栏中，单击 “更多” 按钮（⋯图标）。然后点击 “扬声器设置” 按钮。

2 打开“扬声器设置”对话框

扬声器设置对话框将打开。

3 分配名称

您将看到检测到的扬声器列表（扬声器 1、扬声器 2 等）。在每个名字旁边输入真实姓名（例如“John”、“Sarah”）。点击 “添加行” 如果您需要更多。

点击 “保存”。所有转录行——过去的和未来的——现在都将使用您指定的名称。

旁白（文本转语音）

让翻译大声朗读给您听，这样您就可以听而不是读。这就是“讲述他们的声音”功能。 临

1 打开旁白设置

在控制栏中，单击 旁白按钮 （扬声器图标）。 “旁白设置”对话框打开。

2 选择 TTS 提供商和语音

选择您的语音提供商：

  • 谷歌翻译 （免费）— 无需任何 API 密钥即可工作。 60 多种语言的优质语音。
  • OpenAI (高级) — 更高质量、更自然的声音。需要 OpenAI API 密钥。

选择一个 声音 从下拉菜单中选择 ，然后单击 “测试” 预览它的声音。

调整 语速 和 音量 如果需要的话。

3 开始旁白

点击 “讲述他们的声音” 对话框中的 按钮。现在，每当出现翻译时，都会大声朗读给您听。

要停止，请再次单击旁白按钮，然后单击 “停止”.

讲述我的声音（口译模式）

用您的语言讲话并向其他会议参与者播放翻译后的声音 - 就像拥有私人口译员一样。 临

要求

此功能适用于 Google Meet, 缩放（网页）, Microsoft Teams（网络），以及 WhatsApp 网页版。你需要你的 麦克风已启用 和翻译会议 正在运行 控制栏中的 .

1 打开旁白设置

单击 旁白按钮 打开“旁白设置”对话框。

2 配置“讲述我的声音”

在对话框中，找到 “讲述我的声音” 部分。选择您喜欢的语音和提供商（Google 或 OpenAI）。您可以设置一个 不同的声音 为“我的声音”和“他们的声音”旁白。

点击 “测试” 预览您选择的声音。

3 开始“讲述我的声音”

点击 “开始讲述我的声音”。当你说话时，你的话是：

  1. 以您的语言转录
  2. 翻译成其他语言
  3. 在会议中大声讲话，以便其他人听到您翻译的声音

播放翻译后的音频时，您的麦克风会自动静音，以防止出现回声。

4 停止旁白

单击旁白按钮，然后 “停止”，或按 结束 停止整个会话（这也会停止旁白）。

注意

你只能运行 一次一种旁白模式 ——“讲述我的声音”或“讲述他们的声音”，不能同时两者。工具栏图标会发生变化以显示哪种模式处于活动状态。

通过聊天发送翻译

在会议聊天中自动将翻译后的演讲作为短信发送。 临

1 启用“通过聊天发送”

在控制栏中查找 聊天发送按钮 （麦克风图标旁边）。单击它即可打开 - 按钮会转动 绿色 激活时。

2 说话并自动发送

打开麦克风并运行翻译会话后，当您讲话时，您的翻译文本会自动插入到会议的聊天输入中并发送。

这适用于 Google Meet, 缩放（网页），以及 Microsoft Teams（网络）.

提示

当“讲述我的声音”不适用于您的平台时，或者如果您想要同时进行语音旁白和文本聊天翻译，这非常有用。

字幕模式

切换到最小的电影风格字幕栏，仅在屏幕底部显示最新的翻译。 临

1 启用字幕模式

在翻译器面板的标题栏中，单击 字幕图标 （隐藏式字幕图标）。

整个面板将被屏幕底部的细长字幕栏取代。

2 自定义字幕栏

单击 调整图标 字幕栏上的 （设置档位）调整：

  • 不透明度 — 滑动以使栏或多或少透明。
  • 字体大小 — 选择小、中或大文本。

您还可以 拖动 移动字幕栏，然后 调整大小 通过拖动边缘来实现。

3 退出字幕模式

单击 关闭按钮 字幕栏上的 (✕) 可返回完整翻译面板。

内嵌人工智能助手

直接从记录中获取人工智能驱动的解释并根据对话上下文生成答案。 临

要求

内联人工智能助手需要一个 OpenAI API 密钥 在设置中配置。

解释一行

1 将鼠标悬停在转录行上

当您将鼠标悬停在记录中的任何行上时， 人工智能图标 出现 (🤖)。

2 点击AI图标并选择“解释”

AI 助手在单独的窗口中打开。它将使用完整的文字记录上下文和选定的行来为您提供详细的解释。

生成答案

1 点击AI图标并选择“生成答案”

人工智能助手将根据对话内容生成建议答案。它甚至可以使用网络搜索来提供更准确的答案。

打开助手面板

1 单击控制栏中的“助手”按钮

您也可以直接从桌面打开AI助手面板 助理按钮 控制栏中的 （🤖图标）。这将打开一个可拖动的聊天窗口，您可以在其中键入任何问题。助理可以访问完整的文字记录作为上下文。

上下文设置

提供有关对话主题的上下文以提高翻译准确性。 免费

1 打开上下文设置 控制栏中的

单击 “更多” 按钮(⋯)，然后单击 “上下文”.

2 填写上下文字段

您可以提供：

  • 域/主题 ——对话的内容（例如“医学会议”、“软件工程站立”）。
  • 关键术语 ——可能难以识别的技术术语、专有名词或行话（例如“Kubernetes、API 网关、微服务”）。
  • 演讲者信息 — 参与者的姓名和角色（例如，“Smith 博士 — 心脏病专家，主讲”）。

点击 “申请” 将此上下文发送到当前会话的翻译引擎。

3 保存预设（可选）

如果您有类似主题的定期会议，请单击 “保存预设” 保存当前上下文。下次，只需加载预设即可，而不必再次输入所有内容。

您还可以 导入/导出 预设为 JSON 文件以共享或备份它们。

故障排除

常见问题以及如何解决这些问题。

“没有 Soniox API 密钥”消息

! 解决方案

您尚未输入您的 Soniox API 密钥。前往 输入您的 API 密钥 进行设置。

没有捕获任何音频

! 解决方案
  • 确保您点击了 “允许” 当浏览器请求音频捕获权限时。
  • 确保您选择了 正确的选项卡 出现提示时进行分享。
  • 尝试刷新页面并再次打开翻译器。
  • 检查会议/视频是否确实有音频播放。

翻译似乎不准确

! 解决方案
  • 确保 “他们的语言” 下拉菜单设置为所讲的正确语言。
  • 使用 上下文设置 提供主题信息和关键术语。
  • 确保音频质量清晰——背景噪音会影响准确性。

“讲述我的声音”-其他人听不到翻译

! 解决方案
  • 确认您正在使用 支持的平台 （Google Meet、Zoom Web、Teams Web、WhatsApp Web）。
  • 确保翻译会话已启动 正在运行 控制栏中的 和你的 麦克风已打开.
  • 检查您是否有 专业许可证 已激活。
  • 尝试刷新会议页面并重新开始。

OpenAI 功能不起作用

! 解决方案
  • 验证您的 OpenAI API 密钥 已正确输入 设置.
  • 确保您的 OpenAI 帐户已 计费设置 有足够的学分。
  • 如果不需要 OpenAI 功能，请使用 谷歌翻译语音 用于旁白（免费，无需 API 密钥）。

扩展程序未出现在网站上

! 解决方案
  • 实时翻译仅适用于 支持的平台。它不会出现在不受支持的网站上。
  • 如果该网站受支持但扩展名不显示，请尝试 正在刷新页面.
  • 确保扩展名是 已启用 在浏览器的扩展程序管理页面中。
