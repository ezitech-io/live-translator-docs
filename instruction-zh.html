安装

在任何基于 Chromium 内核的浏览器（Chrome、Edge、Brave、Arc、Opera 等）上安装 Live Translator 扩展。

1 从 Chrome Web Store 安装

打开官方页面：Chrome Web Store - Live Translator，然后点击 Add to Chrome 并在弹窗中确认 Add extension。

完成！

现在您应该能在浏览器工具栏中看到 Live Translator 的图标了。扩展已安装完毕，可以开始配置。

功能概览

快速了解主界面各部分的位置和功能。

Live Translator 面板主要有以下几个区域：

  • 标题栏 — 设置按钮、字幕模式开关、关闭按钮。如果有新版本可用，这里会显示一个更新提示。
  • 语言选择器 — 两个下拉菜单："我的语言"（您说的语言）和"对方语言"（您想翻译的来源语言）。
  • 翻译区 — 翻译后的文字会实时显示在这里，按发言者分组。
  • 控制栏 — 开始/停止/暂停按钮、麦克风开关、朗读设置、AI 助手、字体大小、下载等更多选项。
提示

您可以通过拖动标题栏来移动面板到屏幕上的任意位置，也可以通过拖动边缘来调整面板大小。这样在开会或看视频时就不会挡住重要内容了。

获取 Soniox API 密钥

Soniox 提供实时语音识别和翻译引擎。使用 Live Translator 必须要有 Soniox API 密钥。 必需

重要

Soniox API 密钥是必需的，没有它扩展就无法识别和翻译任何音频。

1 打开 Soniox 网站

在浏览器中打开 https://console.soniox.com/signup，使用您的邮箱或 Google 账号注册。

2 获取您的 API 密钥

登录后，进入 Soniox 控制台的 API 部分。您也可以直接查看 Soniox 入门指南，里面有详细的步骤说明。

复制您的 API 密钥——后面的步骤中会用到。

3 添加付款方式（使用 API 必需）

在您的 Soniox 账户中添加付款方式。可以使用信用卡、PayPal 或其他支付方式。Soniox API 按实际用量计费——用多少付多少。

注意

您需要至少充值 1 美元才能使用 API。如果需要，也可以充值更多。

获取 OpenAI API 密钥（可选）

OpenAI API 密钥是可选的——只有当您想使用 OpenAI 语音进行朗读或使用 AI 智能助手功能时才需要。 Pro

1 打开 OpenAI 平台

在浏览器中打开 https://platform.openai.com/api-keys。

2 注册或登录账号

如果还没有 OpenAI 账号，点击"Sign Up"注册一个。如果已有账号，直接登录即可。

3 创建新的 API 密钥

点击"+ Create new secret key"，输入一个名称（比如"Live Translator"），然后点击"Create secret key"。

请立刻复制并保存密钥——OpenAI 只会显示一次。

4 添加付款方式（使用 API 必需）

前往账单设置，添加付款方式。OpenAI API 也是按用量计费——用多少付多少。

注意

如果您只想使用语音朗读（文字转语音）功能，可以免费使用 Google 翻译的语音。只有当您需要 OpenAI 的高品质语音或 AI 智能助手功能时，才需要 OpenAI 密钥。

输入您的 API 密钥

拿到 API 密钥后，将它们填入 Live Translator 的设置页面。

1 打开 Live Translator 设置

您可以通过以下任意方式打开设置：

  • 点击翻译面板右上角的齿轮图标（⚙）。
  • 右键点击工具栏中的 Live Translator 图标 → 选择"选项"。
  • 在浏览器的扩展管理页面 → 找到 Live Translator → 点击"详情" → "扩展程序选项"。
2 输入 Soniox API 密钥

在设置页面中，点击左侧边栏的"Soniox"，将您的 Soniox API 密钥粘贴到输入框中。密钥会自动保存。

3 输入 OpenAI API 密钥（可选）

点击左侧边栏的"OpenAI"，将您的 OpenAI API 密钥粘贴到输入框中。密钥会自动保存。

安全说明

您的 API 密钥仅保存在您自己的设备上，使用的是 Chrome 的安全扩展存储。密钥不会发送到 Live Translator 的服务器——它们只用于直接与 Soniox 和 OpenAI 通信。

开始翻译

了解如何在会议或视频中启动翻译。

1 打开一个支持的网站

进入任意一个支持的平台（例如 Google Meet、Zoom、YouTube 等），加入会议或播放视频。

2 打开 Live Translator 面板

点击浏览器工具栏中的 Live Translator 图标，翻译面板就会出现在页面上。您也可以点击页面上出现的悬浮图标来打开面板。

3 选择语言

使用面板顶部的两个语言下拉菜单：

  • "我的语言" — 就是您说的语言（例如中文）。
  • "对方语言" — 对方说的语言，也就是您想翻译的语言（例如英语）。
4 点击"开始"按钮

点击控制栏底部的开始按钮（播放图标）。

浏览器可能会请求捕获音频的权限——点击"允许"，然后选择您想要翻译的标签页。

5 实时查看翻译

当有人说话时，翻译区会同时显示原文和译文。内容会自动按发言者分组显示。

自动滚动

翻译记录会自动滚动到最新内容。如果您向上滚动查看之前的内容，自动滚动会暂停。点击出现的"滚动到最新"按钮即可跳回最新内容。

控制：暂停、继续、结束

了解翻译过程中的各个控制按钮。

按钮作用
开始 (▶)开始一个新的翻译会话，之前的记录会被清除。
暂停 (⏸)临时暂停翻译。已有的记录会保留，音频仍在捕获但暂停显示文字。
继续 (▶)暂停后继续翻译，新的文字会接在之前内容后面。
结束 (⏹)完全停止翻译。记录保留在屏幕上直到您开始新的会话。同时关闭音频捕获、麦克风和朗读功能。

切换语言

您可以随时更改翻译语言。

1 点击语言下拉菜单

点击面板顶部的"我的语言"或"对方语言"下拉菜单。

2 选择新的语言

列表中有60 多种语言，按字母顺序排列。您可以滚动或输入关键词来搜索，然后点击选择。

3 翻译自动重启

如果当前正在翻译，切换语言后会自动以新的语言组合重新开始翻译。您选择的语言会被保存，下次打开时无需重新选择。

开启麦克风

开启麦克风后，您自己说的话也会和会议/视频音频一起被翻译。

1 点击麦克风按钮

在控制栏中，点击麦克风图标。麦克风开启后，图标会变为高亮状态。

2 允许麦克风权限

如果是第一次使用，浏览器会弹出麦克风权限请求，点击"允许"即可。

提示

打开面板时，麦克风默认是关闭的。只有当您希望自己的声音也被识别和翻译时才需要打开它。按下结束按钮时，麦克风会自动关闭。

调整字体大小

调整翻译文字的大小，让您阅读更舒适。

1 点击字体大小按钮

在控制栏中，点击字体大小按钮（Aa 图标），会出现三种大小可选：

  • 小 — 紧凑显示，可以看到更多内容
  • 中 — 默认大小，大小适中
  • 大 — 字更大，远一点也能看清

点击您想要的大小即可。更改会立即生效，并自动保存。

发言者名称

给发言者设置真实姓名，方便您在翻译记录中快速分辨是谁在说话。 免费

1 打开"更多"菜单

在控制栏中，点击"更多"按钮（⋯ 图标），然后点击"发言者设置"按钮。

2 打开"发言者设置"对话框

发言者设置对话框会弹出。

3 设置名称

您会看到系统自动识别出的发言者列表（发言者 1、发言者 2 等）。在每个发言者旁边输入真实姓名（例如"张三"、"李四"）。如果需要添加更多，点击"添加行"。

点击"保存"后，所有翻译记录（包括之前的和之后的）都会显示您设置的名称。

语音朗读（文字转语音）

让翻译内容自动朗读出来，您不用看文字，听就行。这就是"朗读对方语音"功能。 Pro

1 打开朗读设置

在控制栏中，点击朗读按钮（喇叭图标），朗读设置对话框会弹出。

2 选择语音引擎和声音

选择您想使用的语音引擎：

  • Google 翻译（免费）— 无需 API 密钥即可使用，支持 60 多种语言，音质不错。
  • OpenAI（高级）— 音质更好，声音更自然。需要 OpenAI API 密钥。

从下拉菜单中选择一个声音，然后点击"测试"试听效果。

您还可以根据需要调整语速和音量。

3 开始朗读

点击对话框中的"朗读对方语音"按钮。之后每当有新的翻译出现，系统就会自动朗读给您听。

如需停止，再次点击朗读按钮，然后点击"停止"。

YouTube 字幕跟随朗读（LS-36）

使用 YouTube 的翻译字幕作为朗读依据，让扩展按您在 YouTube 里已选择的语言进行朗读。在 YouTube 场景下，这项功能可将 Soniox 成本最多降低 99.9%。 Pro

操作顺序很重要

要正常使用此功能，必须先在 YouTube 里选好字幕语言，再在扩展中开始朗读。

1 先在 YouTube 设置翻译字幕

在 YouTube 播放器中打开字幕，然后选择您希望朗读的翻译字幕语言（例如：越南语）。

2 在朗读设置中打开自动识别

打开扩展的朗读设置，开启 Auto detect translated caption。

3 开始朗读

点击 "朗读对方语音"。系统会按照您在 YouTube 选择的翻译字幕语言进行朗读。

示例

如果您想听越南语朗读，请先把 YouTube 字幕切换到越南语，然后再开启 Auto detect translated caption 并点击 "朗读对方语音"。

朗读我的语音（口译模式）

用您的语言说话，系统会将翻译后的语音播放给会议中的其他人听——就像有一位随身翻译。 Pro

使用条件

此功能适用于 Google Meet、Zoom (Web)、Microsoft Teams (Web) 和 WhatsApp Web。使用前需要打开麦克风，并且翻译会话已经开始运行。

1 打开朗读设置

点击控制栏中的朗读按钮，打开朗读设置对话框。

2 配置"朗读我的语音"

在对话框中找到"朗读我的语音"部分。选择您喜欢的声音和语音引擎（Google 或 OpenAI）。您可以为"我的语音"和"对方语音"分别设置不同的声音。

点击"测试"可以试听所选声音的效果。

3 开始"朗读我的语音"

点击"开始朗读我的语音"。当您说话时，系统会：

  1. 将您的语音转录为文字
  2. 翻译成对方的语言
  3. 在会议中播放翻译后的语音，让对方听到

播放翻译语音时，您的麦克风会自动静音，以避免产生回声。

4 停止朗读

点击朗读按钮再点"停止"，或者直接按结束按钮来停止整个翻译会话（朗读也会随之停止）。

注意

同一时间只能使用一种朗读模式——要么"朗读我的语音"，要么"朗读对方语音"，不能同时使用。工具栏图标会显示当前正在使用哪种模式。

通过聊天发送翻译

将您翻译后的语音自动以文字消息发送到会议的聊天框中。 Pro

1 开启"通过聊天发送"

在控制栏中找到聊天发送按钮（在麦克风图标旁边），点击开启——按钮变为绿色表示已开启。

2 说话即自动发送

在麦克风开启并且翻译正在运行的情况下，您说话后，翻译文字会自动填入会议的聊天输入框并发送。

此功能支持 Google Meet、Zoom (Web) 和 Microsoft Teams (Web)。

提示

当您所在的平台不支持"朗读我的语音"功能时，或者您想同时使用语音朗读和文字聊天翻译时，这个功能特别实用。

字幕模式

切换为简洁的电影式字幕条，在屏幕底部只显示最新的翻译内容。 Pro

1 开启字幕模式

在翻译面板的标题栏中，点击字幕图标（CC 图标）。

完整面板会变成屏幕底部的一个细长字幕条。

2 自定义字幕条

点击字幕条上的设置图标（齿轮），可以调整：

  • 透明度 — 拖动滑块可以让字幕条变得更透明或更不透明。
  • 字体大小 — 选择小号、中号或大号文字。

您还可以拖动字幕条来移动位置，拖动边缘来调整大小。

3 退出字幕模式

点击字幕条上的关闭按钮（✕）即可返回完整的翻译面板。

AI 智能助手

根据对话内容，直接在翻译记录中获取 AI 驱动的解释和自动生成的回答。 Pro

使用条件

AI 智能助手需要在设置中配置 OpenAI API 密钥。

解释某句话

1 将鼠标悬停在翻译记录的某一行上

当您把鼠标移到翻译记录中的任意一行上时，会出现一个 AI 图标（🤖）。

2 点击 AI 图标，选择"解释"

AI 助手会在新窗口中打开，它会结合完整的对话记录和您选中的那句话，给出详细的解释。

生成回答

1 点击 AI 图标，选择"生成回答"

AI 助手会根据对话内容自动生成一个建议回答。它还可以通过网络搜索来提供更准确的答案。

打开助手面板

1 点击控制栏中的"助手"按钮

您也可以通过控制栏中的助手按钮（🤖 图标）直接打开 AI 助手面板。这会弹出一个可拖动的聊天窗口，您可以在里面输入任何问题。助手能够读取完整的对话记录作为参考。

场景设置

提供对话主题的背景信息，帮助提升翻译准确度。 免费

1 打开场景设置

点击控制栏中的"更多"按钮（⋯），然后点击"场景"。

2 填写场景信息

您可以填写：

  • 领域/主题 — 对话内容是关于什么的（例如"医学会议"、"软件工程每日站会"）。
  • 关键术语 — 可能难以识别的专业术语、专有名词或行话（例如"Kubernetes、API 网关、微服务"）。
  • 发言者信息 — 参与者的姓名和角色（例如"Smith 医生——心脏科专家，主讲人"）。

点击"应用"，将这些信息发送给翻译引擎，用于当前翻译会话。

3 保存预设（可选）

如果您经常参加类似主题的会议，可以点击"保存预设"来保存当前的场景信息。下次直接加载预设，不用重新输入。

您还可以将预设导入/导出为 JSON 文件，方便分享或备份。

YouTube 自动填充

在 YouTube 上使用 Live Translator 时，场景信息可以自动填充视频的标题和描述，帮助翻译引擎更好地理解视频内容。

下载记录

将翻译记录保存为文件，方便日后查看或在其他工具中使用。 Pro

1 点击下载按钮

翻译结束后（或翻译过程中），点击控制栏中的下载按钮（↓ 图标），下载对话框会弹出。

2 选择格式

选择您需要的格式：

格式适合场景
TXT纯文本文件，包含时间戳和发言者姓名，阅读方便。
JSON结构化数据格式，适合用其他工具或脚本进一步处理。
SRT（原文）原始语言的字幕文件，可导入视频编辑软件使用。
SRT（译文）翻译后的字幕文件，可导入视频编辑软件使用。
3 下载文件

点击对应格式的下载按钮即可。文件会在浏览器中生成，并保存到您的默认下载文件夹中。

隐私

记录文件完全在您的浏览器中生成，下载过程中不会将任何数据发送到服务器。

自动检查更新

1 查看更新提示

当有新版本可用时，翻译面板的标题栏中会出现一个"有更新可用"的提示标记。

常见问题

遇到问题了？来这里看看怎么解决。

提示"没有 Soniox API 密钥"

! 解决方法

您还没有输入 Soniox API 密钥。请前往输入 API 密钥部分进行设置。

没有捕获到任何音频

! 解决方法
  • 确认在浏览器弹出音频捕获权限时点击了"允许"。
  • 确认弹出提示时选择了正确的标签页进行共享。
  • 尝试刷新页面并重新打开翻译面板。
  • 检查会议/视频是否确实有声音在播放。

翻译不够准确

! 解决方法
  • 确认"对方语言"下拉菜单选择的是对方实际在说的语言。
  • 使用场景设置提供对话主题和关键术语信息。
  • 确保音频清晰——背景噪音会影响翻译准确度。

"朗读我的语音"——对方听不到翻译

! 解决方法
  • 确认您使用的是支持的平台（Google Meet、Zoom Web、Teams Web、WhatsApp Web）。
  • 确认翻译会话已开始运行，并且麦克风已开启。
  • 检查是否已激活 Pro 许可证。
  • 尝试刷新会议页面并重新开始。

OpenAI 相关功能无法使用

! 解决方法
  • 检查OpenAI API 密钥是否在设置中正确输入。
  • 确认您的 OpenAI 账户已设置好付款方式且余额充足。
  • 如果不需要 OpenAI 功能，可以使用Google 翻译语音进行朗读（免费，无需 API 密钥）。

扩展没有出现在网站上

! 解决方法
  • Live Translator 仅在支持的平台上可用，不支持的网站上不会显示。
  • 如果网站是支持的但扩展没有显示，请尝试刷新页面。
  • 确认扩展在浏览器的扩展管理页面中处于已启用状态。
