Instalación

Instala la extensión Live Translator en cualquier navegador basado en Chromium (Chrome, Edge, Brave, Arc, Opera, etc.).

1 Instala desde Chrome Web Store

Abre la ficha oficial: Live Translator en Chrome Web Store, luego haz clic en Add to Chrome y confirma con Add extension.

¡Listo!

Ahora deberías ver el icono de Live Translator en la barra de herramientas de tu navegador. La extensión está instalada y lista para configurarse.

Consejo

La extensión captura directamente el audio de la pestaña, así que funciona con cualquier audio que se reproduzca en la pestaña del navegador en los sitios compatibles — no necesitas configurar un micrófono aparte para el audio de la otra persona.

Vista general de la extensión

Un resumen rápido de la interfaz principal y dónde se encuentra cada cosa.

El panel de Live Translator tiene estas áreas principales:

  • Barra de título — Botón de ajustes, interruptor del modo subtítulos y botón de cerrar. Si hay una actualización disponible, verás un indicador aquí.
  • Selector de idioma — Dos menús desplegables: "Mi idioma" (el idioma que tú hablas) y "Su idioma" (el idioma del que quieres traducir).
  • Área de transcripción — Donde aparece el texto traducido en tiempo real, organizado por hablante.
  • Barra de controles — Botón de Iniciar/Detener/Pausar, micrófono, ajustes de narración, asistente de IA, tamaño de fuente, descarga y más opciones.
Consejo

Puedes arrastrar el panel desde su barra de título para moverlo a cualquier lugar de la pantalla, y redimensionarlo arrastrando los bordes. Así no bloqueará contenido importante durante reuniones o videos.

Obtener una clave API de Soniox

Soniox proporciona el motor de reconocimiento de voz y traducción en tiempo real. Necesitas una clave API de Soniox para usar Live Translator. Obligatorio

Importante

Una clave API de Soniox es obligatoria para que la extensión funcione. Sin ella, la extensión no puede transcribir ni traducir ningún audio.

1 Ve al sitio web de Soniox

Abre https://console.soniox.com/signup en tu navegador. Crea una cuenta con tu correo electrónico o inicia sesión con Google.

2 Obtén tu clave API

Después de iniciar sesión, ve a la sección de API en tu panel de control de Soniox. También puedes ir directamente a la guía de inicio de Soniox para ver las instrucciones paso a paso.

Copia tu clave API — la necesitarás en los siguientes pasos.

3 Añadir método de pago (necesario para usar la API)

Añade un método de pago a tu cuenta de Soniox. Puedes usar tarjeta de crédito, PayPal u otros métodos. La API de Soniox funciona con pago por uso — solo pagas por lo que consumes.

Nota

Necesitas depositar un mínimo de $1 en tu cuenta para usar la API. Puedes depositar más si lo deseas.

Obtener una clave API de OpenAI (opcional)

Una clave API de OpenAI es opcional — solo la necesitas si quieres usar las voces de OpenAI para la narración o la función de Asistente de IA integrado. Pro

1 Ve a la plataforma de OpenAI

Abre https://platform.openai.com/api-keys en tu navegador.

2 Crea una cuenta o inicia sesión

Si no tienes una cuenta de OpenAI, haz clic en "Sign Up" y crea una. Si ya tienes una, inicia sesión.

3 Crea una nueva clave API

Haz clic en "+ Create new secret key", ponle un nombre como "Live Translator" y haz clic en "Create secret key".

Copia y guarda la clave de inmediato — OpenAI solo la mostrará una vez.

4 Añadir método de pago (necesario para usar la API)

Ve a la configuración de facturación y añade un método de pago. La API de OpenAI funciona con pago por uso — solo pagas por lo que consumes.

Nota

Si solo quieres narración (texto a voz), puedes usar las voces de Google Translate de forma gratuita. La clave de OpenAI solo es necesaria si quieres las voces premium de OpenAI o la función de Asistente de IA.

Introducir tus claves API

Una vez que tengas tus claves API, introdúcelas en la página de ajustes de Live Translator.

1 Abre los ajustes de Live Translator

Puedes abrir los ajustes de cualquiera de estas formas:

  • Haz clic en el icono de engranaje (⚙) en la esquina superior derecha del panel del traductor.
  • Haz clic derecho en el icono de Live Translator en tu barra de herramientas → "Opciones".
  • Ve a la página de extensiones de tu navegador → busca Live Translator → "Detalles" → "Opciones de extensión".
2 Introduce tu clave API de Soniox

En la página de ajustes, haz clic en "Soniox" en la barra lateral izquierda. Pega tu clave API de Soniox en el campo de texto. La clave se guarda automáticamente.

3 Introduce tu clave API de OpenAI (opcional)

Haz clic en "OpenAI" en la barra lateral izquierda. Pega tu clave API de OpenAI en el campo de texto. La clave se guarda automáticamente.

Nota de seguridad

Tus claves API se almacenan solo de forma local en tu dispositivo mediante el almacenamiento seguro de extensiones de Chrome. Nunca se envían a los servidores de Live Translator — solo se usan para comunicarse directamente con Soniox y OpenAI.

Iniciar una sesión de traducción

Aprende cómo empezar a traducir en una reunión o video.

1 Abre un sitio web compatible

Ve a cualquier plataforma compatible (p. ej., Google Meet, Zoom, YouTube, etc.) y únete a una reunión o reproduce un video.

2 Abre el panel de Live Translator

Haz clic en el icono de Live Translator en la barra de herramientas de tu navegador. El panel del traductor aparecerá en la página. También puedes hacer clic en el icono flotante que aparece en la página.

3 Selecciona tus idiomas

Usa los dos menús desplegables de idioma en la parte superior del panel:

  • "Mi idioma" — El idioma que tú hablas (p. ej., español).
  • "Su idioma" — El idioma que hablan los demás y que quieres traducir (p. ej., inglés).
4 Pulsa el botón Iniciar

Haz clic en el botón Iniciar (icono de reproducción) en la barra de controles en la parte inferior del panel.

Es posible que tu navegador te pida permiso para capturar audio — haz clic en "Permitir" y selecciona la pestaña que quieres capturar.

5 Ve las traducciones en tiempo real

A medida que la gente habla, verás tanto el texto original como el texto traducido en el área de transcripción. Las líneas se agrupan automáticamente por hablante.

Desplazamiento automático

La transcripción se desplaza automáticamente para mostrar el texto más reciente. Si te desplazas hacia arriba para leer texto anterior, el desplazamiento automático se pausa. Haz clic en el botón "Ir al más reciente" que aparece para volver al texto en directo.

Controles: Pausa, Reanudar, Finalizar

Conoce los controles de la sesión de traducción.

BotónQué hace
Iniciar (▶)Comienza una nueva sesión de traducción. Borra cualquier transcripción anterior.
Pausa (⏸)Detiene la traducción temporalmente. La transcripción se conserva. La captura de audio continúa, pero el texto queda en pausa.
Reanudar (▶)Continúa la traducción después de una pausa. El texto nuevo aparece a continuación del contenido anterior.
Finalizar (⏹)Detiene la sesión por completo. La transcripción se mantiene en pantalla hasta que inicies una nueva sesión. Se finaliza la captura de audio y se desactivan el micrófono y la narración.

Cambiar idiomas

Puedes cambiar los idiomas de traducción en cualquier momento.

1 Haz clic en el menú desplegable de idioma

Haz clic en el desplegable de "Mi idioma" o de "Su idioma" en la parte superior del panel.

2 Selecciona un nuevo idioma

La lista incluye más de 60 idiomas ordenados alfabéticamente. Desplázate o escribe para buscar tu idioma y haz clic para seleccionarlo.

3 La traducción se reinicia automáticamente

Si hay una sesión activa, al cambiar el idioma la traducción se reiniciará automáticamente con el nuevo par de idiomas. Tu selección de idioma se guarda para la próxima vez.

Activar tu micrófono

Activa tu micrófono para que tu propia voz también se traduzca junto con el audio de la reunión o del video.

1 Haz clic en el botón del micrófono

En la barra de controles, haz clic en el icono del micrófono. El icono se resalta cuando el micrófono está activado.

2 Permite el acceso al micrófono

Si es la primera vez, tu navegador te pedirá permiso para acceder al micrófono. Haz clic en "Permitir".

Consejo

El micrófono está desactivado por defecto al abrir el panel. Actívalo solo si quieres que tu propia voz se transcriba y traduzca. Cuando pulsas Finalizar, el micrófono se desactiva automáticamente.

Cambiar tamaño de fuente

Ajusta el tamaño del texto de la transcripción para una lectura cómoda.

1 Haz clic en el botón de tamaño de fuente

En la barra de controles, haz clic en el botón de tamaño de fuente (icono Aa). Aparecerá un menú con tres opciones:

  • Pequeño — Vista compacta, cabe más texto
  • Mediano — Tamaño equilibrado por defecto
  • Grande — Más fácil de leer a distancia

Haz clic en el tamaño que prefieras. El cambio se aplica de inmediato y se guarda para la próxima vez.

Nombres de hablantes

Asigna nombres reales a los hablantes para identificar fácilmente quién dijo qué en la transcripción. Gratis

1 Abre el menú "Más"

En la barra de controles, haz clic en el botón "Más" (icono ⋯). Luego haz clic en el botón "Configuración de hablantes".

2 Abre el diálogo "Configuración de hablantes"

Se abrirá el diálogo de configuración de hablantes.

3 Asigna nombres

Verás una lista de los hablantes detectados (Hablante 1, Hablante 2, etc.). Escribe un nombre real junto a cada uno (p. ej., "Juan", "María"). Haz clic en "Añadir fila" si necesitas más.

Haz clic en "Guardar". Todas las líneas de la transcripción — tanto las pasadas como las futuras — usarán los nombres que asignaste.

Narración (texto a voz)

Haz que las traducciones se lean en voz alta para poder escuchar en lugar de leer. Esta es la función "Narrar su voz". Pro

1 Abre los ajustes de narración

En la barra de controles, haz clic en el botón de narración (icono de altavoz). Se abrirá el diálogo de ajustes de narración.

2 Elige un proveedor de TTS y una voz

Selecciona tu proveedor de voz:

  • Google Translate (gratis) — Funciona sin clave API. Voces de buena calidad en más de 60 idiomas.
  • OpenAI (premium) — Voces de mayor calidad y más naturales. Requiere una clave API de OpenAI.

Selecciona una voz del menú desplegable y haz clic en "Test" para escuchar cómo suena.

Ajusta la velocidad de voz y el volumen si lo necesitas.

3 Inicia la narración

Haz clic en el botón "Narrar su voz" en el diálogo. A partir de ahora, cada vez que aparezca una traducción, se leerá en voz alta.

Para detenerla, haz clic de nuevo en el botón de narración y pulsa "Detener".

Narración desde subtítulos de YouTube

Usa los subtítulos traducidos de YouTube como base para la narración, para que la extensión lea en el idioma que ya elegiste en YouTube. Esta función puede ahorrar hasta 99,9% del costo de Soniox al usar YouTube. Pro

Orden importante

Para que funcione bien, primero debes elegir el idioma de subtítulos en YouTube y después iniciar la narración en la extensión.

1 Primero configura los subtítulos traducidos en YouTube

En el reproductor de YouTube, activa los subtítulos y elige el idioma traducido que quieres escuchar (por ejemplo: vietnamita).

2 Activa la detección automática en Narración

Abre los ajustes de Narración de la extensión y activa Auto detect translated caption.

3 Inicia la narración

Pulsa "Narrar su voz". La narración seguirá el idioma de subtítulos traducidos que elegiste en YouTube.

Ejemplo

Si quieres escuchar la narración en vietnamita, primero pon los subtítulos de YouTube en vietnamita. Luego activa Auto detect translated caption e inicia "Narrar su voz".

Narrar mi voz (modo intérprete)

Habla en tu idioma y haz que tu voz traducida se reproduzca para los demás participantes de la reunión — como tener un intérprete personal. Pro

Requisitos

Esta función está disponible en Google Meet, Zoom (Web), Microsoft Teams (Web) y WhatsApp Web. Necesitas tener el micrófono activado y una sesión de traducción en curso.

1 Abre los ajustes de narración

Haz clic en el botón de narración en la barra de controles para abrir el diálogo de ajustes de narración.

2 Configura "Narrar mi voz"

En el diálogo, busca la sección "Narrar mi voz". Elige tu voz y proveedor preferidos (Google u OpenAI). Puedes configurar una voz diferente para la narración de "Mi voz" y de "Su voz".

Haz clic en "Test" para escuchar la voz seleccionada.

3 Inicia "Narrar mi voz"

Haz clic en "Iniciar Narrar mi voz". Cuando hables, tus palabras serán:

  1. Transcritas en tu idioma
  2. Traducidas al otro idioma
  3. Reproducidas en voz alta en la reunión para que la otra persona escuche tu voz traducida

Tu micrófono se silencia automáticamente mientras se reproduce el audio traducido para evitar eco.

4 Detener la narración

Haz clic en el botón de narración y luego en "Detener", o pulsa Finalizar para detener toda la sesión (lo que también detiene la narración).

Nota

Solo puedes usar un modo de narración a la vez — ya sea "Narrar mi voz" o "Narrar su voz", no ambos a la vez. El icono de la barra de herramientas cambia para indicar qué modo está activo.

Enviar traducción por chat

Envía automáticamente tu voz traducida como mensajes de texto en el chat de la reunión. Pro

1 Activa "Enviar por chat"

En la barra de controles, busca el botón de enviar por chat (junto al icono del micrófono). Haz clic para activarlo — el botón se pone en verde cuando está activo.

2 Habla y se envía automáticamente

Con el micrófono activado y una sesión de traducción en marcha, cuando hables tu texto traducido se insertará automáticamente en el chat de la reunión y se enviará.

Funciona en Google Meet, Zoom (Web) y Microsoft Teams (Web).

Consejo

Es útil cuando "Narrar mi voz" no está disponible en tu plataforma, o si quieres usar la narración de voz Y la traducción por chat de texto a la vez.

Modo subtítulos

Cambia a una barra de subtítulos minimalista, estilo cine, que muestra solo la última traducción en la parte inferior de la pantalla. Pro

1 Activa el modo subtítulos

En la barra de título del panel del traductor, haz clic en el icono de subtítulos (icono de closed captions).

El panel completo se reemplazará por una barra de subtítulos fina en la parte inferior de la pantalla.

2 Personaliza la barra de subtítulos

Haz clic en el icono de ajustes (engranaje) en la barra de subtítulos para modificar:

  • Opacidad — Desliza para hacer la barra más o menos transparente.
  • Tamaño de fuente — Elige entre texto pequeño, mediano o grande.

También puedes arrastrar la barra de subtítulos para moverla y redimensionarla arrastrando los bordes.

3 Salir del modo subtítulos

Haz clic en el botón de cerrar (✕) en la barra de subtítulos para volver al panel completo del traductor.

Asistente de IA integrado

Obtén explicaciones y respuestas generadas por IA basadas en el contexto de la conversación, directamente desde la transcripción. Pro

Requisito

El Asistente de IA integrado requiere una clave API de OpenAI configurada en los ajustes.

Explicar una línea

1 Pasa el cursor sobre una línea de la transcripción

Cuando pasas el cursor sobre cualquier línea de la transcripción, aparece un icono de IA (🤖).

2 Haz clic en el icono de IA y elige "Explicar"

El asistente de IA se abre en una ventana aparte. Usará el contexto completo de la transcripción y la línea seleccionada para darte una explicación detallada.

Generar una respuesta

1 Haz clic en el icono de IA y elige "Generar respuesta"

El asistente de IA generará una respuesta sugerida basada en lo que se dijo en la conversación. Incluso puede usar búsqueda web para darte respuestas más precisas.

Abrir el panel del Asistente

1 Haz clic en el botón del Asistente en la barra de controles

También puedes abrir el panel del asistente de IA directamente desde el botón del asistente (icono 🤖) en la barra de controles. Se abre una ventana de chat que puedes arrastrar y donde puedes escribir cualquier pregunta. El asistente tiene acceso a toda la transcripción como contexto.

Configuración de contexto

Proporciona contexto sobre el tema de la conversación para mejorar la precisión de la traducción. Gratis

1 Abre la configuración de contexto

Haz clic en el botón "Más" (⋯) en la barra de controles y luego en "Contexto".

2 Rellena los campos de contexto

Puedes proporcionar:

  • Dominio / Tema — De qué trata la conversación (p. ej., "Conferencia médica", "Daily de ingeniería de software").
  • Términos clave — Términos técnicos, nombres propios o jerga que pueden ser difíciles de reconocer (p. ej., "Kubernetes, API gateway, microservicios").
  • Información del hablante — Nombres y roles de los participantes (p. ej., "Dr. Smith — cardiólogo, presentador").

Haz clic en "Aplicar" para enviar este contexto al motor de traducción durante la sesión actual.

3 Guardar preajustes (opcional)

Si tienes reuniones recurrentes sobre temas similares, haz clic en "Guardar preajuste" para guardar el contexto actual. La próxima vez, solo carga el preajuste en lugar de escribirlo todo de nuevo.

También puedes importar/exportar preajustes como archivos JSON para compartirlos o hacer una copia de seguridad.

Autocompletado en YouTube

Cuando usas Live Translator en YouTube, los campos de contexto se pueden rellenar automáticamente con el título y la descripción del video. Esto ayuda al motor de traducción a comprender mejor el tema del video.

Descargar transcripción

Guarda tus transcripciones traducidas como archivos para consultarlas después o usarlas en otras herramientas. Pro

1 Haz clic en el botón de descarga

Después de una sesión de traducción (o durante una), haz clic en el botón de descarga (icono ↓) en la barra de controles. Se abrirá el diálogo de descarga de transcripción.

2 Elige un formato

Selecciona tu formato preferido:

FormatoIdeal para
TXTArchivo de texto simple con marcas de tiempo y nombres de hablantes. Fácil de leer.
JSONFormato de datos estructurado. Ideal para procesar con otras herramientas o scripts.
SRT (Original)Archivo de subtítulos con el texto en el idioma original. Para usar en software de edición de video.
SRT (Traducido)Archivo de subtítulos con el texto traducido. Para usar en software de edición de video.
3 Descarga el archivo

Haz clic en el botón de descarga del formato que prefieras. El archivo se genera en tu navegador y se guarda en tu carpeta de descargas predeterminada.

Privacidad

Los archivos de transcripción se generan completamente en tu navegador. No se envían datos de transcripción a ningún servidor durante el proceso de descarga.

Actualización automática

La extensión busca actualizaciones automáticamente cada vez que la abres.

1 Busca el indicador de actualización

Cuando haya una nueva versión disponible, verás un indicador de "Actualización disponible" en la barra de título del panel del traductor.

2 Haz clic para actualizar

Haz clic en el indicador para abrir la ficha en Chrome Web Store y verificar la versión más reciente. Las instalaciones desde la Store se actualizan automáticamente cuando se publica una nueva versión.

Solución de problemas

Problemas comunes y cómo resolverlos.

Mensaje "No Soniox API key"

! Solución

Aún no has introducido tu clave API de Soniox. Ve a Introducir tus claves API para configurarla.

No se captura audio

! Solución
  • Asegúrate de haber hecho clic en "Permitir" cuando el navegador pidió permiso para capturar audio.
  • Asegúrate de haber seleccionado la pestaña correcta para compartir cuando se te solicitó.
  • Prueba a actualizar la página y abrir el traductor de nuevo.
  • Comprueba que la reunión o el video realmente tengan audio reproduciéndose.

La traducción parece imprecisa

! Solución
  • Asegúrate de que el desplegable "Su idioma" esté configurado con el idioma correcto que se está hablando.
  • Usa la Configuración de contexto para proporcionar información sobre el tema y los términos clave.
  • Comprueba que la calidad del audio sea buena — el ruido de fondo puede afectar la precisión.

"Narrar mi voz" — los demás no escuchan la traducción

! Solución
  • Confirma que estás en una plataforma compatible (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Asegúrate de que haya una sesión de traducción en curso y tu micrófono esté activado.
  • Verifica que tengas una licencia Pro activada.
  • Prueba a actualizar la página de la reunión y empezar de nuevo.

Las funciones de OpenAI no funcionan

! Solución
  • Verifica que tu clave API de OpenAI esté introducida correctamente en los ajustes.
  • Asegúrate de que tu cuenta de OpenAI tenga facturación configurada con créditos suficientes.
  • Si no necesitas las funciones de OpenAI, usa las voces de Google Translate para la narración (gratis, sin necesidad de clave API).

La extensión no aparece en un sitio web

! Solución
  • Live Translator solo funciona en plataformas compatibles. No aparecerá en sitios web no compatibles.
  • Si el sitio es compatible pero la extensión no aparece, prueba a actualizar la página.
  • Asegúrate de que la extensión esté activada en la página de administración de extensiones de tu navegador.
