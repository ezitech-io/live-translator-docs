Instalación

Instala la extensión Live Translator en cualquier navegador basado en Chromium (Chrome, Edge, Brave, Arc, Opera, etc.).

1 Abre el sitio web de Live Translator

Ir a https://translate.ezitech.io en tu navegador.

2 Haga clic en el botón "Instalar"

Haga clic en Agregar a Chrome botón. Esto descargará el archivo de extensión.

3 Habilite el Modo Desarrollador en su navegador

Abre la página de administración de extensiones de tu navegador:

  • Cromo/Valiente/Arco: Tipo chrome://extensions en la barra de direcciones y presione Enter.
  • Borde: Tipo edge://extensions en la barra de direcciones y presione Enter.

Alternar Modo desarrollador ON en la esquina superior derecha de la página.

4 Instalar el archivo de extensión

Arrastra y suelta el archivo descargado. .zip archivo en la página de extensiones.

¡Listo!

Ahora deberías ver el ícono de Live Translator en la barra de herramientas de tu navegador. La extensión está instalada y lista para ser configurada.

Descripción general de la extensión

Una descripción general rápida de la interfaz principal y dónde se encuentra todo.

El panel Live Translator tiene estas áreas principales:

  • Barra de título — Botón de configuración, alternar modo de subtítulos, botón Cerrar. Si hay una actualización disponible, verá una insignia aquí.
  • Selector de idioma — Dos menús desplegables: "Mi idioma" (el idioma que hablas) y "Su idioma" (el idioma desde el que deseas traducir).
  • Área de transcripción — Donde el texto traducido aparece en tiempo real, organizado por hablante.
  • Barra de control — Botón Iniciar/Detener/Pausar, alternar micrófono, configuración de narración, asistente de IA, tamaño de fuente, descarga y más opciones.
Consejo

Tú puedes arrastra el panel por su barra de título para moverlo a cualquier lugar de la pantalla, y cambiar su tamaño arrastrando los bordes. De esta forma no bloqueará contenido importante durante reuniones o vídeos.

Obtenga una clave API de Soniox

Soniox proporciona el motor de traducción y reconocimiento de voz en tiempo real. Necesita una clave API de Soniox para utilizar Live Translator. Requerido

Importante

Una clave API de Soniox es requerido para que la extensión funcione. Sin él, la extensión no puede transcribir ni traducir ningún audio.

1 Ir al sitio web de Soniox

Abrir https://console.soniox.com/signup en tu navegador. Cree una cuenta utilizando su correo electrónico o inicio de sesión de Google.

2 Obtén tu clave API

Después de iniciar sesión, navega a la sección API de tu panel de control de Soniox. También puedes ir directamente a Guía de introducción a Soniox para obtener instrucciones paso a paso.

Copia tu clave API; la necesitarás en los siguientes pasos.

3 Agregar facturación (requerido para el uso de API)

Agrega un método de pago a tu cuenta Soniox. Puede utilizar una tarjeta de crédito, PayPal u otros métodos de pago. La API de Soniox es de pago por uso: solo paga por lo que usa.

Nota

Necesitas depositar un mínimo de $1 en tu cuenta para usar la API. Puedes depositar más si lo deseas.

Obtenga una clave API de OpenAI (opcional)

Una clave API de OpenAI es opcional : solo lo necesitas si quieres usar voces OpenAI para la narración o la función Inline AI Assistant. Pro

1 Ir a la plataforma OpenAI

Abrir https://platform.openai.com/api-keys en tu navegador.

2 Crea una cuenta o inicia sesión

Si no tienes una cuenta OpenAI, haz clic en "Registrarse" y crea uno. Si ya tienes uno, inicia sesión.

3 Crear una nueva clave API

Haga clic "+ Crear nueva clave secreta", asígnale un nombre como "Traductor en vivo" y haz clic en "Crear clave secreta".

Copia y guarda la clave inmediatamente. — OpenAI solo lo mostrará una vez.

4 Agregar facturación (requerido para el uso de API)

Ir a Configuración de facturación y agrega un método de pago. La API de OpenAI es de pago por uso: solo paga por lo que usa.

Nota

Si solo quieres narración (texto a voz), puedes usar Google Translate voces gratis en su lugar. La clave OpenAI solo es necesaria si desea las voces premium de OpenAI o la función Asistente AI.

Ingrese sus claves API

Una vez que tengas tus claves API, ingrésalas en la página de configuración de Live Translator.

1 Abrir la configuración de Live Translator

Puedes abrir la configuración de cualquiera de estas maneras:

  • Haga clic en icono de engranaje (⚙) en la parte superior derecha del panel del traductor.
  • Haz clic derecho en el ícono de Live Translator en tu barra de herramientas → "Opciones".
  • Vaya a la página de extensiones de su navegador → busque Live Translator → "Detalles" → "Opciones de extensión".
2 Ingrese su clave API de Soniox

En la página de configuración, haga clic en "Soniox" en la barra lateral izquierda. Pegue su clave API de Soniox en el campo de entrada. La clave se guarda automáticamente.

3 Ingresa tu clave API de OpenAI (opcional)

Haga clic en "AI abierta" en la barra lateral izquierda. Pegue su clave API de OpenAI en el campo de entrada. La clave se guarda automáticamente.

Nota de seguridad

Tus claves API se almacenan localmente solo en tu dispositivo usando la extensión de almacenamiento segura de Chrome. Nunca se envían a los servidores de Live Translator; solo se utilizan para comunicarse directamente con Soniox y OpenAI.

Iniciar una sesión de traducción

Aprenda cómo comenzar a traducir en una reunión o video.

1 Abrir un sitio web compatible

Ir a cualquier plataforma compatible (por ejemplo, Google Meet, Zoom, YouTube, etc.) y únete a una reunión o reproduce un vídeo.

2 Abre el panel de Live Translator

Haga clic en Icono de Traductor en vivo en la barra de herramientas de tu navegador. El panel del traductor aparecerá en la página. También puede hacer clic en icono flotante que aparece en la página.

3 Selecciona tus idiomas

Utiliza los dos menús desplegables de idiomas en la parte superior del panel:

  • "Mi idioma" — El idioma tú habla (p. ej., vietnamita).
  • "Su idioma" — El idioma otros habla que quieres traducir (por ejemplo, inglés).
4 Presiona el botón Inicio

Haga clic en Inicio botón (icono de reproducción) en la barra de control en la parte inferior del panel.

Su navegador puede solicitar permiso para capturar audio: haga clic en "Permitir" y selecciona la pestaña que deseas capturar.

5 Ver traducciones en tiempo real

Mientras la gente habla, verás tanto el texto original y el texto traducido aparecen en el área de transcripción. Las líneas se agrupan automáticamente por altavoz.

Desplazamiento automático

La transcripción se desplaza automáticamente para mostrar el texto más reciente. Si se desplaza hacia arriba para leer el texto anterior, el desplazamiento automático se detiene. Haga clic en "Desplazarse hasta lo último" botón que parece regresar al texto en vivo.

Controles: Pausa, Reanudar, Finalizar

Comprende los controles de la sesión de traducción.

BotónQué hace
Inicio (▶)Comienza una nueva sesión de traducción. Borra cualquier transcripción anterior.
Pausa (⏸)Detiene temporalmente la traducción. Se conserva la transcripción. La captura de audio continúa pero el texto está en pausa.
Reanudar (▶)Continúa la traducción después de una pausa. El nuevo texto aparece después del contenido anterior.
Fin (⏹)Detiene la sesión por completo. La transcripción se mantiene en pantalla hasta que inicie una nueva sesión. Finaliza la captura de audio, apaga el micrófono y la narración.

Cambiar idiomas

Puedes cambiar los idiomas de traducción en cualquier momento.

1 Haz clic en el menú desplegable de idiomas

Haz clic en cualquiera de los "Mi idioma" o "Su idioma" menú desplegable en la parte superior del panel.

2 Selecciona un nuevo idioma

La lista admite 60+ idiomas ordenados alfabéticamente. Desplázate o escribe para buscar tu idioma. Haga clic para seleccionarlo.

3 La traducción se reinicia automáticamente

Si hay una sesión activa, cambiar el idioma reiniciará automáticamente la traducción con el nuevo par de idiomas. Su selección de idioma se guarda para la próxima vez.

Enciende tu micrófono

Habilite su micrófono para que su propio discurso también se traduzca junto con el audio de la reunión/video.

1 Haz clic en el botón del micrófono

En la barra de control, haz clic en el icono de micrófono. El icono se activa (resaltado) cuando el micrófono está encendido.

2 Permitir acceso al micrófono

Si es la primera vez, tu navegador te pedirá permiso para el micrófono. Haga clic "Permitir".

Consejo

El micrófono está desactivado de forma predeterminada al abrir el panel. Actívelo sólo si desea que se transcriba y traduzca su propio discurso. Cuando presionas Fin, el micrófono se apaga automáticamente.

Cambiar tamaño de fuente

Ajusta el tamaño del texto de la transcripción para una lectura cómoda.

1 Haga clic en el botón de tamaño de fuente

En la barra de control, haz clic en el botón de tamaño de fuente (ícono Aa). Aparece un menú con tres tamaños:

  • Pequeño — Vista compacta, se adapta a más texto
  • Medio — Tamaño equilibrado predeterminado
  • Grande — Más fácil de leer a distancia

Haz clic en tu tamaño preferido. El cambio se aplica inmediatamente y se guarda para la próxima vez.

Nombres de los oradores

Asigna nombres reales a los hablantes para que puedas identificar fácilmente quién dijo qué en la transcripción. Gratis

1 Abre el menú "Más"

En la barra de control, haz clic en el "Más" Botón (icono ⋯). Luego haga clic "Configuración de los altavoces" botón.

2 Abra el cuadro de diálogo "Configuración de los altavoces"

Se abrirá el cuadro de diálogo Configuración de altavoces.

3 Asignar nombres

Verás una lista de los oradores detectados (Altavoz 1, Orador 2, etc.). Escriba un nombre real al lado de cada uno (por ejemplo, "John", "Sarah"). Haga clic "Agregar fila" si necesitas más.

Haga clic "Guardar". Todas las líneas de transcripción (pasadas y futuras) ahora usarán los nombres que usted asignó.

Narración (Texto a voz)

Haz que te lean las traducciones en voz alta para que puedas escuchar en lugar de leer. Esta es la función "Narrar su voz". Pro

1 Abrir configuración de narración

En la barra de control, haz clic en el botón de narración (icono de altavoz). Se abre el cuadro de diálogo Configuración de narración.

2 Elija un proveedor de TTS y voz

Selecciona tu proveedor de voz:

  • Traductor de Google (Gratis) — Funciona sin ninguna clave API. Voces de buena calidad en más de 60 idiomas.
  • OpenAI (Premium) — Voces más naturales y de mayor calidad. Requiere una clave API de OpenAI.

Selecciona un voz en el menú desplegable y haga clic en "Prueba" para obtener una vista previa de cómo suena.

Ajustar velocidad de voz y volumen si es necesario.

3 Iniciar narración

Haga clic "Narra su voz" botón en el cuadro de diálogo. Ahora, cada vez que aparezca una traducción, se la leerá en voz alta.

Para detener, haz clic nuevamente en el botón de narración y haz clic en "Detener".

Narrar mi voz (modo intérprete)

Hable en su idioma y reproduzca su voz traducida a otros participantes de la reunión, como si tuviera un intérprete personal. Pro

Requisitos

Esta función funciona en Google Meet, Zoom (Web), Equipos de Microsoft (Web), y WhatsAppWeb. Necesitas tu micrófono habilitado y una sesión de traducción corriendo.

1 Abrir configuración de narración

Haga clic en botón de narración en la barra de control para abrir el cuadro de diálogo Configuración de narración.

2 Configurar "Narrar Mi Voz"

En el cuadro de diálogo, busque el "Narrar Mi Voz" sección. Elija su voz y proveedor preferidos (Google u OpenAI). Puedes establecer un voz diferente para la narración de "Mi voz" y "Su voz".

Haga clic "Prueba" para obtener una vista previa de la voz seleccionada.

3 Inicia "Narrar mi voz"

Haga clic "Empezar a Narrar Mi Voz". Cuando hablas, tus palabras son:

  1. Transcrito en tu idioma
  2. Traducido al otro idioma
  3. Hablado en voz alta en la reunión para que la otra persona escuche tu voz traducida.

Tu micrófono se silencia automáticamente mientras se reproduce el audio traducido para evitar el eco.

4 Detener narración

Haz clic en el botón de narración y luego "Detener", o presiona Fin para detener toda la sesión (que también detiene la narración).

Nota

Solo puedes ejecutar un modo de narración a la vez — ya sea "Narrar mi voz" o "Narrar su voz", no ambas simultáneamente. El icono de la barra de herramientas cambia para mostrar qué modo está activo.

Enviar traducción vía chat

Envía automáticamente tu discurso traducido como mensajes de texto en el chat de la reunión. Pro

1 Habilitar "Enviar vía Chat"

En la barra de control, busca el botón de envío de chat (al lado del ícono del micrófono). Haga clic en él para activarlo: el botón se activa. verde cuando está activo.

2 Habla y envía automáticamente

Con el micrófono encendido y una sesión de traducción en ejecución, cuando hablas, tu texto traducido se inserta automáticamente en la entrada del chat de la reunión y se envía.

Esto funciona en Google Meet, Zoom (Web), y Equipos de Microsoft (Web).

Consejo

Esto es útil cuando "Narrar mi voz" no está disponible para tu plataforma, o si deseas narración de voz Y traducción de chat de texto simultáneamente.

Modo de subtítulos

Cambie a una barra de subtítulos mínima, estilo película, que muestra solo la última traducción en la parte inferior de la pantalla. Pro

1 Habilitar modo de subtítulos

En la barra de título del panel del traductor, haz clic en el icono de subtítulo (icono de subtítulos).

El panel completo será reemplazado por una delgada barra de subtítulos en la parte inferior de la pantalla.

2 Personaliza la barra de subtítulos

Haga clic en icono de melodía (engranaje de configuración) en la barra de subtítulos para ajustar:

  • Opacidad — Desliza para hacer la barra más o menos transparente.
  • Tamaño de fuente —Elige entre texto pequeño, mediano o grande.

También puedes arrastrar la barra de subtítulos para moverla, y cambiar tamaño arrastrando los bordes.

3 Salir del modo de subtítulos

Haga clic en botón cerrar (✕) en la barra de subtítulos para regresar al panel completo del traductor.

Asistente de IA en línea

Obtenga explicaciones basadas en IA y respuestas generadas según el contexto de la conversación, directamente desde la transcripción. Pro

Requisito

El Asistente de IA en línea requiere un Clave API de OpenAI para configurar en ajustes.

Explica una línea

1 Pase el cursor sobre una línea de transcripción

Cuando pasas el cursor sobre cualquier línea de la transcripción, aparece un Icono de IA Aparece (🤖).

2 Haz clic en el ícono AI y elige "Explicar"

El asistente de IA se abre en una ventana separada. Utilizará el contexto de la transcripción completa y la línea seleccionada para brindarle una explicación detallada.

Generar una respuesta

1 Haz clic en el ícono AI y elige "Generar respuesta"

El asistente de IA generará una respuesta sugerida basada en lo que se dijo en la conversación. Incluso puede utilizar la búsqueda web para proporcionar respuestas más precisas.

Abre el panel Asistente

1 Haz clic en el botón Asistente en la barra de control.

También puedes abrir el panel del asistente de IA directamente desde la botón asistente (icono 🤖) en la barra de control. Esto abre una ventana de chat que se puede arrastrar donde puedes escribir cualquier pregunta. El asistente tiene acceso a la transcripción completa como contexto.

Configuración de contexto

Proporciona contexto sobre el tema de conversación para mejorar la precisión de la traducción. Gratis

1 Abrir configuración de contexto

Haga clic en "Más" Botón botón (⋯) en la barra de control, luego haga clic en "Contexto".

2 Rellena los campos de contexto

Puedes proporcionar:

  • Dominio / Tema —De qué trata la conversación (p. ej., "Conferencia médica", "Stand-up de ingeniería de software").
  • Términos clave —Términos técnicos, nombres propios o jerga que pueden ser difíciles de reconocer (por ejemplo, "Kubernetes, puerta de enlace API, microservicios").
  • Información del orador —Nombres y funciones de los participantes (p. ej., "Dr. Smith, cardiólogo, presentador").

Haga clic "Aplicar" para enviar este contexto al motor de traducción para la sesión actual.

3 Guardar ajustes preestablecidos (opcional)

Si tienes reuniones recurrentes con temas similares, haz clic en "Guardar preajuste" para guardar el contexto actual. La próxima vez, simplemente cargue el ajuste preestablecido en lugar de escribir todo nuevamente.

También puedes Importar/Exportar ajustes preestablecidos como archivos JSON para compartirlos o hacer una copia de seguridad de ellos.

Autocompletar de YouTube

Cuando se utiliza Live Translator en YouTube, los campos de contexto pueden ser llenado automáticamente con el título y la descripción del video. Esto ayuda al motor de traducción a comprender mejor el tema del video.

Descargar Transcripción

Guarde sus transcripciones traducidas como archivos para consultarlas más adelante o utilizarlas en otras herramientas. Pro

1 Haga clic en el botón Descargar

Después de una sesión de traducción (o durante una), haz clic en el botón de descarga (icono ↓) en la barra de control. Se abre el cuadro de diálogo Descargar transcripción.

2 Elige un formato

Selecciona tu formato preferido:

FormatoLo mejor para
TXTArchivo de texto simple con marcas de tiempo y nombres de oradores. Fácil de leer.
JSONFormato de datos estructurados. Bueno para procesar con otras herramientas o scripts.
SRT (Original)Archivo de subtítulos con el texto en idioma original. Uso en software de edición de vídeo.
SRT (Traducido)Archivo de subtítulos con el texto traducido. Uso en software de edición de vídeo.
3 Descargar el archivo

Haz clic en el botón de descarga del formato elegido. El archivo se genera en su navegador y se guarda en su carpeta de descarga predeterminada.

Privacidad

Se generan archivos de transcripción completamente en tu navegador. No se envían datos de transcripción a ningún servidor durante el proceso de descarga.

Verificación de actualización automática

La extensión busca actualizaciones automáticamente cada vez que la abres.

1 Busque la insignia de actualización

Cuando haya una nueva versión disponible, verás un "Actualización disponible" insignia en la barra de título del panel del traductor.

2 Haga clic para actualizar

Haz clic en la insignia para ir a la página de instalación donde podrás descargar la última versión. sigue lo mismo pasos de instalación para actualizar.

Solución de problemas

Problemas comunes y cómo solucionarlos.

Mensaje "Sin clave API de Soniox"

! Solución

Aún no has ingresado tu clave API de Soniox. Ir a Ingrese sus claves API para configurarlo.

No se está capturando audio

! Solución
  • Asegúrate de haber hecho clic "Permitir" cuando el navegador solicitó permiso de captura de audio.
  • Asegúrate de haber seleccionado el pestaña correcta para compartir cuando se le solicite.
  • Intenta actualizar la página y abrir el traductor nuevamente.
  • Verifique que la reunión/video realmente tenga reproducción de audio.

La traducción parece inexacta

! Solución
  • Asegúrate de que "Su idioma" el menú desplegable está configurado en el idioma correcto que se habla.
  • Usar Configuración de contexto para proporcionar información sobre el tema y términos clave.
  • Asegúrate de que la calidad del audio sea clara: el ruido de fondo puede afectar la precisión.

"Narrate My Voice" — otros no pueden escuchar la traducción

! Solución
  • Confirma que estás en un plataforma compatible (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Asegúrate de que haya una sesión de traducción corriendo y tu el micrófono está encendido.
  • Comprueba que tienes un Licencia profesional activado.
  • Intenta actualizar la página de la reunión y comenzar de nuevo.

Las funciones de OpenAI no funcionan

! Solución
  • Verifica tu Clave API de OpenAI se ingresó correctamente en configuración.
  • Asegúrate de que tu cuenta OpenAI tenga configuración de facturación con créditos suficientes.
  • Si no necesitas las funciones de OpenAI, usa Voces del Traductor de Google para la narración (gratis, no se necesita clave API).

La extensión no aparece en un sitio web

! Solución
  • Live Translator solo funciona en plataformas compatibles. No aparecerá en sitios web no compatibles.
  • Si el sitio es compatible pero la extensión no se muestra, intenta actualizando la página.
  • Asegúrate de que la extensión sea habilitado en la página de administración de extensiones de tu navegador.
