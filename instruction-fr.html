Installation

Installez l'extension Live Translator sur n'importe quel navigateur basé sur Chromium (Chrome, Edge, Brave, Arc, Opera, etc.).

1 Ouvrir le site Web de Live Translator

Rendez-vous sur https://translate.ezitech.io dans votre navigateur.

2 Cliquer sur le bouton « Installer »

Cliquez sur le bouton Add To Chrome. Le fichier de l'extension sera alors téléchargé.

3 Activer le mode développeur dans votre navigateur

Ouvrez la page de gestion des extensions de votre navigateur :

  • Chrome/Brave/Arc : Saisissez chrome://extensions dans la barre d'adresse et appuyez sur Entrée.
  • Edge : Saisissez edge://extensions dans la barre d'adresse et appuyez sur Entrée.

Activez le Mode développeur dans le coin supérieur droit de la page.

4 Installer le fichier de l'extension

Faites glisser et déposez le fichier .zip téléchargé sur la page des extensions.

Terminé !

Vous devriez maintenant voir l'icône Live Translator dans la barre d'outils de votre navigateur. L'extension est installée et prête à être configurée.

Présentation de l'extension

Un aperçu rapide de l'interface principale et de l'emplacement de chaque élément.

Le panneau Live Translator comprend les zones principales suivantes :

  • Barre de titre — Bouton Paramètres, bouton Mode sous-titres, bouton Fermer. Si une mise à jour est disponible, un badge s'affiche ici.
  • Sélecteur de langue — Deux menus déroulants : « Ma langue » (la langue que vous parlez) et « Leur langue » (la langue depuis laquelle vous souhaitez traduire).
  • Zone de transcription — L'endroit où le texte traduit apparaît en temps réel, organisé par locuteur.
  • Barre de contrôle — Bouton Démarrer/Arrêter/Pause, bouton microphone, paramètres de narration, assistant IA, taille de police, téléchargement et autres options.
Astuce

Vous pouvez déplacer le panneau en le faisant glisser par sa barre de titre, et le redimensionner en faisant glisser ses bords. Ainsi, il ne masquera pas le contenu important pendant vos réunions ou vidéos.

Obtenir une clé API Soniox

Soniox fournit le moteur de reconnaissance vocale et de traduction en temps réel. Vous avez besoin d'une clé API Soniox pour utiliser Live Translator. Requis

Important

Une clé API Soniox est indispensable au fonctionnement de l'extension. Sans elle, l'extension ne peut ni transcrire ni traduire l'audio.

1 Accéder au site de Soniox

Ouvrez https://console.soniox.com/signup dans votre navigateur. Créez un compte avec votre adresse e-mail ou via votre compte Google.

2 Obtenir votre clé API

Une fois connecté, rendez-vous dans la section API de votre tableau de bord Soniox. Vous pouvez également consulter directement le guide de démarrage de Soniox pour des instructions détaillées.

Copiez votre clé API — vous en aurez besoin dans les étapes suivantes.

3 Ajouter un moyen de paiement (requis pour utiliser l'API)

Ajoutez un moyen de paiement à votre compte Soniox. Vous pouvez utiliser une carte bancaire, PayPal ou d'autres méthodes de paiement. L'API Soniox fonctionne à l'usage — vous ne payez que ce que vous consommez.

Remarque

Vous devez déposer au minimum 1 $ sur votre compte pour utiliser l'API. Vous pouvez déposer davantage si vous le souhaitez.

Obtenir une clé API OpenAI (facultatif)

Une clé API OpenAI est facultative — elle n'est nécessaire que si vous souhaitez utiliser les voix OpenAI pour la narration ou la fonctionnalité Assistant IA intégré. Pro

1 Accéder à la plateforme OpenAI

Ouvrez https://platform.openai.com/api-keys dans votre navigateur.

2 Créer un compte ou se connecter

Si vous n'avez pas encore de compte OpenAI, cliquez sur « Sign Up » pour en créer un. Si vous en avez déjà un, connectez-vous.

3 Créer une nouvelle clé API

Cliquez sur « + Create new secret key », donnez-lui un nom comme « Live Translator », puis cliquez sur « Create secret key ».

Copiez et sauvegardez la clé immédiatement — OpenAI ne l'affichera qu'une seule fois.

4 Ajouter un moyen de paiement (requis pour utiliser l'API)

Rendez-vous dans les paramètres de facturation et ajoutez un moyen de paiement. L'API OpenAI fonctionne à l'usage — vous ne payez que ce que vous consommez.

Remarque

Si vous souhaitez uniquement la narration (synthèse vocale), vous pouvez utiliser gratuitement les voix Google Translate. La clé OpenAI n'est requise que si vous souhaitez bénéficier des voix premium d'OpenAI ou de la fonctionnalité Assistant IA.

Saisir vos clés API

Une fois vos clés API obtenues, saisissez-les dans la page des paramètres de Live Translator.

1 Ouvrir les paramètres de Live Translator

Vous pouvez accéder aux paramètres de plusieurs façons :

  • Cliquez sur l'icône d'engrenage (⚙) en haut à droite du panneau de traduction.
  • Faites un clic droit sur l'icône Live Translator dans votre barre d'outils → « Options ».
  • Rendez-vous sur la page des extensions de votre navigateur → trouvez Live Translator → « Détails » → « Options de l'extension ».
2 Saisir votre clé API Soniox

Dans la page des paramètres, cliquez sur « Soniox » dans la barre latérale gauche. Collez votre clé API Soniox dans le champ de saisie. La clé est enregistrée automatiquement.

3 Saisir votre clé API OpenAI (facultatif)

Cliquez sur « OpenAI » dans la barre latérale gauche. Collez votre clé API OpenAI dans le champ de saisie. La clé est enregistrée automatiquement.

Note de sécurité

Vos clés API sont stockées uniquement en local sur votre appareil, via le stockage sécurisé des extensions Chrome. Elles ne sont jamais envoyées aux serveurs de Live Translator — elles servent uniquement à communiquer directement avec Soniox et OpenAI.

Démarrer une session de traduction

Découvrez comment lancer une traduction lors d'une réunion ou d'une vidéo.

1 Ouvrir un site compatible

Rendez-vous sur l'une des plateformes compatibles (par exemple, Google Meet, Zoom, YouTube, etc.) et rejoignez une réunion ou lancez une vidéo.

2 Ouvrir le panneau Live Translator

Cliquez sur l'icône Live Translator dans la barre d'outils de votre navigateur. Le panneau de traduction apparaîtra sur la page. Vous pouvez également cliquer sur l'icône flottante qui s'affiche sur la page.

3 Sélectionner vos langues

Utilisez les deux menus déroulants de langue en haut du panneau :

  • « Ma langue » — La langue que vous parlez (par exemple, le français).
  • « Leur langue » — La langue que parlent les autres et que vous souhaitez traduire (par exemple, l'anglais).
4 Appuyer sur le bouton Démarrer

Cliquez sur le bouton Démarrer (icône lecture ▶) dans la barre de contrôle en bas du panneau.

Votre navigateur peut vous demander l'autorisation de capturer l'audio — cliquez sur « Autoriser » et sélectionnez l'onglet que vous souhaitez capturer.

5 Voir les traductions en temps réel

Au fur et à mesure que les participants parlent, le texte original et le texte traduit apparaissent dans la zone de transcription. Les lignes sont automatiquement regroupées par locuteur.

Défilement automatique

La transcription défile automatiquement pour afficher le texte le plus récent. Si vous faites défiler vers le haut pour relire un passage précédent, le défilement automatique se met en pause. Cliquez sur le bouton « Aller au plus récent » qui apparaît pour revenir au texte en direct.

Contrôles : Pause, Reprise, Fin

Présentation des commandes de la session de traduction.

BoutonFonction
Démarrer (▶)Lance une nouvelle session de traduction. Efface toute transcription précédente.
Pause (⏸)Interrompt temporairement la traduction. La transcription est conservée. La capture audio continue, mais le texte est mis en pause.
Reprendre (▶)Reprend la traduction après une pause. Le nouveau texte s'affiche à la suite du contenu précédent.
Fin (⏹)Arrête complètement la session. La transcription reste affichée jusqu'au démarrage d'une nouvelle session. Met fin à la capture audio et désactive le microphone et la narration.

Changer de langue

Vous pouvez modifier les langues de traduction à tout moment.

1 Cliquer sur le menu déroulant de langue

Cliquez sur le menu déroulant « Ma langue » ou « Leur langue » en haut du panneau.

2 Sélectionner une nouvelle langue

La liste propose plus de 60 langues classées par ordre alphabétique. Faites défiler ou tapez pour rechercher votre langue, puis cliquez pour la sélectionner.

3 La traduction redémarre automatiquement

Si une session est en cours, le changement de langue relancera automatiquement la traduction avec la nouvelle paire de langues. Votre sélection de langue est mémorisée pour la prochaine utilisation.

Activer votre microphone

Activez votre microphone pour que votre propre voix soit également transcrite et traduite en parallèle de l'audio de la réunion ou de la vidéo.

1 Cliquer sur le bouton microphone

Dans la barre de contrôle, cliquez sur l'icône du microphone. L'icône s'allume (en surbrillance) lorsque le micro est activé.

2 Autoriser l'accès au microphone

Si c'est la première fois, votre navigateur vous demandera l'autorisation d'accéder au microphone. Cliquez sur « Autoriser ».

Astuce

Le microphone est désactivé par défaut à l'ouverture du panneau. Activez-le uniquement si vous souhaitez que votre propre voix soit transcrite et traduite. Lorsque vous appuyez sur Fin, le microphone se désactive automatiquement.

Modifier la taille de la police

Ajustez la taille du texte de la transcription pour un confort de lecture optimal.

1 Cliquer sur le bouton de taille de police

Dans la barre de contrôle, cliquez sur le bouton de taille de police (icône Aa). Un menu apparaît avec trois options :

  • Petit — Vue compacte, affiche davantage de texte
  • Moyen — Taille équilibrée par défaut
  • Grand — Plus facile à lire à distance

Cliquez sur la taille souhaitée. La modification s'applique immédiatement et est mémorisée pour la prochaine utilisation.

Noms des locuteurs

Attribuez de vrais noms aux locuteurs afin d'identifier facilement qui a dit quoi dans la transcription. Gratuit

1 Ouvrir le menu « Plus »

Dans la barre de contrôle, cliquez sur le bouton « Plus » (icône ⋯). Puis cliquez sur le bouton « Paramètres des locuteurs ».

2 Ouvrir la boîte de dialogue « Paramètres des locuteurs »

La boîte de dialogue Paramètres des locuteurs s'ouvrira.

3 Attribuer des noms

Vous verrez la liste des locuteurs détectés (Locuteur 1, Locuteur 2, etc.). Saisissez un vrai nom à côté de chacun d'eux (par exemple, « Jean », « Sarah »). Cliquez sur « Ajouter une ligne » si vous avez besoin de plus d'entrées.

Cliquez sur « Enregistrer ». Toutes les lignes de transcription — passées et futures — utiliseront désormais les noms que vous avez attribués.

Narration (synthèse vocale)

Faites lire les traductions à voix haute pour pouvoir écouter au lieu de lire. Il s'agit de la fonctionnalité « Narrer leur voix ». Pro

1 Ouvrir les paramètres de narration

Dans la barre de contrôle, cliquez sur le bouton de narration (icône haut-parleur). La boîte de dialogue Paramètres de narration s'ouvre.

2 Choisir un fournisseur TTS et une voix

Sélectionnez votre fournisseur de voix :

  • Google Translate (Gratuit) — Fonctionne sans clé API. Voix de bonne qualité dans plus de 60 langues.
  • OpenAI (Premium) — Voix de meilleure qualité, plus naturelles. Nécessite une clé API OpenAI.

Sélectionnez une voix dans le menu déroulant et cliquez sur « Tester » pour écouter un aperçu.

Ajustez la vitesse de la voix et le volume si nécessaire.

3 Démarrer la narration

Cliquez sur le bouton « Narrer leur voix » dans la boîte de dialogue. Désormais, chaque traduction sera automatiquement lue à voix haute.

Pour arrêter, cliquez à nouveau sur le bouton de narration, puis sur « Arrêter ».

Narration via sous-titres YouTube

Utilisez les sous-titres traduits de YouTube comme source de narration, afin que l'extension lise dans la langue déjà choisie sur YouTube. Cette fonctionnalité peut réduire jusqu'à 99,9 % les coûts Soniox sur YouTube. Pro

Ordre important

Pour que cette fonctionnalité marche correctement, choisissez d'abord la langue des sous-titres sur YouTube, puis lancez la narration dans l'extension.

1 Configurez d'abord les sous-titres traduits sur YouTube

Dans le lecteur YouTube, activez les sous-titres puis choisissez la langue traduite que vous souhaitez entendre (par exemple : vietnamien).

2 Activez l'auto-détection dans les paramètres de narration

Ouvrez les paramètres de narration de l'extension et activez Auto detect translated caption.

3 Démarrez la narration

Cliquez sur « Narrer leur voix ». La narration suivra la langue de sous-titres traduits choisie sur YouTube.

Exemple

Si vous voulez une narration en vietnamien, réglez d'abord les sous-titres YouTube en vietnamien. Ensuite seulement, activez Auto detect translated caption puis lancez « Narrer leur voix ».

Narrer ma voix (mode Interprète)

Parlez dans votre langue et faites entendre votre voix traduite aux autres participants de la réunion — comme si vous aviez un interprète personnel. Pro

Prérequis

Cette fonctionnalité est disponible sur Google Meet, Zoom (Web), Microsoft Teams (Web) et WhatsApp Web. Votre microphone doit être activé et une session de traduction doit être en cours.

1 Ouvrir les paramètres de narration

Cliquez sur le bouton de narration dans la barre de contrôle pour ouvrir la boîte de dialogue Paramètres de narration.

2 Configurer « Narrer ma voix »

Dans la boîte de dialogue, repérez la section « Narrer ma voix ». Choisissez votre voix et votre fournisseur préférés (Google ou OpenAI). Vous pouvez attribuer une voix différente pour la narration de « Ma voix » et « Leur voix ».

Cliquez sur « Tester » pour écouter un aperçu de la voix sélectionnée.

3 Démarrer « Narrer ma voix »

Cliquez sur « Démarrer Narrer ma voix ». Lorsque vous parlez, vos paroles sont :

  1. Transcrites dans votre langue
  2. Traduites dans l'autre langue
  3. Prononcées à voix haute dans la réunion pour que l'autre personne entende votre voix traduite

Votre microphone est automatiquement mis en sourdine pendant la diffusion de l'audio traduit pour éviter les échos.

4 Arrêter la narration

Cliquez sur le bouton de narration puis sur « Arrêter », ou appuyez sur Fin pour mettre fin à toute la session (ce qui arrête également la narration).

Remarque

Vous ne pouvez utiliser qu'un seul mode de narration à la fois — soit « Narrer ma voix », soit « Narrer leur voix », mais pas les deux simultanément. L'icône de la barre d'outils change pour indiquer le mode actif.

Envoyer la traduction par chat

Envoyez automatiquement votre discours traduit sous forme de messages texte dans le chat de la réunion. Pro

1 Activer « Envoyer par chat »

Dans la barre de contrôle, repérez le bouton d'envoi par chat (à côté de l'icône du microphone). Cliquez dessus pour l'activer — le bouton devient vert lorsqu'il est actif.

2 Parlez et l'envoi se fait automatiquement

Avec votre microphone activé et une session de traduction en cours, lorsque vous parlez, votre texte traduit est automatiquement inséré dans le chat de la réunion et envoyé.

Cette fonctionnalité est disponible sur Google Meet, Zoom (Web) et Microsoft Teams (Web).

Astuce

Cette option est utile lorsque « Narrer ma voix » n'est pas disponible pour votre plateforme, ou si vous souhaitez combiner la narration vocale ET l'envoi par chat simultanément.

Mode sous-titres

Passez à une barre de sous-titres minimaliste, style cinéma, qui affiche uniquement la dernière traduction en bas de l'écran. Pro

1 Activer le mode sous-titres

Dans la barre de titre du panneau de traduction, cliquez sur l'icône de sous-titres (icône sous-titres codés).

Le panneau complet sera remplacé par une fine barre de sous-titres en bas de l'écran.

2 Personnaliser la barre de sous-titres

Cliquez sur l'icône de réglage (engrenage) sur la barre de sous-titres pour ajuster :

  • Opacité — Déplacez le curseur pour rendre la barre plus ou moins transparente.
  • Taille de la police — Choisissez entre petit, moyen ou grand.

Vous pouvez également déplacer la barre de sous-titres par glisser-déposer, et la redimensionner en faisant glisser ses bords.

3 Quitter le mode sous-titres

Cliquez sur le bouton de fermeture (✕) de la barre de sous-titres pour revenir au panneau de traduction complet.

Assistant IA intégré

Obtenez des explications et des réponses générées par l'IA en fonction du contexte de la conversation, directement depuis la transcription. Pro

Prérequis

L'Assistant IA intégré nécessite qu'une clé API OpenAI soit configurée dans les paramètres.

Expliquer une ligne

1 Survoler une ligne de la transcription

Lorsque vous survolez n'importe quelle ligne de la transcription, une icône IA (🤖) apparaît.

2 Cliquer sur l'icône IA et choisir « Expliquer »

L'assistant IA s'ouvre dans une fenêtre séparée. Il utilise le contexte complet de la transcription et la ligne sélectionnée pour vous fournir une explication détaillée.

Générer une réponse

1 Cliquer sur l'icône IA et choisir « Générer une réponse »

L'assistant IA génère une suggestion de réponse basée sur ce qui a été dit dans la conversation. Il peut même effectuer une recherche sur le Web pour fournir des réponses plus précises.

Ouvrir le panneau de l'assistant

1 Cliquer sur le bouton Assistant dans la barre de contrôle

Vous pouvez également ouvrir le panneau de l'assistant IA directement depuis le bouton Assistant (icône 🤖) dans la barre de contrôle. Une fenêtre de discussion déplaçable s'ouvre, dans laquelle vous pouvez poser n'importe quelle question. L'assistant a accès à l'intégralité de la transcription comme contexte.

Paramètres de contexte

Fournissez des informations sur le sujet de la conversation pour améliorer la précision de la traduction. Gratuit

1 Ouvrir les paramètres de contexte

Cliquez sur le bouton « Plus » (⋯) dans la barre de contrôle, puis cliquez sur « Contexte ».

2 Remplir les champs de contexte

Vous pouvez renseigner :

  • Domaine / Sujet — Le thème de la conversation (par exemple, « Conférence médicale », « Réunion quotidienne d'ingénierie logicielle »).
  • Termes clés — Termes techniques, noms propres ou jargon susceptibles d'être difficiles à reconnaître (par exemple, « Kubernetes, passerelle API, microservices »).
  • Informations sur les locuteurs — Noms et rôles des participants (par exemple, « Dr Dupont — cardiologue, présentateur »).

Cliquez sur « Appliquer » pour envoyer ce contexte au moteur de traduction pour la session en cours.

3 Enregistrer des préréglages (facultatif)

Si vous avez des réunions récurrentes sur des sujets similaires, cliquez sur « Enregistrer le préréglage » pour sauvegarder le contexte actuel. La prochaine fois, il vous suffira de charger le préréglage au lieu de tout ressaisir.

Vous pouvez également importer/exporter des préréglages au format JSON pour les partager ou les sauvegarder.

Remplissage automatique YouTube

Lorsque vous utilisez Live Translator sur YouTube, les champs de contexte peuvent être remplis automatiquement avec le titre et la description de la vidéo. Cela aide le moteur de traduction à mieux comprendre le sujet de la vidéo.

Télécharger la transcription

Enregistrez vos transcriptions traduites sous forme de fichiers pour les consulter ultérieurement ou les utiliser dans d'autres outils. Pro

1 Cliquer sur le bouton Télécharger

Après une session de traduction (ou pendant celle-ci), cliquez sur le bouton de téléchargement (icône ↓) dans la barre de contrôle. La boîte de dialogue Télécharger la transcription s'ouvre.

2 Choisir un format

Sélectionnez votre format préféré :

FormatIdéal pour
TXTFichier texte simple avec horodatages et noms des locuteurs. Facile à lire.
JSONFormat de données structuré. Idéal pour le traitement avec d'autres outils ou scripts.
SRT (Original)Fichier de sous-titres avec le texte dans la langue d'origine. À utiliser dans un logiciel de montage vidéo.
SRT (Traduit)Fichier de sous-titres avec le texte traduit. À utiliser dans un logiciel de montage vidéo.
3 Télécharger le fichier

Cliquez sur le bouton de téléchargement correspondant au format choisi. Le fichier est généré dans votre navigateur et enregistré dans votre dossier de téléchargement par défaut.

Les fichiers de transcription sont générés entièrement dans votre navigateur. Aucune donnée de transcription n'est envoyée à un serveur lors du processus de téléchargement.

Mise à jour automatique

L'extension vérifie automatiquement la disponibilité de mises à jour à chaque ouverture.

1 Repérer le badge de mise à jour

Lorsqu'une nouvelle version est disponible, un badge « Mise à jour disponible » s'affiche dans la barre de titre du panneau de traduction.

2 Cliquer pour mettre à jour

Cliquez sur le badge pour accéder à la page d'installation où vous pourrez télécharger la dernière version. Suivez les mêmes étapes d'installation pour effectuer la mise à jour.

Dépannage

Problèmes courants et comment les résoudre.

Message « Aucune clé API Soniox »

! Solution

Vous n'avez pas encore saisi votre clé API Soniox. Consultez la section Saisir vos clés API pour la configurer.

Aucun audio n'est capturé

! Solution
  • Assurez-vous d'avoir cliqué sur « Autoriser » lorsque le navigateur a demandé l'autorisation de capture audio.
  • Vérifiez que vous avez sélectionné le bon onglet à partager lorsque vous y avez été invité.
  • Essayez d'actualiser la page et de rouvrir le traducteur.
  • Vérifiez que la réunion ou la vidéo diffuse bien de l'audio.

La traduction semble inexacte

! Solution
  • Vérifiez que le menu déroulant « Leur langue » est bien réglé sur la langue réellement parlée.
  • Utilisez les paramètres de contexte pour fournir des informations sur le sujet et les termes clés.
  • Assurez-vous que la qualité audio est bonne — le bruit de fond peut affecter la précision.

« Narrer ma voix » — les autres n'entendent pas la traduction

! Solution
  • Vérifiez que vous êtes sur une plateforme compatible (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Assurez-vous qu'une session de traduction est en cours et que votre microphone est activé.
  • Vérifiez que vous disposez d'une licence Pro activée.
  • Essayez d'actualiser la page de la réunion et de recommencer.

Les fonctionnalités OpenAI ne fonctionnent pas

! Solution
  • Vérifiez que votre clé API OpenAI est correctement saisie dans les paramètres.
  • Assurez-vous que votre compte OpenAI dispose d'un moyen de paiement configuré avec suffisamment de crédits.
  • Si vous n'avez pas besoin des fonctionnalités OpenAI, utilisez les voix Google Translate pour la narration (gratuites, sans clé API).

L'extension n'apparaît pas sur un site Web

! Solution
  • Live Translator ne fonctionne que sur les plateformes compatibles. L'extension n'apparaîtra pas sur les sites non pris en charge.
  • Si le site est compatible mais que l'extension ne s'affiche pas, essayez d'actualiser la page.
  • Assurez-vous que l'extension est activée dans la page de gestion des extensions de votre navigateur.
