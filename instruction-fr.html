Installation

Installez l'extension Live Translator sur n'importe quel navigateur basé sur Chromium (Chrome, Edge, Brave, Arc, Opera, etc.).

1 Ouvrez le site Web Live Translator

Aller à https://translate.ezitech.io dans votre navigateur.

2 Cliquez sur le bouton "Installer"

Cliquez sur le Ajouter à Chrome bouton . Cela téléchargera le fichier d'extension.

3 Activez le mode développeur dans votre navigateur

Ouvrez la page de gestion des extensions de votre navigateur :

  • Chrome/Brave/Arc : Type chrome://extensions dans la barre d'adresse et appuyez sur Entrée.
  • Bord : Type edge://extensions dans la barre d'adresse et appuyez sur Entrée.

Basculer Mode développeur ON dans le coin supérieur droit de la page.

4 Installer le fichier d'extension

Faites glisser et déposez le fichier téléchargé .zip fichier sur la page des extensions.

Fait !

Vous devriez maintenant voir l'icône Live Translator dans la barre d'outils de votre navigateur. L'extension est installée et prête à être configurée.

Présentation des extensions

Un aperçu rapide de l'interface principale et de l'endroit où tout se trouve.

Le panneau Live Translator comprend les zones principales suivantes :

  • Barre de titre — Bouton Paramètres, bascule du mode Sous-titres, bouton Fermer. Si une mise à jour est disponible, vous verrez un badge ici.
  • Sélecteur de langue — Deux listes déroulantes : « Ma langue » (la langue que vous parlez) et « Leur langue » (la langue à partir de laquelle vous souhaitez traduire).
  • Zone de transcription — Où le texte traduit apparaît en temps réel, organisé par locuteur.
  • Barre de contrôle — Bouton Démarrer/Arrêter/Pause, bascule du microphone, paramètres de narration, assistant IA, taille de police, téléchargement et autres options.
Astuce

Vous pouvez faites glisser le panneau par sa barre de titre pour le déplacer n'importe où sur l'écran, et redimensionner en faisant glisser les bords. De cette façon, il ne bloquera pas le contenu important lors de réunions ou de vidéos.

Obtenir une clé API Soniox

Soniox fournit le moteur de reconnaissance vocale et de traduction en temps réel. Vous avez besoin d'une clé API Soniox pour utiliser Live Translator. Obligatoire

Important

Une clé API Soniox est obligatoire pour que l'extension fonctionne. Sans cela, l’extension ne peut transcrire ou traduire aucun audio.

1 Aller sur le site de Soniox

Ouvrir https://console.soniox.com/signup dans votre navigateur. Créez un compte en utilisant votre adresse e-mail ou votre connexion Google.

2 Obtenez votre clé API

Après vous être connecté, accédez à la section API de votre tableau de bord Soniox. Vous pouvez également vous rendre directement sur Guide de démarrage de Soniox pour des instructions étape par étape.

Copiez votre clé API : vous en aurez besoin dans les prochaines étapes.

3 Ajouter une facturation (obligatoire pour l'utilisation de l'API)

Ajoutez un mode de paiement à votre compte Soniox. Vous pouvez utiliser une carte de crédit, PayPal ou d'autres méthodes de paiement. L'API Soniox est payante : vous ne payez que pour ce que vous utilisez.

Remarque

Vous devez déposer au moins 1 $ sur votre compte pour utiliser l'API. Vous pouvez déposer davantage si vous le souhaitez.

Obtenir une clé API OpenAI (facultatif)

Une clé API OpenAI est facultatif — vous n'en avez besoin que si vous souhaitez utiliser les voix OpenAI pour la narration ou la fonction Inline AI Assistant. Pro

1 Accéder à la plateforme OpenAI

Ouvrir https://platform.openai.com/api-keys dans votre navigateur.

2 Créez un compte ou connectez-vous

Si vous n'avez pas de compte OpenAI, cliquez sur "Inscription" et créez-en un. Si vous en avez déjà un, connectez-vous.

3 Créer une nouvelle clé API

Cliquez "+ Créer une nouvelle clé secrète", donnez-lui un nom comme "Live Translator", et cliquez sur "Créer une clé secrète".

Copiez et enregistrez la clé immédiatement — OpenAI ne l'affichera qu'une seule fois.

4 Ajouter une facturation (obligatoire pour l'utilisation de l'API)

Aller à Paramètres de facturation et ajoutez un mode de paiement. L'API OpenAI est payante : vous ne payez que pour ce que vous utilisez.

Remarque

Si vous souhaitez uniquement une narration (texte-parole), vous pouvez utiliser Voix Google Translate gratuites à la place. La clé OpenAI n'est requise que si vous souhaitez les voix premium d'OpenAI ou la fonctionnalité AI Assistant.

Entrez vos clés API

Une fois que vous avez vos clés API, saisissez-les dans la page des paramètres de Live Translator.

1 Ouvrir les paramètres du traducteur en direct

Vous pouvez ouvrir les paramètres de l'une des manières suivantes :

  • Cliquez sur le icône d'engrenage (⚙) en haut à droite du panneau de traduction.
  • Cliquez avec le bouton droit sur l'icône Live Translator dans votre barre d'outils → "Options".
  • Accédez à la page des extensions de votre navigateur → recherchez Live Translator → "Détails" → "Options d'extension".
2 Entrez votre clé API Soniox

Dans la page des paramètres, cliquez sur "Soniox" dans la barre latérale gauche. Collez votre clé API Soniox dans le champ de saisie. La clé est enregistrée automatiquement.

3 Entrez votre clé API OpenAI (facultatif)

Cliquez sur "OpenAI" dans la barre latérale gauche. Collez votre clé API OpenAI dans le champ de saisie. La clé est enregistrée automatiquement.

Note de sécurité

Vos clés API sont stockées localement sur votre appareil uniquement en utilisant le stockage d'extensions sécurisé de Chrome. Ils ne sont jamais envoyés aux serveurs de Live Translator — ils sont uniquement utilisés pour communiquer directement avec Soniox et OpenAI.

Démarrer une session de traduction

Découvrez comment commencer à traduire lors d'une réunion ou d'une vidéo.

1 Ouvrir un site Web pris en charge

Aller à n'importe quel plateforme prise en charge (par exemple, Google Meet, Zoom, YouTube, etc.) et rejoignez une réunion ou lisez une vidéo.

2 Ouvrez le panneau Live Translator

Cliquez sur le Icône du traducteur en direct dans la barre d'outils de votre navigateur. Le panneau de traduction apparaîtra sur la page. Vous pouvez également cliquer sur le icône flottante qui apparaît sur la page.

3 Sélectionnez vos langues

Utilisez les deux listes déroulantes de langue en haut du panneau :

  • "Ma langue" — La langue vous parler (par exemple, le vietnamien).
  • "Leur langue" — La langue autres parlez que vous souhaitez traduire (par exemple, l'anglais).
4 Appuyez sur le bouton Démarrer

Cliquez sur le Démarrer (icône de lecture) dans la barre de contrôle en bas du panneau.

Votre navigateur peut demander l'autorisation de capturer de l'audio — cliquez sur "Autoriser" et sélectionnez l'onglet que vous souhaitez capturer.

5 Voir les traductions en temps réel

Au fur et à mesure que les gens parlent, vous verrez à la fois le texte original et le texte traduit apparaissent dans la zone de transcription. Les lignes sont automatiquement regroupées par haut-parleur.

Défilement automatique

La transcription défile automatiquement pour afficher le dernier texte. Si vous faites défiler vers le haut pour lire un texte précédent, le défilement automatique s'arrête. Cliquez sur le "Faire défiler jusqu'au dernier" Bouton qui semble revenir au texte en direct.

Contrôles : Pause, Reprise, Fin

Comprendre les commandes de la session de traduction.

BoutonCe qu'il fait
Démarrer (▶)Démarre une nouvelle session de traduction. Efface toute transcription précédente.
Pause (⏸)Arrête temporairement la traduction. La transcription est conservée. La capture audio continue mais le texte est mis en pause.
Reprendre (▶)Continue la traduction après une pause. Un nouveau texte apparaît après le contenu précédent.
Fin (⏹)Arrête complètement la session. La transcription est conservée à l'écran jusqu'à ce que vous démarriez une nouvelle session. Termine la capture audio, désactive le micro et la narration.

Changer de langue

Vous pouvez modifier les langues de traduction à tout moment.

1 Cliquez sur la liste déroulante des langues

Cliquez sur l'un ou l'autre "Ma langue" ou "Leur langue" liste déroulante en haut du panneau.

2 Sélectionner une nouvelle langue

La liste prend en charge 60+ langues triés par ordre alphabétique. Faites défiler ou tapez pour rechercher votre langue. Cliquez pour le sélectionner.

3 La traduction redémarre automatiquement

Si une session est active, le changement de langue relancera automatiquement la traduction avec la nouvelle paire de langues. Votre sélection de langue est enregistrée pour la prochaine fois.

Allumez votre microphone

Activez votre microphone pour que votre propre discours soit également traduit parallèlement à l'audio de la réunion/vidéo.

1 Cliquez sur le bouton du microphone

Dans la barre de contrôle, cliquez sur l'icône icône de microphone. L'icône devient active (en surbrillance) lorsque le micro est allumé.

2 Autoriser l'accès au microphone

Si c'est la première fois, votre navigateur vous demandera l'autorisation du microphone. Cliquez "Autoriser".

Astuce

Le microphone est désactivé par défaut lorsque vous ouvrez le panneau. Activez-le uniquement si vous souhaitez que votre propre discours soit transcrit et traduit. Quand vous appuyez Fin, le microphone s'éteint automatiquement.

Modifier la taille de la police

Ajustez la taille du texte de la transcription pour une lecture confortable.

1 Cliquez sur le bouton de taille de police

Dans la barre de contrôle, cliquez sur l'icône bouton de taille de police (Icône Aa). Un menu apparaît avec trois tailles :

  • Petit  – Vue compacte, peut contenir davantage de texte
  • Moyen — Taille équilibrée par défaut
  • Grand — Plus facile à lire à distance

Cliquez sur votre taille préférée. La modification s'applique immédiatement et est enregistrée pour la prochaine fois.

Noms des intervenants

Attribuez de vrais noms aux orateurs afin de pouvoir facilement identifier qui a dit quoi dans la transcription. Gratuit

1 Ouvrez le menu "Plus"

Dans la barre de contrôle, cliquez sur l'icône "Plus" Bouton (icône ⋯). Puis cliquez "Paramètres des enceintes" bouton .

2 Ouvrir la boîte de dialogue « Paramètres des enceintes »

La boîte de dialogue Paramètres des haut-parleurs s'ouvrira.

3 Attribuer des noms

Vous verrez une liste des haut-parleurs détectés (Haut-parleur 1, Haut-parleur 2, etc.). Tapez un vrai nom à côté de chacun (par exemple, « John », « Sarah »). Cliquez "Ajouter une ligne" si vous avez besoin de plus.

Cliquez "Enregistrer". Toutes les lignes de transcription – passées et futures – utiliseront désormais les noms que vous avez attribués.

Narration (texte-parole)

Faites lire les traductions à haute voix afin que vous puissiez les écouter au lieu de les lire. Il s'agit de la fonctionnalité « Narrer leur voix ». Pro

1 Ouvrir les paramètres de narration

Dans la barre de contrôle, cliquez sur l'icône bouton de narration (icône de haut-parleur). La boîte de dialogue Paramètres de narration s'ouvre.

2 Choisissez un fournisseur TTS et une voix

Sélectionnez votre fournisseur de voix :

  • Google Traduction (Gratuit) — Fonctionne sans aucune clé API. Des voix de bonne qualité dans plus de 60 langues.
  • OpenAI (Premium) — Des voix de meilleure qualité et plus naturelles. Nécessite une clé API OpenAI.

Sélectionnez un voix dans la liste déroulante et cliquez sur "Tester" pour prévisualiser le son.

Ajuster vitesse de la voix et volume si nécessaire.

3 Démarrer la narration

Cliquez « Racontez leur voix » bouton dans la boîte de dialogue. Désormais, chaque fois qu'une traduction apparaîtra, elle vous sera lue à haute voix.

Pour arrêter, cliquez à nouveau sur le bouton de narration, puis cliquez sur "Arrêter".

Raconter ma voix (mode interprète)

Parlez dans votre langue et faites écouter votre voix traduite aux autres participants à la réunion, comme si vous aviez un interprète personnel. Pro

Exigences

Cette fonctionnalité fonctionne sur Google Meet, Zoom (Web), Microsoft Teams (Web), et WhatsApp Web. Vous avez besoin de votre microphone activé et une séance de traduction en cours d'exécution.

1 Ouvrir les paramètres de narration

Cliquez sur le bouton de narration dans la barre de contrôle pour ouvrir la boîte de dialogue Paramètres de narration.

2 Configurer « Narrer ma voix »

Dans la boîte de dialogue, recherchez le « Racontez ma voix » section. Choisissez votre voix et votre fournisseur préférés (Google ou OpenAI). Vous pouvez définir un voix différente pour la narration « Ma voix » et « Leur voix ».

Cliquez "Tester" pour prévisualiser la voix sélectionnée.

3 Démarrer « Narrer ma voix »

Cliquez « Commencer à raconter ma voix ». Lorsque vous parlez, vos mots sont :

  1. Transcrit dans votre langue
  2. Traduit dans l'autre langue
  3. Parlé à voix haute lors de la réunion pour que l'autre personne entende votre voix traduite

Votre microphone est automatiquement coupé pendant la lecture de l'audio traduit pour éviter l'écho.

4 Arrêter la narration

Cliquez sur le bouton de narration puis "Arrêter", ou appuyez sur Fin pour arrêter toute la session (ce qui arrête également la narration).

Remarque

Vous ne pouvez exécuter que un mode de narration à la fois — soit « Narrate My Voice » ou « Narrate Their Voice », pas les deux simultanément. L'icône de la barre d'outils change pour indiquer quel mode est actif.

Envoyer la traduction via le chat

Envoyez automatiquement votre discours traduit sous forme de messages texte dans le chat de la réunion. Pro

1 Activer « Envoyer via Chat »

Dans la barre de contrôle, recherchez le bouton d'envoi de chat (à côté de l'icône du microphone). Cliquez dessus pour l'activer - le bouton s'allume vert lorsqu'il est actif.

2 Parlez et ça envoie automatiquement

Avec votre microphone activé et une session de traduction en cours, lorsque vous parlez, votre texte traduit est automatiquement inséré dans le chat de la réunion et envoyé.

Cela fonctionne sur Google Meet, Zoom (Web), et Microsoft Teams (Web).

Astuce

Ceci est utile lorsque « Narrate My Voice » n'est pas disponible pour votre plate-forme, ou si vous souhaitez simultanément une narration vocale ET une traduction de chat textuel.

Mode sous-titres

Passez à une barre de sous-titres minimale de style film qui affiche uniquement la dernière traduction en bas de l'écran. Pro

1 Activer le mode sous-titres

Dans la barre de titre du panneau de traduction, cliquez sur l'icône icône de sous-titre (icône de sous-titres codés).

Le panneau complet sera remplacé par une fine barre de sous-titres en bas de l'écran.

2 Personnaliser la barre de sous-titres

Cliquez sur le icône de réglage (engrenage de paramètres) sur la barre de sous-titres pour régler :

  • Opacité — Faites glisser pour rendre la barre plus ou moins transparente.
  • Taille de la police — Choisissez parmi un texte petit, moyen ou grand.

Vous pouvez également glisser la barre de sous-titre pour la déplacer, et redimensionner en faisant glisser les bords.

3 Quitter le mode sous-titres

Cliquez sur le bouton de fermeture (✕) sur la barre de sous-titres pour revenir au panneau de traduction complet.

Assistant IA en ligne

Obtenez des explications basées sur l'IA et des réponses générées en fonction du contexte de la conversation, directement à partir de la transcription. Pro

Exigence

L'assistant IA en ligne nécessite un Clé API OpenAI à configurer dans les paramètres.

Expliquez une ligne

1 Survolez une ligne de transcription

Lorsque vous survolez n'importe quelle ligne de la transcription, un Icône IA (🤖) apparaît.

2 Cliquez sur l'icône IA et choisissez "Expliquer"

L'assistant IA s'ouvre dans une fenêtre séparée. Il utilisera le contexte complet de la transcription et la ligne sélectionnée pour vous donner une explication détaillée.

Générer une réponse

1 Cliquez sur l'icône AI et choisissez "Générer la réponse"

L'assistant IA générera une suggestion de réponse basée sur ce qui a été dit lors de la conversation. Il peut même utiliser la recherche sur le Web pour fournir des réponses plus précises.

Ouvrez le panneau Assistant

1 Cliquez sur le bouton Assistant dans la barre de contrôle

Vous pouvez également ouvrir le panneau de l'assistant AI directement depuis le bouton Assistant (icône 🤖) dans la barre de contrôle. Cela ouvre une fenêtre de discussion déplaçable dans laquelle vous pouvez saisir n'importe quelle question. L'assistant a accès à la transcription complète comme contexte.

Paramètres contextuels

Fournissez un contexte sur le sujet de la conversation pour améliorer la précision de la traduction. Gratuit

1 Ouvrir les paramètres contextuels

Cliquez sur le "Plus" Bouton bouton (⋯) dans la barre de contrôle, puis cliquez sur "Contexte".

2 Remplissez les champs contextuels

Vous pouvez fournir :

  • Domaine / Sujet  – Le sujet de la conversation (par exemple, "Conférence médicale", "Standup sur l'ingénierie logicielle").
  • Termes clés — Termes techniques, noms propres ou jargon qui peuvent être difficiles à reconnaître (par exemple, « Kubernetes, passerelle API, microservices »).
  • Informations sur le haut-parleur — Noms et rôles des participants (par exemple, « Dr Smith — cardiologue, présentateur »).

Cliquez "Appliquer" pour envoyer ce contexte au moteur de traduction de la session en cours.

3 Enregistrer les préréglages (facultatif)

Si vous organisez des réunions récurrentes sur des sujets similaires, cliquez sur "Enregistrer le préréglage" pour enregistrer le contexte actuel. La prochaine fois, chargez simplement le préréglage au lieu de tout retaper.

Vous pouvez également Importer/Exporter préréglages sous forme de fichiers JSON pour les partager ou les sauvegarder.

Remplissage automatique YouTube

Lors de l'utilisation de Live Translator sur YouTube, les champs contextuels peuvent être rempli automatiquement avec le titre et la description de la vidéo. Cela aide le moteur de traduction à mieux comprendre le sujet de la vidéo.

Télécharger la transcription

Enregistrez vos transcriptions traduites sous forme de fichiers pour référence ultérieure ou pour utilisation dans d'autres outils. Pro

1 Cliquez sur le bouton Télécharger

Après une session de traduction (ou pendant une), cliquez sur l'icône bouton de téléchargement (icône ↓) dans la barre de contrôle. La boîte de dialogue Télécharger la transcription s'ouvre.

2 Choisissez un format

Sélectionnez votre format préféré :

FormatIdéal pour
TXTFichier texte simple avec horodatages et noms de locuteurs. Facile à lire.
JSONFormat de données structurées. Idéal pour le traitement avec d'autres outils ou scripts.
SRT (original)Fichier de sous-titres avec le texte dans la langue d'origine. Utilisation dans un logiciel de montage vidéo.
SRT (Traduit)Fichier de sous-titres avec le texte traduit. Utilisation dans un logiciel de montage vidéo.
3 Télécharger le fichier

Cliquez sur le bouton de téléchargement correspondant au format choisi. Le fichier est généré dans votre navigateur et enregistré dans votre dossier de téléchargement par défaut.

Confidentialité

Les fichiers de transcription sont générés entièrement dans votre navigateur. Aucune donnée de transcription n'est envoyée à un serveur pendant le processus de téléchargement.

Vérification des mises à jour automatiques

L'extension recherche automatiquement les mises à jour à chaque fois que vous l'ouvrez.

1 Recherchez le badge de mise à jour

Lorsqu'une nouvelle version est disponible, vous verrez un "Mise à jour disponible" badge dans la barre de titre du panneau de traduction.

2 Cliquez pour mettre à jour

Cliquez sur le badge pour accéder à la page d'installation où vous pourrez télécharger la dernière version. Suivez la même chose étapes d'installation à mettre à jour.

Aucun son n'est capturé

! Solution
  • Assurez-vous d'avoir cliqué sur "Autoriser" lorsque le navigateur a demandé l'autorisation de capture audio.
  • Assurez-vous d'avoir sélectionné le onglet correct à partager lorsque vous y êtes invité.
  • Essayez d'actualiser la page et d'ouvrir à nouveau le traducteur.
  • Vérifiez que la réunion/la vidéo comporte réellement une lecture audio.

La traduction semble inexacte

! Solution
  • Assurez-vous que La liste déroulante "Leur langue" est définie sur la langue correcte parlée.
  • Utiliser Paramètres contextuels pour fournir des informations sur le sujet et des termes clés.
  • Assurez-vous que la qualité audio est claire : le bruit de fond peut affecter la précision.

« Narrate My Voice » — les autres n'entendent pas la traduction

! Solution
  • Confirmez que vous êtes sur un plateforme prise en charge (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Assurez-vous qu'une session de traduction est en cours d'exécution et votre le microphone est allumé.
  • Vérifiez que vous disposez d'un Licence Pro activé.
  • Essayez d'actualiser la page de la réunion et de recommencer.

Les fonctionnalités OpenAI ne fonctionnent pas

! Solution
  • Vérifiez votre Clé API OpenAI est saisi correctement dans paramètres.
  • Assurez-vous que votre compte OpenAI a configuration de la facturation avec suffisamment de crédits.
  • Si vous n'avez pas besoin des fonctionnalités OpenAI, utilisez Voix Google Traduction pour la narration à la place (gratuit, aucune clé API requise).

L'extension n'apparaît pas sur un site Web

! Solution
  • Live Translator ne fonctionne que sur plateformes prises en charge. Il n'apparaîtra pas sur les sites Web non pris en charge.
  • Si le site est pris en charge mais que l'extension ne s'affiche pas, essayez actualisation de la page.
  • Assurez-vous que l'extension est activé dans la page de gestion des extensions de votre navigateur.
