Установка

Установите расширение Live Translator в любой браузер на базе Chromium (Chrome, Edge, Brave, Arc, Opera и др.).

1 Установите через Chrome Web Store

Откройте официальную страницу: Live Translator в Chrome Web Store, нажмите Add to Chrome и подтвердите установку через Add extension.

Готово!

Теперь вы увидите значок Live Translator на панели инструментов браузера. Расширение установлено и готово к настройке.

Обзор расширения

Краткий обзор основного интерфейса и расположения элементов.

Панель Live Translator состоит из следующих основных областей:

  • Строка заголовка — кнопка настроек, переключатель режима субтитров, кнопка закрытия. Если доступно обновление, здесь появится соответствующий значок.
  • Выбор языка — два выпадающих списка: «Мой язык» (язык, на котором говорите вы) и «Их язык» (язык, с которого нужно переводить).
  • Область стенограммы — здесь в реальном времени отображается переведённый текст, сгруппированный по говорящим.
  • Панель управления — кнопки «Старт/Стоп/Пауза», переключатель микрофона, настройки озвучивания, AI-помощник, размер шрифта, скачивание и другие опции.
Совет

Вы можете перетащить панель за строку заголовка в любое место на экране, а также изменить её размер, потянув за края. Это позволяет не перекрывать важное содержимое во время встреч или просмотра видео.

Получение API-ключа Soniox

Soniox обеспечивает распознавание речи и перевод в реальном времени. Для работы Live Translator вам понадобится API-ключ Soniox. Обязательно

Важно

API-ключ Soniox обязателен для работы расширения. Без него расширение не сможет распознавать и переводить аудио.

1 Перейдите на сайт Soniox

Откройте https://console.soniox.com/signup в браузере. Создайте учётную запись с помощью электронной почты или входа через Google.

2 Получите API-ключ

После входа перейдите в раздел API на панели управления Soniox. Вы также можете воспользоваться руководством по началу работы Soniox с пошаговыми инструкциями.

Скопируйте API-ключ — он понадобится вам на следующих шагах.

3 Добавьте способ оплаты (обязательно для использования API)

Привяжите способ оплаты к вашей учётной записи Soniox. Можно использовать банковскую карту, PayPal или другие методы оплаты. API Soniox работает по принципу оплаты по мере использования — вы платите только за фактический объём.

Примечание

Для использования API необходимо пополнить счёт минимум на $1. При желании вы можете внести большую сумму.

Получение API-ключа OpenAI (необязательно)

API-ключ OpenAI не обязателен — он нужен только для использования голосов OpenAI при озвучивании или функции встроенного AI-помощника. Pro

1 Перейдите на платформу OpenAI

Откройте https://platform.openai.com/api-keys в браузере.

2 Создайте аккаунт или войдите

Если у вас нет учётной записи OpenAI, нажмите «Sign Up» и создайте её. Если аккаунт уже есть — просто войдите.

3 Создайте новый API-ключ

Нажмите «+ Create new secret key», задайте имя, например «Live Translator», и нажмите «Create secret key».

Сразу скопируйте и сохраните ключ — OpenAI покажет его только один раз.

4 Добавьте способ оплаты (обязательно для использования API)

Перейдите в настройки оплаты и привяжите способ оплаты. API OpenAI работает по принципу оплаты по мере использования — вы платите только за фактический объём.

Примечание

Если вам нужно только озвучивание (текст в речь), вы можете бесплатно использовать голоса Google Translate. API-ключ OpenAI требуется только для премиум-голосов OpenAI или функции AI-помощника.

Ввод API-ключей

Получив API-ключи, введите их на странице настроек Live Translator.

1 Откройте настройки Live Translator

Открыть настройки можно любым из следующих способов:

  • Нажмите на значок шестерёнки (⚙) в правом верхнем углу панели переводчика.
  • Щёлкните правой кнопкой мыши по значку Live Translator на панели инструментов → «Options».
  • Перейдите на страницу расширений в браузере → найдите Live Translator → «Подробнее» → «Параметры расширения».
2 Введите API-ключ Soniox

На странице настроек нажмите «Soniox» в левой боковой панели. Вставьте API-ключ Soniox в поле ввода. Ключ сохранится автоматически.

3 Введите API-ключ OpenAI (необязательно)

Нажмите «OpenAI» в левой боковой панели. Вставьте API-ключ OpenAI в поле ввода. Ключ сохранится автоматически.

Безопасность

Ваши API-ключи хранятся только локально на вашем устройстве в защищённом хранилище расширений Chrome. Они никогда не отправляются на серверы Live Translator — они используются только для прямого взаимодействия с Soniox и OpenAI.

3 Завершите оплату

Вы будете перенаправлены на страницу оплаты Polar. Введите адрес электронной почты и платёжные данные, затем завершите покупку.

После оплаты вы получите лицензионный ключ на электронную почту. Ключ выглядит как длинный код (например, LT-XXXX-XXXX-XXXX-XXXX). Скопируйте его.

Активация лицензии

После покупки активируйте лицензионный ключ в настройках расширения.

1 Введите и активируйте лицензионный ключ

Откройте настройки расширения (значок шестерёнки или правая кнопка мыши → Options). Нажмите «License» в левой боковой панели. Вставьте лицензионный ключ в поле ввода и нажмите «Activate». Расширение проверит вашу лицензию на сервере.

После активации вы увидите статус лицензии, привязанный адрес электронной почты и дату окончания действия. Все Pro-функции теперь разблокированы.

Смена устройства?

Если вы ранее активировали лицензию на другом устройстве, воспользуйтесь функцией «Force Activate», чтобы перенести лицензию на текущее устройство. Каждая лицензия может быть активна только на одном устройстве одновременно.

Запуск сеанса перевода

Узнайте, как начать перевод во время встречи или при просмотре видео.

1 Откройте поддерживаемый сайт

Перейдите на любую поддерживаемую платформу (например, Google Meet, Zoom, YouTube и т. д.) и присоединитесь к встрече или откройте видео.

2 Откройте панель Live Translator

Нажмите на значок Live Translator на панели инструментов браузера. На странице появится панель переводчика. Вы также можете нажать на плавающий значок, который появляется на странице.

3 Выберите языки

Используйте два выпадающих списка языков в верхней части панели:

  • «My language» — язык, на котором говорите вы (например, русский).
  • «Their language» — язык, на котором говорят другие и который нужно переводить (например, английский).
4 Нажмите кнопку «Старт»

Нажмите кнопку Start (значок воспроизведения) на панели управления в нижней части панели.

Браузер может запросить разрешение на захват звука — нажмите «Разрешить» и выберите нужную вкладку.

5 Смотрите перевод в реальном времени

Когда участники говорят, в области стенограммы появляются исходный текст и перевод. Строки автоматически группируются по говорящим.

Автопрокрутка

Стенограмма автоматически прокручивается к последнему тексту. Если вы прокрутите вверх, чтобы прочитать предыдущие строки, автопрокрутка приостановится. Нажмите появившуюся кнопку «Scroll to latest», чтобы вернуться к текущему тексту.

Управление: пауза, продолжение, завершение

Основные элементы управления сеансом перевода.

КнопкаДействие
Start (▶)Начинает новый сеанс перевода. Предыдущая стенограмма очищается.
Pause (⏸)Временно приостанавливает перевод. Стенограмма сохраняется. Захват звука продолжается, но текст не обновляется.
Resume (▶)Возобновляет перевод после паузы. Новый текст появляется после предыдущего содержимого.
End (⏹)Полностью завершает сеанс. Стенограмма остаётся на экране до начала нового сеанса. Захват звука прекращается, микрофон и озвучивание отключаются.

Смена языка

Вы можете изменить язык перевода в любой момент.

1 Нажмите на выпадающий список языков

Нажмите на выпадающий список «My language» или «Their language» в верхней части панели.

2 Выберите новый язык

Список содержит более 60 языков, отсортированных в алфавитном порядке. Прокручивайте или начните вводить название для поиска. Нажмите, чтобы выбрать нужный язык.

3 Перевод перезапустится автоматически

Если сеанс активен, смена языка автоматически перезапустит перевод с новой языковой парой. Ваш выбор языка сохраняется для следующих сеансов.

Включение микрофона

Включите микрофон, чтобы ваша речь также распознавалась и переводилась наряду со звуком встречи или видео.

1 Нажмите кнопку микрофона

На панели управления нажмите значок микрофона. Значок подсвечивается, когда микрофон включён.

2 Разрешите доступ к микрофону

При первом использовании браузер запросит разрешение на доступ к микрофону. Нажмите «Разрешить».

Совет

По умолчанию микрофон выключен при открытии панели. Включайте его только в том случае, если хотите, чтобы ваша речь распознавалась и переводилась. При нажатии End микрофон автоматически отключается.

Изменение размера шрифта

Настройте размер текста стенограммы для комфортного чтения.

1 Нажмите кнопку размера шрифта

На панели управления нажмите кнопку размера шрифта (значок Aa). Появится меню с тремя вариантами:

  • Small — компактный вид, больше текста на экране.
  • Medium — сбалансированный размер по умолчанию.
  • Large — крупный текст, удобен для чтения на расстоянии.

Выберите нужный размер. Изменение применяется сразу и сохраняется на будущее.

Имена говорящих

Назначьте настоящие имена говорящим, чтобы легко определять, кто что сказал в стенограмме. Бесплатно

1 Откройте меню «Ещё»

На панели управления нажмите кнопку «More» (значок ⋯). Затем нажмите «Speaker Settings».

2 Откройте диалог «Speaker Settings»

Откроется диалоговое окно настроек говорящих.

3 Присвойте имена

Вы увидите список обнаруженных говорящих (Speaker 1, Speaker 2 и т. д.). Введите настоящее имя напротив каждого (например, «Иван», «Мария»). Нажмите «Add row», если нужно добавить строки.

Нажмите «Save». Все строки стенограммы — как уже существующие, так и новые — будут отображаться с указанными именами.

Озвучивание (текст в речь)

Слушайте перевод вслух вместо чтения. Это функция «Narrate Their Voice» — озвучивание голоса собеседника. Pro

1 Откройте настройки озвучивания

На панели управления нажмите кнопку озвучивания (значок динамика). Откроется диалоговое окно настроек озвучивания.

2 Выберите провайдера и голос

Выберите провайдера озвучивания:

  • Google Translate (бесплатно) — работает без API-ключа. Качественные голоса на 60+ языках.
  • OpenAI (премиум) — более высокое качество и естественное звучание. Требуется API-ключ OpenAI.

Выберите голос из выпадающего списка и нажмите «Test», чтобы прослушать его.

При необходимости отрегулируйте скорость и громкость голоса.

3 Запустите озвучивание

Нажмите кнопку «Narrate Their Voice» в диалоговом окне. Теперь при появлении каждого нового перевода он будет озвучен вслух.

Чтобы остановить, снова нажмите кнопку озвучивания и выберите «Stop».

Озвучивание по субтитрам YouTube

Используйте переведённые субтитры YouTube как источник для озвучивания, чтобы расширение читало на языке, который вы уже выбрали в YouTube. Эта функция может снизить расходы Soniox на YouTube до 99,9%. Pro

Важный порядок

Чтобы функция работала корректно, сначала выберите язык субтитров в YouTube, а затем запускайте озвучивание в расширении.

1 Сначала настройте переведённые субтитры в YouTube

В плеере YouTube включите субтитры и выберите язык перевода, который хотите слышать (например: вьетнамский).

2 Включите автоопределение в настройках озвучивания

Откройте настройки озвучивания в расширении и включите Auto detect translated caption.

3 Запустите озвучивание

Нажмите «Narrate Their Voice». Озвучивание будет идти на языке переведённых субтитров, выбранном в YouTube.

Пример

Если вам нужно озвучивание на вьетнамском, сначала переключите субтитры YouTube на вьетнамский. Затем включите Auto detect translated caption и запустите «Narrate Their Voice».

Озвучить мой голос (режим переводчика)

Говорите на своём языке — ваш переведённый голос будет воспроизведён для других участников встречи, как если бы у вас был личный переводчик. Pro

Требования

Эта функция работает в Google Meet, Zoom (Web), Microsoft Teams (Web) и WhatsApp Web. Необходимо, чтобы микрофон был включён, а сеанс перевода запущен.

1 Откройте настройки озвучивания

Нажмите кнопку озвучивания на панели управления, чтобы открыть диалоговое окно настроек.

2 Настройте «Narrate My Voice»

В диалоговом окне найдите раздел «Narrate My Voice». Выберите предпочитаемый голос и провайдера (Google или OpenAI). Для «My voice» и «Their voice» можно задать разные голоса.

Нажмите «Test», чтобы прослушать выбранный голос.

3 Запустите «Narrate My Voice»

Нажмите «Start Narrate My Voice». Когда вы говорите, ваши слова:

  1. Распознаются на вашем языке
  2. Переводятся на другой язык
  3. Озвучиваются на встрече, и собеседник слышит переведённую версию вашей речи

Пока воспроизводится переведённый звук, ваш микрофон автоматически отключается для предотвращения эха.

4 Остановите озвучивание

Нажмите кнопку озвучивания и выберите «Stop», или нажмите End, чтобы полностью завершить сеанс (озвучивание также прекратится).

Примечание

Одновременно может работать только один режим озвучивания — либо «Narrate My Voice», либо «Narrate Their Voice». Значок на панели инструментов показывает, какой режим сейчас активен.

Отправка перевода в чат

Автоматически отправляйте переведённую речь текстовыми сообщениями в чат встречи. Pro

1 Включите «Send via Chat»

На панели управления найдите кнопку отправки в чат (рядом со значком микрофона). Нажмите на неё — при активации кнопка станет зелёной.

2 Говорите — текст отправляется автоматически

При включённом микрофоне и активном сеансе перевода ваша переведённая речь автоматически вставляется в поле чата встречи и отправляется.

Функция работает в Google Meet, Zoom (Web) и Microsoft Teams (Web).

Совет

Эта функция полезна, когда «Narrate My Voice» недоступна на вашей платформе, или если вы хотите одновременно использовать и голосовое озвучивание, и текстовый перевод в чате.

Режим субтитров

Переключитесь на минималистичную панель субтитров в стиле кино — она отображает только последний перевод в нижней части экрана. Pro

1 Включите режим субтитров

В строке заголовка панели переводчика нажмите значок субтитров (значок скрытых субтитров).

Полная панель сменится узкой полосой субтитров в нижней части экрана.

2 Настройте панель субтитров

Нажмите значок настроек (шестерёнка) на панели субтитров, чтобы изменить:

  • Прозрачность — ползунок для настройки степени прозрачности панели.
  • Размер шрифта — мелкий, средний или крупный текст.

Панель субтитров можно перетаскивать и изменять по размеру, потянув за края.

3 Выход из режима субтитров

Нажмите кнопку закрытия (✕) на панели субтитров, чтобы вернуться к полной панели переводчика.

Встроенный AI-помощник

Получайте объяснения и подготовленные ответы на основе контекста разговора прямо из стенограммы. Pro

Требование

Для работы встроенного AI-помощника необходимо указать API-ключ OpenAI в настройках.

Объяснение строки

1 Наведите курсор на строку стенограммы

При наведении курсора на любую строку стенограммы появится значок AI (🤖).

2 Нажмите значок AI и выберите «Explain»

AI-помощник откроется в отдельном окне. Он проанализирует полный контекст стенограммы и выбранную строку, чтобы дать вам подробное объяснение.

Генерация ответа

1 Нажмите значок AI и выберите «Generate Answer»

AI-помощник сформирует предложенный ответ на основе сказанного в разговоре. Он также может использовать веб-поиск для более точных ответов.

Панель помощника

1 Нажмите кнопку помощника на панели управления

Вы также можете открыть панель AI-помощника, нажав кнопку помощника (значок 🤖) на панели управления. Откроется перемещаемое окно чата, в котором можно задать любой вопрос. Помощник имеет доступ к полной стенограмме в качестве контекста.

Настройки контекста

Укажите тему разговора, чтобы повысить точность перевода. Бесплатно

1 Откройте настройки контекста

Нажмите кнопку «More» (⋯) на панели управления, затем выберите «Context».

2 Заполните поля контекста

Вы можете указать:

  • Тема / Область — о чём идёт разговор (например, «Медицинская конференция», «Стендап по разработке»).
  • Ключевые термины — специальные термины, имена собственные или профессиональный жаргон, которые могут быть сложны для распознавания (например, «Kubernetes, API gateway, микросервисы»).
  • Информация о говорящих — имена и роли участников (например, «Доктор Иванов — кардиолог, докладчик»).

Нажмите «Apply», чтобы передать этот контекст системе перевода для текущего сеанса.

3 Сохранение пресетов (необязательно)

Если у вас регулярно проходят встречи на схожие темы, нажмите «Save Preset», чтобы сохранить текущий контекст. В следующий раз просто загрузите пресет, не вводя всё заново.

Пресеты можно импортировать и экспортировать в формате JSON для обмена или резервного копирования.

Автозаполнение на YouTube

При использовании Live Translator на YouTube поля контекста могут быть заполнены автоматически на основе названия и описания видео. Это помогает системе перевода лучше понять тему.

Скачать стенограмму

Сохраняйте переведённые стенограммы в файлы для дальнейшего использования или обработки в других инструментах. Pro

1 Нажмите кнопку скачивания

После завершения сеанса перевода (или во время него) нажмите кнопку скачивания (значок ↓) на панели управления. Откроется диалоговое окно скачивания стенограммы.

2 Выберите формат

Выберите нужный формат:

ФорматДля чего подходит
TXTПростой текстовый файл с временными метками и именами говорящих. Легко читать.
JSONСтруктурированный формат данных. Удобен для обработки скриптами и другими инструментами.
SRT (оригинал)Файл субтитров с текстом на языке оригинала. Для использования в видеоредакторах.
SRT (перевод)Файл субтитров с переведённым текстом. Для использования в видеоредакторах.
3 Скачайте файл

Нажмите кнопку скачивания для выбранного формата. Файл создаётся прямо в вашем браузере и сохраняется в папку загрузок по умолчанию.

Конфиденциальность

Файлы стенограмм создаются полностью в вашем браузере. Никакие данные стенограммы не передаются на сервер при скачивании.

Автоматическая проверка обновлений

Расширение автоматически проверяет наличие обновлений при каждом открытии.

1 Обратите внимание на значок обновления

Когда доступна новая версия, в строке заголовка панели переводчика появится значок «Update available».

2 Нажмите для обновления

Нажмите на значок, чтобы открыть страницу Chrome Web Store и проверить актуальную версию. Расширения, установленные из Store, обновляются автоматически после публикации новой версии.

Устранение неполадок

Распространённые проблемы и способы их решения.

Сообщение «No Soniox API key»

! Решение

Вы ещё не ввели API-ключ Soniox. Перейдите в раздел Ввод API-ключей, чтобы настроить его.

Звук не захватывается

! Решение
  • Убедитесь, что вы нажали «Разрешить», когда браузер запросил разрешение на захват звука.
  • Убедитесь, что вы выбрали правильную вкладку при запросе.
  • Попробуйте обновить страницу и снова открыть переводчик.
  • Проверьте, что на встрече или в видео действительно воспроизводится звук.

Перевод неточный

! Решение
  • Убедитесь, что в выпадающем списке «Their language» выбран правильный язык, на котором ведётся речь.
  • Воспользуйтесь настройками контекста, чтобы указать тему и ключевые термины.
  • Убедитесь, что качество звука достаточное — фоновый шум может снижать точность.

«Narrate My Voice» — собеседники не слышат перевод

! Решение
  • Убедитесь, что вы находитесь на поддерживаемой платформе (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Убедитесь, что сеанс перевода запущен и микрофон включён.
  • Проверьте, что Pro-лицензия активирована.
  • Попробуйте обновить страницу встречи и начать заново.

Функции OpenAI не работают

! Решение
  • Убедитесь, что API-ключ OpenAI правильно введён в настройках.
  • Проверьте, что в вашем аккаунте OpenAI настроена оплата и на счету достаточно средств.
  • Если вам не нужны функции OpenAI, используйте для озвучивания голоса Google Translate (бесплатно, API-ключ не требуется).

Расширение не появляется на сайте

! Решение
  • Live Translator работает только на поддерживаемых платформах. На неподдерживаемых сайтах расширение не отображается.
  • Если сайт поддерживается, но расширение не появляется, попробуйте обновить страницу.
  • Убедитесь, что расширение включено на странице управления расширениями вашего браузера.
