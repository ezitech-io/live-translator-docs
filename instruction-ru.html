Установка

Установите расширение Live Translator в любой браузер на базе Chromium (Chrome, Edge, Brave, Arc, Opera и т. д.).

1 Откройте сайт Live Translator

Перейти https://translate.ezitech.io в вашем браузере.

2 Нажмите кнопку «Установить».

Нажмите кнопку Добавить в Chrome Кнопка . Это позволит загрузить файл расширения.

3 Включите режим разработчика в вашем браузере

Откройте страницу управления расширениями вашего браузера:

  • Хром/Храбрый/Арка: Тип chrome://extensions в адресной строке и нажмите Enter.
  • Край: Тип edge://extensions в адресной строке и нажмите Enter.

Переключить Режим разработчика ON в правом верхнем углу страницы.

4 Установите файл расширения

Перетащите загруженный .zip файл на страницу расширений.

Готово!

Теперь вы должны увидеть значок Live Translator на панели инструментов браузера. Расширение установлено и готово к настройке.

Обзор расширения

Краткий обзор основного интерфейса и того, где все находится.

Панель Live Translator имеет следующие основные области:

  • Строка заголовка — кнопка «Настройки», переключатель режима субтитров, кнопка «Закрыть». Если доступно обновление, здесь вы увидите значок.
  • Выбор языка — Два раскрывающихся списка: «Мой язык» (язык, на котором вы говорите) и «Их язык» (язык, с которого вы хотите перевести).
  • Область стенограммы — Здесь в реальном времени отображается переведенный текст, упорядоченный по докладчику.
  • Панель управления — кнопка «Пуск/Стоп/Пауза», переключение микрофона, настройки комментирования, AI-помощник, размер шрифта, загрузка и другие параметры.
Совет

Вы можете перетащите панель по строке заголовка, чтобы переместить его в любое место экрана, и измените размер перетаскивая края. Таким образом, он не будет блокировать важный контент во время встреч или видео.

Получите ключ API Soniox

Soniox предоставляет механизм распознавания и перевода речи в реальном времени. Для использования Live Translator вам понадобится ключ API Soniox. Обязательно

Важно

Ключ API Soniox обязательно чтобы расширение работало. Без него расширение не сможет транскрибировать или переводить аудио.

1 Перейти на сайт Soniox

Открыть https://console.soniox.com/signup в вашем браузере. Создайте учетную запись, используя свою электронную почту или вход в Google.

2 Получите ключ API

После входа в систему перейдите в раздел API на панели инструментов Soniox. Вы также можете перейти непосредственно к Руководство по началу работы с Soniox пошаговые инструкции.

Скопируйте ключ API — он понадобится вам на следующих шагах.

3 Добавить биллинг (требуется для использования API)

Добавьте способ оплаты в свою учетную запись Soniox. Вы можете использовать кредитную карту, PayPal или другие способы оплаты. API Soniox оплачивается по мере использования — вы платите только за то, что используете.

Примечание

Чтобы использовать API, вам необходимо внести на свой счет минимум 1 доллар США. Если хотите, можете внести больше.

Получите ключ API OpenAI (необязательно)

Ключ API OpenAI необязательно — он понадобится вам только в том случае, если вы хотите использовать голоса OpenAI для повествования или функцию встроенного AI Assistant. Профи

1 Перейти на платформу OpenAI

Открыть https://platform.openai.com/api-keys в вашем браузере.

2 Создайте аккаунт или войдите в систему

Если у вас нет учетной записи OpenAI, нажмите "Зарегистрироваться" и создайте его. Если он у вас уже есть, войдите в систему.

3 Создайте новый ключ API

Нажмите "+ Создать новый секретный ключ", дайте ему имя, например «Живой переводчик», и нажмите "Создать секретный ключ".

Немедленно скопируйте и сохраните ключ — OpenAI покажет его только один раз.

4 Добавить биллинг (требуется для использования API)

Перейти Платежные настройки и добавьте способ оплаты. API OpenAI оплачивается по мере использования — вы платите только за то, что используете.

Примечание

Если вам нужно только повествование (преобразование текста в речь), вы можете использовать Голоса Google Translate бесплатно вместо этого. Ключ OpenAI требуется только в том случае, если вам нужны премиум-голоса OpenAI или функция AI Assistant.

Введите свои ключи API

После получения ключей API введите их на странице настроек Live Translator.

1 Откройте настройки Live Translator

Открыть настройки можно любым из этих способов:

  • Нажмите кнопку значок шестеренки (⚙) в правом верхнем углу панели переводчика.
  • Щелкните правой кнопкой мыши значок Live Translator на панели инструментов → "Опции".
  • Перейдите на страницу расширений вашего браузера → найдите Live Translator → "Подробности" → "Параметры расширения".
2 Введите ключ API Soniox

На странице настроек нажмите на "Сониокс" на левой боковой панели. Вставьте свой ключ API Soniox в поле ввода. Ключ сохраняется автоматически.

3 Введите ключ API OpenAI (необязательно)

Нажмите "Открытый AI" на левой боковой панели. Вставьте ключ API OpenAI в поле ввода. Ключ сохраняется автоматически.

Примечание по безопасности

Ваши ключи API хранятся только локально на вашем устройстве используя безопасное хранилище расширений Chrome. Они никогда не отправляются на серверы Live Translator — они используются только для прямого взаимодействия с Soniox и OpenAI.

Начать сеанс перевода

Узнайте, как начать переводить во время встречи или видео.

1 Откройте поддерживаемый веб-сайт

Перейти к любому поддерживаемая платформа (например, Google Meet, Zoom, YouTube и т. д.) и присоединитесь к собранию или воспроизведите видео.

2 Откройте панель Live Translator

Нажмите кнопку Значок Live Translator на панели инструментов вашего браузера. На странице появится панель переводчика. Вы также можете нажать кнопку плавающий значок , который появляется на странице.

3 Выберите языки

Используйте два раскрывающихся списка языков в верхней части панели:

  • "Мой язык" — Язык ты говорить (например, по-вьетнамски).
  • "Их язык" — Язык другие скажите, что вы хотите перевести (например, английский).
4 Нажмите кнопку «Пуск».

Нажмите кнопку Начать (значок воспроизведения) на панели управления в нижней части панели.

Ваш браузер может запросить разрешение на запись звука — нажмите "Разрешить" и выберите вкладку, которую хотите захватить.

5 Просматривайте переводы в режиме реального времени

Пока люди говорят, вы увидите как исходный текст и переведенный текст появляются в области расшифровки. Линии автоматически группируются по говорящим.

Автопрокрутка

Стенограмма автоматически прокручивается и показывает последний текст. Если вы прокрутите вверх, чтобы прочитать предыдущий текст, автоматическая прокрутка приостановится. Нажмите кнопку "Перейти к последней версии" кнопка, которая позволяет вернуться к живому тексту.

Элементы управления: Пауза, Возобновление, Конец

Понимание элементов управления сеансом перевода.

КнопкаЧто он делает
Начать (▶)Начинает новый сеанс перевода. Очищает любую предыдущую расшифровку.
Пауза (⏸)Временно останавливает трансляцию. Стенограмма сохраняется. Захват звука продолжается, но текст приостанавливается.
Резюме (▶)Продолжает перевод после паузы. Новый текст появляется после предыдущего содержимого.
Конец (⏹)Полностью останавливает сеанс. Транскрипт сохраняется на экране до тех пор, пока вы не начнете новый сеанс. Завершает захват звука, отключает микрофон и дикторство.

Изменить язык

Вы можете изменить язык перевода в любое время.

1 Нажмите на раскрывающийся список языков.

Нажмите на значок "Мой язык" или "Их язык" раскрывающийся список в верхней части панели.

2 Выберите новый язык

Список поддерживает более 60 языков отсортировано в алфавитном порядке. Прокрутите или введите текст для поиска вашего языка. Нажмите, чтобы выбрать его.

3 Перевод автоматически возобновляется

Если сеанс активен, изменение языка автоматически перезапустит перевод с новой языковой парой. Выбор языка будет сохранен для следующего раза.

Включите микрофон

Включите микрофон, чтобы ваша речь также переводилась вместе со звуком встречи/видео.

1 Нажмите кнопку микрофона.

На панели управления нажмите кнопку значок микрофона. Значок становится активным (подсвечивается), когда микрофон включен.

2 Разрешить доступ к микрофону

Если это первый раз, ваш браузер запросит разрешение на доступ к микрофону. Нажмите "Разрешить".

Совет

Микрофон выключено по умолчанию при открытии панели. Включайте его только в том случае, если хотите, чтобы ваша собственная речь была расшифрована и переведена. Когда вы нажимаете Конец микрофон автоматически выключается.

Изменить размер шрифта

Отрегулируйте размер текста расшифровки для удобства чтения.

1 Нажмите кнопку размера шрифта.

На панели управления нажмите кнопку кнопка размера шрифта (Значок Aa). Появится меню трех размеров:

  • Маленький — компактный вид, помещается больше текста.
  • Средний — сбалансированный размер по умолчанию.
  • Большой — легче читать на расстоянии.

Выберите нужный размер. Изменение вступает в силу немедленно и сохраняется для следующего раза.

Имена докладчиков

Назначайте выступающим настоящие имена, чтобы вы могли легко определить, кто что сказал в стенограмме. Бесплатно

1 Откройте меню «Еще»

На панели управления нажмите кнопку "Ещё" Кнопка (значок ⋯). Затем нажмите "Настройки динамика" Кнопка .

2 Откройте диалоговое окно «Настройки динамика».

Откроется диалоговое окно «Настройки динамиков».

3 Присвоение имен

Вы увидите список обнаруженных динамиков (Динамик 1, Динамик 2 и т. д.). Рядом с каждым введите настоящее имя (например, «Джон», «Сара»). Нажмите "Добавить строку" если вам нужно больше.

Нажмите "Сохранить". Все строки расшифровки — прошлые и будущие — теперь будут использовать назначенные вами имена.

Повествование (преобразование текста в речь)

Пусть переводы прочитаются вам вслух, чтобы вы могли слушать, а не читать. Это функция «Рассказать их голос». Профи

1 Открыть настройки дикторского текста

На панели управления нажмите кнопку кнопка повествования (значок динамика). Откроется диалоговое окно «Настройки повествования».

2 Выберите провайдера TTS и голосовую связь

Выберите своего провайдера голосовой связи:

  • Google Переводчик (Бесплатно) — Работает без ключа API. Голоса хорошего качества на более чем 60 языках.
  • ОткрытьAI (Премиум) — Более высокое качество, более естественные голоса. Требуется ключ API OpenAI.

Выберите голос из раскрывающегося списка и нажмите "Тест" , чтобы просмотреть, как это звучит.

Отрегулировать скорость голоса и объем при необходимости.

3 Начать повествование

Нажмите "Расскажи их голос" Кнопка в диалоговом окне. Теперь всякий раз, когда появится перевод, он будет прочитан вам вслух.

Чтобы остановить, нажмите кнопку повествования еще раз и нажмите "Стоп".

Рассказывайте моим голосом (режим переводчика)

Говорите на своем языке, и ваш переведенный голос будет воспроизводиться для других участников встречи — как будто у вас есть личный переводчик. Профи

Требования

Эта функция работает на Google Meet, Масштаб (Интернет), Команды Microsoft (Интернет) и Веб-сайт WhatsApp. Вам нужен ваш микрофон включен и сеанс перевода бег.

1 Открыть настройки дикторского текста

Нажмите кнопку кнопка повествования на панели управления, чтобы открыть диалоговое окно «Настройки дикторского текста».

2 Настройте «Рассказывайте моим голосом»

В диалоговом окне найдите "Расскажи мой голос" раздел. Выберите предпочитаемый голос и провайдера (Google или OpenAI). Вы можете установить другой голос за повествование «Мой голос» и «Их голос».

Нажмите "Тест" для предварительного просмотра выбранного голоса.

3 Начать «Рассказать моим голосом»

Нажмите "Начни озвучивать мой голос". Когда вы говорите, ваши слова:

  1. Расшифровано на вашем языке
  2. Переведено на другой язык
  3. Говорите вслух на собрании, чтобы собеседник слышал ваш переведенный голос

Во время воспроизведения переведенного звука ваш микрофон автоматически отключается, чтобы предотвратить эхо.

4 Остановить повествование

Нажмите кнопку повествования, а затем "Стоп" или нажмите Конец , чтобы остановить весь сеанс (что также останавливает повествование).

Примечание

Вы можете запускать только по одному режиму повествования за раз — либо «Расскажи мой голос», либо «Расскажи их голос», но не оба одновременно. Значок на панели инструментов изменится, показывая, какой режим активен.

Отправить перевод через чат

Автоматически отправляйте переведенную речь в виде текстовых сообщений в чат собрания. Профи

1 Включите «Отправить через чат»

На панели управления найдите значок кнопка отправки в чат (рядом со значком микрофона). Нажмите на нее, чтобы включить — кнопка повернется зеленый когда активно.

2 Говорите, и оно отправится автоматически

Если у вас включен микрофон и запущен сеанс перевода, когда вы говорите, переведенный текст автоматически вставляется во входные данные чата собрания и отправляется.

Это работает на Google Meet, Масштаб (Интернет) и Команды Microsoft (Интернет).

Совет

Это полезно, когда функция «Рассказывайте моим голосом» недоступна для вашей платформы или если вы хотите одновременно и голосовое повествование, и перевод текстового чата.

Режим субтитров

Переключитесь на минимальную панель субтитров в стиле кино, которая отображает только последний перевод в нижней части экрана. Профи

1 Включить режим субтитров

В заголовке панели переводчика нажмите кнопку значок субтитров (значок субтитров).

Полная панель будет заменена тонкой полосой субтитров в нижней части экрана.

2 Настройка панели субтитров

Нажмите кнопку значок настройки (шестеренка настроек) в строке субтитров, чтобы настроить:

  • Непрозрачность — сдвиньте ползунок, чтобы сделать панель более или менее прозрачной.
  • Размер шрифта — выберите мелкий, средний или крупный текст.

Вы также можете перетащите панель субтитров, чтобы переместить ее, и изменить размер его, перетаскивая края.

3 Выйти из режима субтитров

Нажмите кнопку кнопка закрытия (✕) на панели субтитров, чтобы вернуться к полной панели переводчика.

Встроенный AI-помощник

Получайте объяснения с помощью искусственного интеллекта и генерируйте ответы на основе контекста разговора прямо из стенограммы. Профи

Требование

Для работы встроенного AI Assistant требуется Ключ OpenAI API настраивается в настройках.

Объясните строку

1 Наведите курсор на строку расшифровки

При наведении курсора на любую строку стенограммы отображается Значок AI Появляется (🤖).

2 Нажмите значок AI и выберите «Объяснить».

ИИ-помощник открывается в отдельном окне. Он будет использовать полный контекст расшифровки и выбранную строку, чтобы дать вам подробное объяснение.

Сгенерировать ответ

1 Нажмите значок AI и выберите «Создать ответ».

ИИ-помощник сгенерирует предложенный ответ на основе того, что было сказано в разговоре. Он даже может использовать веб-поиск для предоставления более точных ответов.

Откройте панель Ассистента

1 Нажмите кнопку «Ассистент» на панели управления.

Вы также можете открыть панель AI-помощника прямо из кнопка помощника (значок 🤖) на панели управления. Откроется перетаскиваемое окно чата, где вы можете ввести любой вопрос. Помощник имеет доступ к полной стенограмме в качестве контекста.

Настройки контекста

Предоставьте контекст темы разговора, чтобы повысить точность перевода. Бесплатно

1 Открыть настройки контекста

Нажмите кнопку "Ещё" Кнопка кнопку (⋯) на панели управления, затем нажмите "Контекст".

2 Заполните поля контекста

Вы можете предоставить:

  • Домен/Тема — о чем разговор (например, «Медицинская конференция», «Стендап по разработке программного обеспечения»).
  • Ключевые термины — технические термины, имена собственные или жаргонизмы, которые может быть трудно распознать (например, «Kubernetes, шлюз API, микросервисы»).
  • Информация о докладчике — имена и роли участников (например, «Доктор Смит — кардиолог, ведущий»).

Нажмите "Применить" чтобы отправить этот контекст в систему перевода для текущего сеанса.

3 Сохранить пресеты (необязательно)

Если у вас есть повторяющиеся встречи на схожие темы, нажмите "Сохранить предустановку" для сохранения текущего контекста. В следующий раз просто загрузите пресет вместо того, чтобы вводить все заново.

Вы также можете Импорт/Экспорт предустановлены как файлы JSON для совместного использования или резервного копирования.

Автозаполнение YouTube

При использовании Live Translator на YouTube, поля контекста могут быть автоматически заполняется с названием и описанием видео. Это помогает системе перевода лучше понять тему видео.

Загрузить стенограмму

Сохраните переведенные расшифровки в виде файлов для дальнейшего использования или использования в других инструментах. Профи

1 Нажмите кнопку «Загрузить».

После сеанса перевода (или во время него) нажмите кнопку кнопка загрузки (значок ↓) на панели управления. Откроется диалоговое окно «Загрузить стенограмму».

2 Выберите формат

Выберите предпочитаемый формат:

ФорматЛучше всего подходит для
TXTПростой текстовый файл с временными метками и именами докладчиков. Легко читать.
JSONФормат структурированных данных. Подходит для обработки другими инструментами или скриптами.
SRT (оригинал)Файл субтитров с текстом на языке оригинала. Используйте в программах для редактирования видео.
SRT (переведено)Файл субтитров с переведенным текстом. Используйте в программах для редактирования видео.
3 Загрузить файл

Нажмите кнопку загрузки выбранного формата. Файл создается в вашем браузере и сохраняется в папке загрузки по умолчанию.

Конфиденциальность

Файлы транскриптов генерируются полностью в вашем браузере. Во время процесса загрузки данные расшифровки не отправляются ни на один сервер.

Автоматическая проверка обновлений

Расширение автоматически проверяет наличие обновлений каждый раз, когда вы его открываете.

1 Найдите значок обновления

Когда будет доступна новая версия, вы увидите "Доступно обновление" значок в заголовке панели переводчика.

2 Нажмите, чтобы обновить

Нажмите значок, чтобы перейти на страницу установки, где можно загрузить последнюю версию. Следуй тому же этапы установки обновить.

Устранение неполадок

Распространенные проблемы и способы их устранения.

Сообщение «Нет ключа API Soniox»

! Решение

Вы еще не ввели свой ключ API Soniox. Перейти к Введите свои ключи API , чтобы настроить его.

Аудио не записывается

! Решение
  • Убедитесь, что вы нажали "Разрешить" когда браузер запросил разрешение на захват звука.
  • Убедитесь, что вы выбрали правильная вкладка чтобы поделиться, когда будет предложено.
  • Попробуйте обновить страницу и снова открыть переводчик.
  • Убедитесь, что на встрече/видео действительно воспроизводится звук.

Перевод кажется неточным

! Решение
  • Убедитесь, что В раскрывающемся списке "Их язык" указан правильный язык общения.
  • Использовать Настройки контекста для предоставления информации по теме и ключевых терминов.
  • Убедитесь, что качество звука четкое — фоновый шум может повлиять на точность.

"Расскажи моим голосом" — другие не слышат перевод

! Решение
  • Подтвердите, что вы находитесь на поддерживаемая платформа (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Убедитесь, что сеанс перевода бег и ваш микрофон включен.
  • Проверьте, есть ли у вас Профессиональная лицензия активирован.
  • Попробуйте обновить страницу собрания и начать заново.

Функции OpenAI не работают

! Решение
  • Подтвердите свой Ключ OpenAI API введено правильно в Настройки.
  • Убедитесь, что в вашей учетной записи OpenAI настроены платежные данные с достаточным количеством кредитов.
  • Если вам не нужны функции OpenAI, используйте Голоса в Google Translate Вместо этого для повествования (бесплатно, ключ API не требуется).

Расширение не отображается на веб-сайте

! Решение
  • Живой переводчик работает только поддерживаемые платформы. Он не будет отображаться на неподдерживаемых веб-сайтах.
  • Если сайт поддерживается, но расширение не отображается, попробуйте обновление страницы.
  • Убедитесь, что расширение включено на странице управления расширениями вашего браузера.
