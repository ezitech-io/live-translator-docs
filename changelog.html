Changelog


v0.14.7 (April 25, 2026)

What's new for you

  • OpenAI warning is now shown only when needed — The extension no longer warns for missing OpenAI key at session start. You now see the warning only when using OpenAI-dependent actions (Narration with OpenAI voice, AI Assistant, and AI downloads).
  • Cleaner state when YouTube caption narration is active — When narration switches to YouTube translated captions and Soniox is intentionally paused, the status now stays in Paused instead of showing a false Request timeout error.
  • Close button now fully shuts down active voice features — Pressing the X closes the translator session and correctly stops active narration flows, including Narrate My Voice runtime state.
  • Immediate original-volume recovery on close — If narration had ducked the original video volume, the extension now restores the original level before closing the translator panel.

v0.14.3 (April 23, 2026)

What's new for you

  • Interpreter mode now works on Microsoft Teams chat — Teams CKEditor injection was upgraded so your translated speech can be inserted and sent reliably instead of failing with an empty-message warning.
  • Narrate My Voice is now audible on Teams meetings — Teams voice routing was fixed so remote participants can hear your translated voice in real time.
  • Clear guidance when Teams chat panel is closed — If Interpreter cannot send because Teams chat is not open, the popup now shows a user-facing warning instead of a console-only failure.
  • New AI export: Download meeting summary — From the Download dialog, you can now generate and save a meeting summary as a .txt file.
  • New AI export: Download task breakdown — You can now export action items with assignee details as .txt, and customize both summary/task prompts in OpenAI settings.

v0.13.1 (April 20, 2026)

What's new for you

  • Browser local voice provider for Narrate their voice — You can now choose a Browser TTS provider and use local/browser-installed voices directly for speaker playback.
  • Narrate My Voice keeps streaming injection path — My voice narration remains on OpenAI/Google streaming providers to preserve virtual mic injection reliability.
  • Fixed long-queue sentence cutting — Browser narration queue handling was hardened to prevent skipped/truncated middle sentences in dense conversations.
  • Reduced browser narration delay — Browser TTS batching/coalescing and provider-specific timing were tuned to lower repeated startup gaps between short segments.
  • Faster and steadier speed behavior — Browser defaults are now optimized (1.75x default, 1.5x minimum), and auto-sync effective speed no longer drops below 1.5x.

v0.12.2 (April 19, 2026)

What's new for you

  • YouTube caption-first narration for Their voice — On YouTube, Narrate their voice now prefers YouTube translated captions directly, so narration can start with lower translation cost and less delay.
  • Safer fallback when captions are unavailable — If matching captions cannot be extracted, the extension now falls back automatically to the Soniox translation pipeline without breaking your narration flow.
  • More stable timeline sync in YouTube narration — Caption narration now tracks start-at-offset playback and seek changes more accurately, while avoiding overlap between caption narration and Soniox narration.
  • New Minimize action in translator header — You can now hide the panel using Minimize while keeping the extension runtime active in the background.
  • Stop and Close now fully tear down runtime — Ending a session now consistently stops mic, narration, interpreter mode, and voice injection; pressing Stop from controls also auto-opens transcript download when data is available.
  • Clear unsupported-page behavior for call actions — Microphone, interpreter, and Narrate My Voice actions now show disabled states with clearer tooltip feedback on unsupported pages.

v0.11.5 (April 11, 2026)

What's new for you

  • Faster Narrate My Voice playback — Narrate My Voice now flushes earlier at punctuation and shorter buffer windows, so translated voice starts sooner during long sentences.
  • Separate voice selection for each narration mode — You can now choose one voice for Narrate their voice and another voice for Narrate my voice, with each mode using exactly the selected voice at runtime.
  • Voice preview for both voice selectors — Added a dedicated Test action for both Their voice and My voice so you can hear the selected voice before starting narration.
  • Clearer narration controls — Added explanatory tooltips for narration actions and voice selectors, and moved Speaker settings into the control menu for a more consistent toolbar layout.
  • Better first-time narration defaults — Default voice selections are now optimized for bilingual meetings: Their voice = Google Vietnamese, My voice = Google English.

v0.11.2 (April 10, 2026)

What's new for you

  • New Context Settings for Soniox translation — You can now open context settings directly from the control bar via the control menu and define session context (domain, topic, terms, and speakers).
  • Reusable saved contexts — Save context presets and quickly apply them in future sessions. Saved contexts now persist reliably after page reload and popup reopen.
  • YouTube auto context mapping — On YouTube, the extension now auto-fills context using video metadata and auto-applies it for the current session (title -> domain, description -> topic).
  • Import / Export context JSON — Export your saved contexts to JSON for backup, and import shared context packs from other users in one click.
