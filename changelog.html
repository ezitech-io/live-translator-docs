Changelog


v0.10.1 (April 6, 2026)

What's new for you

  • New: WhatsApp Web support — Live Translator now works on WhatsApp Web, including real-time translation and Narrate My Voice for cross-language calls.
  • Cleaner, more consistent controls — Start and End controls are now visually consistent and easier to use during fast conversations.
  • Smarter session ending — Pressing End now fully stops active voice states: microphone, narration, and voice-injection mode.
  • Better first-time narration defaults — Voice speed now starts at 60% and original volume at 20% for a more balanced listening experience.
  • Easier setup guidance — Empty state messaging is clearer, and if OpenAI-dependent features are available but OpenAI key is missing, you'll see a direct warning with what to configure.

v0.9.3 (April 4, 2026)

What's new for you

  • Auto update check when you open the popup — Live Translator now checks for a newer extension version each time you open the translator popup.
  • In-app update badge — If a newer version is available, you'll see an Update available badge in the title bar. Click it to open the install page instantly.
  • New subtitle mode for cleaner reading — Switch to a horizontal subtitle popup at the bottom-center of the screen to focus on translated captions without the full control-heavy layout.
  • Subtitle customization — Subtitle mode now includes opacity and font-size controls so you can tune readability to your preference.
  • Draggable and resizable subtitle view — Move and resize subtitle mode to match your meeting/video layout and screen space.

v0.8.5 (April 3, 2026)

What's new for you

  • Echo-free meetings with extension enabled — Fixed the root echo issue so your calls stay clean even when the extension is installed and running.
  • More stable Narrate My Voice pipeline — Improved virtual mic + translation routing behavior so continuous speech narration is more reliable.
  • New narration mode flow — Narration Settings now uses two clear actions: Narrate my voice and Narrate their voice, with one-mode-at-a-time behavior.
  • Simpler active state — When narration is active, the dialog now shows only Stop, making control easier and less confusing.
  • Smarter narration status icon — The control icon now reflects mode state: voice_over_off (off), record_voice_over (my voice), transcribe (their voice).

v0.8.1 (April 2, 2026)

What's new for you

  • Google + OpenAI in one voice picker — Narration settings now use one combined voice list, so you can switch engines by selecting a voice directly.
  • Faster voice search — Voice selector now supports filtering/autocomplete for quick lookup by provider, language, or voice name.
  • Google speed and volume controls improved — Voice speed and volume settings now apply properly when using Google voices.
  • New platform support — Extension now supports running on www.domestika.org.

v0.7.4 (April 1, 2026)

What's new for you

  • Lower narration latency — Narration now starts faster and responds quicker during continuous speech, so conversations feel more real time.
  • Stable original volume while narrating — Original meeting/video audio now stays at your configured reduced level during narration playback instead of jumping up and down between sentences.
  • Smart volume recovery — After narration stops, original volume stays reduced and only returns to normal if no new narration plays for 5 seconds.
  • Smoother listening experience — Overall narration transitions are cleaner, with less distracting audio fluctuation.

v0.7.1 (March 31, 2026)

What's new for you

  • Narrate My Voice is now more affordable — You can keep using translated voice in meetings with significantly lower API cost.

v0.6.5 (March 30, 2026)

What's new for you

  • Faster chat send control — The Send my translation via chat action is now a quick button right in the main control bar (next to Microphone), so you can turn it on/off faster.
  • Smoother interaction — Fixed the issue where this feature sometimes looked clickable but did not react in Settings.
  • Clearer app info — The extension version is now shown in the title bar, so you can quickly verify what version you're using.

v0.6.2 (March 29, 2026)

What's new for you

  • Narrate My Voice — Your translated speech is now spoken directly into the meeting. The other participant hears your words in their language in real time, so they don't need to read chat messages anymore.
  • Smarter Narration — Narration now only reads translations from other participants and automatically skips your own voice. No more hearing your own words repeated back to you.
  • SRT Subtitle Export — Download meeting transcripts as .srt subtitle files. Choose between original or translated text — ready to use in video editors or media players.
  • Improved Controls — Microphone now defaults to off when you open the extension. Narration uses cleaner Start/Stop buttons and toggle switches for a simpler experience.
  • Echo-Free Audio — Fixed audio feedback loop that could occur when narration and translation were both active. Conversations are now clean and echo-free.

v0.4.1 (March 24, 2026)

What's new for you

  • Added Two-way Translation with My language and Their language selectors in the translator header.
  • New Interpreter mode button: when enabled, your speech is translated into their language and can be sent directly to meeting chat.
  • Language pair settings are now remembered automatically for future meetings.
  • Improved Google Meet chat injection reliability with better input and send-button detection.

v0.3.1 (March 24, 2026)

What's new for you

  • Fixed AI Assistant issue where your typed question could be missing in some requests during ongoing live translation.
  • AI Assistant now uses one consistent assistant behavior, so responses are more predictable when you chat.
  • You can now edit the General Assistant Prompt directly in OpenAI Settings (with a quick reset to default).

v0.3.0 (March 24, 2026)

What's new for you

  • Inline AI Assistant is now built into each translation line, with quick actions to Explain and Generate answer.
  • New dedicated Settings page: manage Soniox and OpenAI keys in one place with automatic save.
  • Fresh, cleaner popup interface for better readability and easier control during live translation.
  • Narration and Speaker settings dialogs are redesigned for a more consistent and simpler experience.

v0.2.0 (March 23, 2026)

What's new for you

  • New Inline AI Assistant: hover any translation line and tap the assistant icon to get instant help.
  • Two quick AI actions are now available: Explain (understand what the other person means) and Generate answer (get reply suggestions faster).

v0.1.1 (March 21, 2026)

  • You can now change translation text size (Small, Medium, Large) directly from the control bar.
  • Translation text size settings are saved, so your preferred view stays the same next time.
  • New Narration mode is available: the app can read translated text out loud in real time.
  • Narration now includes a simple settings panel (voice, speed, volume, and quick test).
  • Playback is smoother and more reliable during live sessions, with better audio balance between narration and original video sound.
