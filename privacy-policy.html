Live Translator — Privacy Policy

Developer: Doan Pham  ·  Contact: phampdoan@gmail.com  ·  Last updated: March 2026

Summary: Live Translator does not collect, store, or share any personal data on its own servers. Audio is processed exclusively by the Soniox API using your own API key. All settings are stored locally on your device.

1. What Is Live Translator?

Live Translator is a Chrome browser extension that provides real-time speech transcription and translation during online meetings and video calls on platforms such as Google Meet, Zoom, Microsoft Teams, and YouTube. It captures audio from your browser tab or microphone and sends it to the Soniox speech-to-text API for processing, then displays the transcription and translation in a floating overlay panel.

2. Data We Collect

Live Translator collects and processes only the minimum data necessary to function:

  • Audio data — When you start a translation session, the extension captures audio from the current browser tab (via the browser's screen-sharing API) or, as a fallback, from your microphone. This audio stream is transmitted in real time to the Soniox API over an encrypted WebSocket connection (wss://stt-rt.soniox.com). Audio data is not stored by this extension; it is processed transiently by Soniox for transcription purposes only.
  • Soniox API key — You provide your own Soniox API key in the extension settings. This key is stored locally on your device using the browser's built-in chrome.storage.local API and is never transmitted to any server other than Soniox's own API endpoints.
  • Target language preference — Your chosen translation target language is stored locally in chrome.storage.local on your device. It is not shared or transmitted anywhere.

We do not collect: browsing history, page content, cookies, personally identifiable information, usage analytics, crash reports, or any data beyond what is described above.

3. Transcription and Translation Data

Transcription and translation results (the text displayed in the panel) exist only in memory during an active session. When you close or minimize the panel, this data is discarded.

A future version of the extension will include an opt-in feature allowing you to export your transcription history as a local file. If implemented, this data will remain entirely on your device and will not be transmitted to any external server without your explicit action.

4. Third-Party Services

Live Translator uses the Soniox API for real-time speech recognition and translation. Audio data sent to Soniox is governed by Soniox's own Privacy Policy and Terms of Service. You are responsible for reviewing and accepting Soniox's policies when you obtain an API key.

Live Translator has no affiliation with Google, Zoom, Microsoft, YouTube, or any other platform it operates alongside.

5. Permissions Explanation

The extension requests the following browser permissions:

  • storage — To save your API key and language preference locally on your device.
  • activeTab — To inject the translation panel into the currently active browser tab when you click the extension icon.
  • scripting — To programmatically inject the panel UI into tabs on demand (when used on pages outside the auto-injected list).
  • Microphone / display-capture — Prompted at runtime. Required to capture audio for transcription. The extension only accesses audio when you explicitly start a translation session. Video tracks are immediately discarded if display-capture is used.

The extension does not request broad host permissions (*://*/*). It auto-injects only on Google Meet, Zoom, Microsoft Teams, and YouTube. On other sites, injection happens only when you explicitly click the extension icon.

6. Data Security

All communication between the extension and the Soniox API uses encrypted connections (WSS/HTTPS). Your API key is stored locally and is never logged or transmitted by the extension itself. No data is stored on external servers operated by the developer.

7. Data Retention

No data is retained by this extension beyond your active browser session. Local settings (API key, language preference) persist across sessions but are stored only on your device. You can remove all stored data at any time by uninstalling the extension or clearing extension data in Chrome settings (chrome://extensions → Details → Clear data).

8. Children's Privacy

Live Translator is not directed at children under the age of 13 and does not knowingly collect any information from children.
