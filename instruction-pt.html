Instalação

Instale a extensão Live Translator em qualquer navegador baseado em Chromium (Chrome, Edge, Brave, Arc, Opera, etc.).

1 Instale pela Chrome Web Store

Abra a página oficial: Live Translator na Chrome Web Store, clique em Add to Chrome e confirme com Add extension.

Pronto!

Agora você deve ver o ícone do Live Translator na barra de ferramentas do navegador. A extensão está instalada e pronta para ser configurada.

Visão Geral da Extensão

Uma visão rápida da interface principal e de onde cada recurso está localizado.

O painel do Live Translator tem as seguintes áreas principais:

  • Barra de título — Botão de configurações, botão do modo legendas, botão de fechar. Quando houver uma atualização disponível, um indicador aparecerá aqui.
  • Seletor de idioma — Dois menus suspensos: "Meu idioma" (o idioma que você fala) e "Idioma deles" (o idioma do qual você quer traduzir).
  • Área de transcrição — Onde o texto traduzido aparece em tempo real, organizado por locutor.
  • Barra de controle — Botão Iniciar/Parar/Pausar, microfone, configurações de narração, assistente de IA, tamanho da fonte, download e mais opções.
Dica

Você pode arrastar o painel pela barra de título para movê-lo para qualquer lugar da tela e redimensioná-lo arrastando as bordas. Assim, ele não vai atrapalhar o conteúdo importante durante reuniões ou vídeos.

Obter a Chave de API do Soniox

O Soniox é o mecanismo de reconhecimento de fala e tradução em tempo real. Você precisa de uma chave de API do Soniox para usar o Live Translator. Obrigatório

Importante

A chave de API do Soniox é obrigatória para o funcionamento da extensão. Sem ela, a extensão não consegue transcrever nem traduzir nenhum áudio.

1 Acesse o site do Soniox

Abra https://console.soniox.com/signup no seu navegador. Crie uma conta usando seu e-mail ou login do Google.

2 Obtenha sua chave de API

Depois de fazer login, acesse a seção de API no painel do Soniox. Você também pode ir diretamente ao Guia de Primeiros Passos do Soniox para ver instruções detalhadas.

Copie sua chave de API — você vai precisar dela nos próximos passos.

3 Adicionar cobrança (obrigatório para uso da API)

Adicione um método de pagamento à sua conta do Soniox. Você pode usar cartão de crédito, PayPal ou outros métodos de pagamento. A API do Soniox funciona no modelo pague conforme o uso — você paga apenas pelo que consumir.

Nota

Você precisa depositar no mínimo US$ 1 na sua conta para usar a API. Pode depositar mais se desejar.

Obter a Chave de API da OpenAI (Opcional)

A chave de API da OpenAI é opcional — você só precisa dela se quiser usar vozes da OpenAI para narração ou o recurso Assistente de IA Integrado. Pro

1 Acesse a plataforma da OpenAI

Abra https://platform.openai.com/api-keys no seu navegador.

2 Crie uma conta ou faça login

Se você ainda não tem uma conta na OpenAI, clique em "Sign Up" e crie uma. Se já tiver, faça login.

3 Crie uma nova chave de API

Clique em "+ Create new secret key", dê um nome como "Live Translator" e clique em "Create secret key".

Copie e salve a chave imediatamente — a OpenAI só mostra a chave uma vez.

4 Adicionar cobrança (obrigatório para uso da API)

Acesse as Configurações de cobrança e adicione um método de pagamento. A API da OpenAI funciona no modelo pague conforme o uso — você paga apenas pelo que consumir.

Nota

Se você quiser apenas a narração (texto para fala), pode usar as vozes gratuitas do Google Tradutor. A chave da OpenAI só é necessária se você quiser as vozes premium da OpenAI ou o recurso Assistente de IA.

Inserir suas Chaves de API

Depois de obter suas chaves de API, insira-as na página de configurações do Live Translator.

1 Abra as configurações do Live Translator

Você pode abrir as configurações de qualquer uma destas formas:

  • Clique no ícone de engrenagem (⚙) no canto superior direito do painel do tradutor.
  • Clique com o botão direito no ícone do Live Translator na barra de ferramentas → "Opções".
  • Acesse a página de extensões do navegador → encontre o Live Translator → "Detalhes" → "Opções da extensão".
2 Insira sua chave de API do Soniox

Na página de configurações, clique em "Soniox" na barra lateral esquerda. Cole sua chave de API do Soniox no campo de entrada. A chave é salva automaticamente.

3 Insira sua chave de API da OpenAI (opcional)

Clique em "OpenAI" na barra lateral esquerda. Cole sua chave de API da OpenAI no campo de entrada. A chave é salva automaticamente.

Nota de Segurança

Suas chaves de API são armazenadas apenas localmente no seu dispositivo, usando o armazenamento seguro de extensões do Chrome. Elas nunca são enviadas para os servidores do Live Translator — são usadas apenas para se comunicar diretamente com o Soniox e a OpenAI.

Iniciar uma Sessão de Tradução

Veja como começar a traduzir em uma reunião ou vídeo.

1 Abra um site compatível

Acesse qualquer plataforma compatível (por exemplo, Google Meet, Zoom, YouTube, etc.) e entre em uma reunião ou reproduza um vídeo.

2 Abra o painel do Live Translator

Clique no ícone do Live Translator na barra de ferramentas do navegador. O painel do tradutor aparecerá na página. Você também pode clicar no ícone flutuante que aparece na página.

3 Selecione seus idiomas

Use os dois menus suspensos de idioma na parte superior do painel:

  • "Meu idioma" — O idioma que você fala (por exemplo, português).
  • "Idioma deles" — O idioma que os outros falam e que você quer traduzir (por exemplo, inglês).
4 Pressione o botão Iniciar

Clique no botão Iniciar (ícone de play) na barra de controle na parte inferior do painel.

O navegador pode pedir permissão para capturar o áudio — clique em "Permitir" e selecione a aba que deseja capturar.

5 Veja as traduções em tempo real

Conforme as pessoas falam, você verá tanto o texto original quanto o texto traduzido na área de transcrição. As linhas são agrupadas automaticamente por locutor.

Rolagem automática

A transcrição rola automaticamente para mostrar o texto mais recente. Se você rolar para cima para ler algo anterior, a rolagem automática é pausada. Clique no botão "Rolar para o mais recente" que aparece para voltar ao texto ao vivo.

Controles: Pausar, Retomar, Encerrar

Conheça os controles da sessão de tradução.

BotãoO que faz
Iniciar (▶)Inicia uma nova sessão de tradução. Limpa qualquer transcrição anterior.
Pausar (⏸)Interrompe a tradução temporariamente. A transcrição é mantida. A captura de áudio continua, mas o texto fica pausado.
Retomar (▶)Continua a tradução após uma pausa. O novo texto aparece após o conteúdo anterior.
Encerrar (⏹)Para a sessão completamente. A transcrição permanece na tela até você iniciar uma nova sessão. Encerra a captura de áudio, desliga o microfone e a narração.

Alterar Idiomas

Você pode alterar os idiomas da tradução a qualquer momento.

1 Clique no menu suspenso de idioma

Clique no menu suspenso "Meu idioma" ou "Idioma deles" na parte superior do painel.

2 Selecione um novo idioma

A lista inclui mais de 60 idiomas em ordem alfabética. Role pela lista ou digite para buscar o idioma desejado. Clique para selecioná-lo.

3 A tradução reinicia automaticamente

Se houver uma sessão ativa, a alteração do idioma reiniciará a tradução automaticamente com o novo par de idiomas. Sua seleção de idioma é salva para a próxima vez.

Ativar o Microfone

Ative seu microfone para que sua própria fala também seja traduzida junto com o áudio da reunião ou do vídeo.

1 Clique no botão do microfone

Na barra de controle, clique no ícone do microfone. O ícone fica destacado quando o microfone está ativado.

2 Permita o acesso ao microfone

Se for a primeira vez, o navegador pedirá permissão para acessar o microfone. Clique em "Permitir".

Dica

O microfone fica desativado por padrão quando você abre o painel. Ative-o apenas se quiser que sua própria fala seja transcrita e traduzida. Quando você pressiona Encerrar, o microfone é desligado automaticamente.

Alterar o Tamanho da Fonte

Ajuste o tamanho do texto da transcrição para uma leitura confortável.

1 Clique no botão de tamanho da fonte

Na barra de controle, clique no botão de tamanho da fonte (ícone Aa). Um menu aparecerá com três opções:

  • Pequeno — Visualização compacta, cabe mais texto
  • Médio — Tamanho equilibrado padrão
  • Grande — Mais fácil de ler à distância

Clique no tamanho que preferir. A alteração é aplicada imediatamente e salva para a próxima vez.

Nomes dos Locutores

Atribua nomes reais aos locutores para identificar facilmente quem disse o quê na transcrição. Gratuito

1 Abra o menu "Mais"

Na barra de controle, clique no botão "Mais" (ícone ⋯). Em seguida, clique no botão "Configurações de Locutores".

2 Abra a janela "Configurações de Locutores"

A janela de Configurações de Locutores será aberta.

3 Atribua nomes

Você verá uma lista de locutores detectados (Locutor 1, Locutor 2, etc.). Digite um nome real ao lado de cada um (por exemplo, "João", "Maria"). Clique em "Adicionar linha" se precisar de mais.

Clique em "Salvar". Todas as linhas da transcrição — passadas e futuras — passarão a usar os nomes que você definiu.

Narração (Texto para Fala)

Ouça as traduções lidas em voz alta, em vez de precisar ler. Este é o recurso "Narrar a Voz Deles". Pro

1 Abra as Configurações de Narração

Na barra de controle, clique no botão de narração (ícone de alto-falante). A janela de Configurações de Narração será aberta.

2 Escolha um provedor de TTS e uma voz

Selecione o provedor de voz:

  • Google Tradutor (Gratuito) — Funciona sem nenhuma chave de API. Vozes de boa qualidade em mais de 60 idiomas.
  • OpenAI (Premium) — Vozes mais naturais e de qualidade superior. Requer uma chave de API da OpenAI.

Selecione uma voz no menu suspenso e clique em "Testar" para ouvir como ela soa.

Ajuste a velocidade da voz e o volume se necessário.

3 Inicie a narração

Clique no botão "Narrar a Voz Deles" na janela. A partir de agora, sempre que uma tradução aparecer, ela será lida em voz alta para você.

Para parar, clique novamente no botão de narração e depois em "Parar".

Narração por legenda do YouTube

Use a legenda traduzida do YouTube como base da narração, para a extensão ler no idioma que você já escolheu no YouTube. Este recurso pode economizar até 99,9% do custo do Soniox ao usar o YouTube. Pro

Ordem importante

Para funcionar corretamente, primeiro escolha o idioma da legenda no YouTube e só depois inicie a narração na extensão.

1 Primeiro configure a legenda traduzida no YouTube

No player do YouTube, ative as legendas e escolha o idioma traduzido que você quer ouvir (por exemplo: vietnamita).

2 Ative a detecção automática nas configurações de Narração

Abra as configurações de Narração da extensão e ative Auto detect translated caption.

3 Inicie a narração

Clique em "Narrar a Voz Deles". A narração seguirá o idioma da legenda traduzida que você escolheu no YouTube.

Exemplo

Se você quiser ouvir a narração em vietnamita, primeiro defina a legenda do YouTube para vietnamita. Depois ative Auto detect translated caption e inicie "Narrar a Voz Deles".

Narrar Minha Voz (Modo Intérprete)

Fale no seu idioma e tenha sua voz traduzida reproduzida para os outros participantes da reunião — como ter um intérprete pessoal. Pro

Requisitos

Este recurso funciona no Google Meet, Zoom (Web), Microsoft Teams (Web) e WhatsApp Web. Você precisa ter o microfone ativado e uma sessão de tradução em execução.

1 Abra as Configurações de Narração

Clique no botão de narração na barra de controle para abrir a janela de Configurações de Narração.

2 Configure "Narrar Minha Voz"

Na janela, encontre a seção "Narrar Minha Voz". Escolha a voz e o provedor de sua preferência (Google ou OpenAI). Você pode definir uma voz diferente para a narração de "Minha voz" e "A voz deles".

Clique em "Testar" para ouvir a voz selecionada.

3 Inicie "Narrar Minha Voz"

Clique em "Iniciar Narrar Minha Voz". Quando você falar, suas palavras serão:

  1. Transcritas no seu idioma
  2. Traduzidas para o outro idioma
  3. Reproduzidas em voz alta na reunião para que a outra pessoa ouça sua voz traduzida

Seu microfone é silenciado automaticamente enquanto o áudio traduzido é reproduzido, para evitar eco.

4 Pare a narração

Clique no botão de narração e depois em "Parar", ou pressione Encerrar para parar toda a sessão (o que também interrompe a narração).

Nota

Só é possível usar um modo de narração por vez — "Narrar Minha Voz" ou "Narrar a Voz Deles", não os dois ao mesmo tempo. O ícone na barra de ferramentas muda para indicar qual modo está ativo.

Enviar Tradução via Chat

Envie automaticamente sua fala traduzida como mensagens de texto no chat da reunião. Pro

1 Ative "Enviar via Chat"

Na barra de controle, procure o botão de envio pelo chat (ao lado do ícone do microfone). Clique nele para ativar — o botão fica verde quando está ativo.

2 Fale e o envio acontece automaticamente

Com o microfone ligado e uma sessão de tradução ativa, quando você falar, o texto traduzido será inserido automaticamente no chat da reunião e enviado.

Funciona no Google Meet, Zoom (Web) e Microsoft Teams (Web).

Dica

Isso é útil quando o recurso "Narrar Minha Voz" não está disponível na sua plataforma, ou se você quer usar a narração de voz E a tradução pelo chat ao mesmo tempo.

Modo Legendas

Mude para uma barra de legendas minimalista, estilo cinema, que mostra apenas a tradução mais recente na parte inferior da tela. Pro

1 Ative o Modo Legendas

Na barra de título do painel do tradutor, clique no ícone de legendas (ícone de closed captions).

O painel completo será substituído por uma barra fina de legendas na parte inferior da tela.

2 Personalize a barra de legendas

Clique no ícone de ajustes (engrenagem) na barra de legendas para configurar:

  • Opacidade — Deslize para deixar a barra mais ou menos transparente.
  • Tamanho da fonte — Escolha entre texto pequeno, médio ou grande.

Você também pode arrastar a barra de legendas para movê-la e redimensioná-la arrastando as bordas.

3 Saia do Modo Legendas

Clique no botão de fechar (✕) na barra de legendas para voltar ao painel completo do tradutor.

Assistente de IA Integrado

Receba explicações geradas por IA e respostas sugeridas com base no contexto da conversa, diretamente na transcrição. Pro

Requisito

O Assistente de IA Integrado exige que uma chave de API da OpenAI esteja configurada nas configurações.

Explicar uma linha

1 Passe o mouse sobre uma linha da transcrição

Ao passar o mouse sobre qualquer linha na transcrição, um ícone de IA (🤖) aparecerá.

2 Clique no ícone de IA e escolha "Explicar"

O assistente de IA abre em uma janela separada. Ele usará o contexto completo da transcrição e a linha selecionada para oferecer uma explicação detalhada.

Gerar uma resposta

1 Clique no ícone de IA e escolha "Gerar Resposta"

O assistente de IA vai gerar uma sugestão de resposta com base no que foi dito na conversa. Ele pode até usar busca na web para fornecer respostas mais precisas.

Abrir o painel do Assistente

1 Clique no botão do Assistente na barra de controle

Você também pode abrir o painel do assistente de IA diretamente pelo botão do assistente (ícone 🤖) na barra de controle. Isso abre uma janela de chat arrastável onde você pode digitar qualquer pergunta. O assistente tem acesso à transcrição completa como contexto.

Configurações de Contexto

Forneça informações sobre o tema da conversa para melhorar a precisão da tradução. Gratuito

1 Abra as Configurações de Contexto

Clique no botão "Mais" (⋯) na barra de controle e depois clique em "Contexto".

2 Preencha os campos de contexto

Você pode informar:

  • Área / Tema — Sobre o que é a conversa (por exemplo, "Conferência médica", "Daily de engenharia de software").
  • Termos-chave — Termos técnicos, nomes próprios ou jargões que podem ser difíceis de reconhecer (por exemplo, "Kubernetes, API gateway, microsserviços").
  • Informações dos locutores — Nomes e funções dos participantes (por exemplo, "Dr. Silva — cardiologista, apresentando").

Clique em "Aplicar" para enviar esse contexto ao mecanismo de tradução na sessão atual.

3 Salvar predefinições (opcional)

Se você tem reuniões recorrentes sobre temas semelhantes, clique em "Salvar Predefinição" para salvar o contexto atual. Na próxima vez, basta carregar a predefinição em vez de digitar tudo de novo.

Você também pode Importar/Exportar predefinições como arquivos JSON para compartilhar ou fazer backup.

Preenchimento automático no YouTube

Ao usar o Live Translator no YouTube, os campos de contexto podem ser preenchidos automaticamente com o título e a descrição do vídeo. Isso ajuda o mecanismo de tradução a entender melhor o tema do vídeo.

Baixar Transcrição

Salve suas transcrições traduzidas como arquivos para consultar depois ou usar em outras ferramentas. Pro

1 Clique no botão de Download

Após uma sessão de tradução (ou durante uma), clique no botão de download (ícone ↓) na barra de controle. A janela de Download de Transcrição será aberta.

2 Escolha um formato

Selecione o formato desejado:

FormatoIdeal para
TXTArquivo de texto simples com timestamps e nomes dos locutores. Fácil de ler.
JSONFormato de dados estruturado. Bom para processar com outras ferramentas ou scripts.
SRT (Original)Arquivo de legendas com o texto no idioma original. Use em softwares de edição de vídeo.
SRT (Traduzido)Arquivo de legendas com o texto traduzido. Use em softwares de edição de vídeo.
3 Baixe o arquivo

Clique no botão de download do formato escolhido. O arquivo é gerado no seu navegador e salvo na pasta de downloads padrão.

Privacidade

Os arquivos de transcrição são gerados inteiramente no seu navegador. Nenhum dado de transcrição é enviado a qualquer servidor durante o download.

Atualização Automática

A extensão verifica automaticamente se há atualizações sempre que você a abre.

1 Fique de olho no indicador de atualização

Quando uma nova versão estiver disponível, você verá um indicador "Atualização disponível" na barra de título do painel do tradutor.

2 Clique para atualizar

Clique no indicador para abrir a página da Chrome Web Store e verificar a versão mais recente. Instalações pela Store são atualizadas automaticamente quando uma nova versão é publicada.

Solução de Problemas

Problemas comuns e como resolvê-los.

Mensagem "Nenhuma chave de API do Soniox"

! Solução

Você ainda não inseriu sua chave de API do Soniox. Acesse Inserir suas Chaves de API para configurá-la.

Nenhum áudio está sendo capturado

! Solução
  • Verifique se você clicou em "Permitir" quando o navegador pediu permissão para capturar áudio.
  • Confirme que você selecionou a aba correta para compartilhar quando solicitado.
  • Tente atualizar a página e abrir o tradutor novamente.
  • Verifique se a reunião ou o vídeo realmente tem áudio sendo reproduzido.

A tradução parece imprecisa

! Solução
  • Verifique se o menu suspenso "Idioma deles" está configurado para o idioma correto que está sendo falado.
  • Use as Configurações de Contexto para fornecer informações sobre o tema e termos-chave.
  • Garanta que a qualidade do áudio esteja boa — ruído de fundo pode prejudicar a precisão.

"Narrar Minha Voz" — os outros não conseguem ouvir a tradução

! Solução
  • Confirme que você está em uma plataforma compatível (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Verifique se a sessão de tradução está em execução e se o microfone está ativado.
  • Confirme que você tem uma licença Pro ativada.
  • Tente atualizar a página da reunião e começar novamente.

Recursos da OpenAI não funcionam

! Solução
  • Verifique se sua chave de API da OpenAI está inserida corretamente nas configurações.
  • Confirme que sua conta da OpenAI tem a cobrança configurada com créditos suficientes.
  • Se você não precisa dos recursos da OpenAI, use as vozes do Google Tradutor para narração (gratuitas, sem necessidade de chave de API).

A extensão não aparece em um site

! Solução
  • O Live Translator só funciona em plataformas compatíveis. Ele não aparecerá em sites não compatíveis.
  • Se o site é compatível mas a extensão não aparece, tente atualizar a página.
  • Verifique se a extensão está ativada na página de gerenciamento de extensões do navegador.
