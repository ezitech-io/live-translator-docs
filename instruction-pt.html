Instalação

Instale a extensão Live Translator em qualquer navegador baseado em Chromium (Chrome, Edge, Brave, Arc, Opera, etc.).

1 Abra o site do Live Translator

Vá para https://translate.ezitech.io no seu navegador.

2 Clique no botão "Instalar"

Clique no Adicionar ao Chrome botão. Isso fará o download do arquivo de extensão.

3 Ative o modo de desenvolvedor em seu navegador

Abra a página de gerenciamento de extensões do seu navegador:

  • Chrome/Brave/Arc: Tipo chrome://extensions na barra de endereço e pressione Enter.
  • Borda: Tipo edge://extensions na barra de endereço e pressione Enter.

Alternar Modo de desenvolvedor ON no canto superior direito da página.

4 Instale o arquivo de extensão

Arraste e solte o arquivo baixado .zip arquivo na página de extensões.

Concluído!

Agora você deve ver o ícone do Live Translator na barra de ferramentas do seu navegador. A extensão está instalada e pronta para ser configurada.

Visão geral da extensão

Uma rápida visão geral da interface principal e onde tudo está localizado.

O painel do Live Translator possui estas áreas principais:

  • Barra de título — Botão Configurações, alternância do modo de legenda, botão Fechar. Se uma atualização estiver disponível, você verá um emblema aqui.
  • Seletor de idioma — Dois menus suspensos: "Meu idioma" (o idioma que você fala) e "O idioma deles" (o idioma do qual você deseja traduzir).
  • Área de transcrição — Onde o texto traduzido aparece em tempo real, organizado por locutor.
  • Barra de controle — Botão Iniciar/Parar/Pausar, alternância de microfone, configurações de narração, assistente de IA, tamanho da fonte, download e mais opções.
Dica

Você pode arraste o painel pela barra de título para movê-lo para qualquer lugar da tela e redimensionar arrastando as bordas. Dessa forma não bloqueará conteúdos importantes durante reuniões ou vídeos.

Obtenha uma chave de API Soniox

Soniox fornece reconhecimento de fala em tempo real e mecanismo de tradução. Você precisa de uma chave de API Soniox para usar o Live Translator. Obrigatório

Importante

Uma chave de API Soniox é obrigatório para que a extensão funcione. Sem ele, a extensão não pode transcrever ou traduzir nenhum áudio.

1 Acesse o site da Soniox

Abrir https://console.soniox.com/signup no seu navegador. Crie uma conta usando seu e-mail ou login do Google.

2 Obtenha sua chave de API

Depois de fazer login, navegue até a seção API do seu painel Soniox. Você também pode ir diretamente para Guia de primeiros passos do Soniox para obter instruções passo a passo.

Copie sua chave de API — você precisará dela nas próximas etapas.

3 Adicionar faturamento (obrigatório para uso da API)

Adicione uma forma de pagamento à sua conta Soniox. Você pode usar cartão de crédito, PayPal ou outros métodos de pagamento. A API Soniox é pré-paga – você paga apenas pelo que usa.

Nota

Você precisa depositar no mínimo US$ 1 em sua conta para usar a API. Você pode depositar mais se quiser.

Obtenha uma chave de API OpenAI (opcional)

Uma chave de API OpenAI é opcional — você só precisa dele se quiser usar vozes OpenAI para narração ou o recurso Inline AI Assistant. Pró

1 Acesse a plataforma OpenAI

Abrir https://platform.openai.com/api-keys no seu navegador.

2 Crie uma conta ou faça login

Se você não possui uma conta OpenAI, clique em "Inscreva-se" e crie um. Se você já possui um, faça login.

3 Crie uma nova chave de API

Clique "+ Criar nova chave secreta", dê um nome como "Live Translator" e clique "Criar chave secreta".

Copie e salve a chave imediatamente — OpenAI mostrará apenas uma vez.

4 Adicionar faturamento (obrigatório para uso da API)

Vá para Configurações de faturamento e adicione uma forma de pagamento. A API OpenAI é pré-paga – você paga apenas pelo que usa.

Nota

Se você deseja apenas narração (conversão de texto em fala), você pode usar Vozes do Google Tradutor gratuitamente em vez disso. A chave OpenAI só é necessária se você deseja as vozes premium do OpenAI ou o recurso AI Assistant.

Insira suas chaves de API

Depois de obter suas chaves de API, insira-as na página de configurações do Live Translator.

1 Abrir configurações do Tradutor ao vivo

Você pode abrir as configurações de qualquer uma destas maneiras:

  • Clique no ícone de engrenagem (⚙) no canto superior direito do painel do tradutor.
  • Clique com o botão direito no ícone do Live Translator em sua barra de ferramentas → "Opções".
  • Vá para a página de extensões do seu navegador → encontre o Live Translator → "Detalhes" → "Opções de extensão".
2 Insira sua chave de API Soniox

Na página de configurações, clique em "Soniox" na barra lateral esquerda. Cole sua chave API Soniox no campo de entrada. A chave é salva automaticamente.

3 Insira sua chave de API OpenAI (opcional)

Clique em "OpenAI" na barra lateral esquerda. Cole sua chave de API OpenAI no campo de entrada. A chave é salva automaticamente.

Nota de segurança

Suas chaves de API são armazenadas localmente apenas no seu dispositivo usando o armazenamento seguro de extensões do Chrome. Eles nunca são enviados para os servidores do Live Translator — eles são usados ​​apenas para comunicação direta com Soniox e OpenAI.

Iniciar uma sessão de tradução

Aprenda como começar a traduzir em uma reunião ou vídeo.

1 Abra um site compatível

Vá para qualquer plataforma suportada (por exemplo, Google Meet, Zoom, YouTube, etc.) e participe de uma reunião ou reproduza um vídeo.

2 Abra o painel do Live Translator

Clique no Ícone do Tradutor ao vivo na barra de ferramentas do seu navegador. O painel do tradutor aparecerá na página. Você também pode clicar no ícone flutuante que aparece na página.

3 Selecione seus idiomas

Use os dois menus suspensos de idiomas na parte superior do painel:

  • "Meu idioma" — O idioma você falar (por exemplo, vietnamita).
  • "A linguagem deles" — O idioma outros fale que deseja traduzir (por exemplo, inglês).
4 Pressione o botão Iniciar

Clique no Iniciar (ícone de reprodução) na barra de controle na parte inferior do painel.

Seu navegador pode pedir permissão para capturar áudio — clique "Permitir" e selecione a aba que deseja capturar.

5 Veja traduções em tempo real

À medida que as pessoas falam, você verá tanto o texto original e o texto traduzido aparecem na área de transcrição. As linhas são agrupadas automaticamente por alto-falante.

Rolagem automática

A transcrição rola automaticamente para mostrar o texto mais recente. Se você rolar para cima para ler o texto anterior, a rolagem automática será pausada. Clique no "Rolar para o mais recente" botão que parece voltar ao texto ativo.

Controles: Pausar, Continuar, Finalizar

Entenda os controles da sessão de tradução.

BotãoO que faz
Iniciar (▶)Inicia uma nova sessão de tradução. Limpa qualquer transcrição anterior.
Pausa (⏸)Interrompe temporariamente a tradução. A transcrição é mantida. A captura de áudio continua, mas o texto está pausado.
Retomar (▶)Continua a tradução após uma pausa. O novo texto aparece após o conteúdo anterior.
Fim (⏹)Para a sessão completamente. A transcrição é mantida na tela até você iniciar uma nova sessão. Termina a captura de áudio, desliga o microfone e a narração.

Alterar idioma

Você pode alterar os idiomas de tradução a qualquer momento.

1 Clique no menu suspenso de idiomas

Clique no "Meu idioma" ou "A linguagem deles" menu suspenso na parte superior do painel.

2 Selecione um novo idioma

A lista suporta mais de 60 idiomas classificados em ordem alfabética. Role ou digite para pesquisar seu idioma. Clique para selecioná-lo.

3 A tradução é reiniciada automaticamente

Se uma sessão estiver ativa, a alteração do idioma reiniciará automaticamente a tradução com o novo par de idiomas. Sua seleção de idioma será salva para a próxima vez.

Ligue seu microfone

Ative seu microfone para que sua própria fala também seja traduzida junto com o áudio da reunião/vídeo.

1 Clique no botão do microfone

Na barra de controle, clique no botão ícone de microfone. O ícone fica ativo (destacado) quando o microfone está ligado.

2 Permitir acesso ao microfone

Se esta for a primeira vez, seu navegador solicitará permissão de microfone. Clique "Permitir".

Dica

O microfone está desativado por padrão quando você abre o painel. Ative-o apenas se desejar que seu próprio discurso seja transcrito e traduzido. Quando você pressiona Fim, o microfone é desligado automaticamente.

Alterar tamanho da fonte

Ajuste o tamanho do texto da transcrição para uma leitura confortável.

1 Clique no botão de tamanho da fonte

Na barra de controle, clique no botão botão de tamanho da fonte (ícone Aa). Um menu aparece com três tamanhos:

  • Pequeno — Visualização compacta, cabe mais texto
  • Médio — Tamanho balanceado padrão
  • Grande — Mais fácil de ler à distância

Clique no tamanho de sua preferência. A alteração se aplica imediatamente e é salva para a próxima vez.

Nomes dos palestrantes

Atribua nomes reais aos palestrantes para que você possa identificar facilmente quem disse o quê na transcrição. Grátis

1 Abra o menu "Mais"

Na barra de controle, clique no botão "Mais" Botão (ícone ⋯). Então clique "Configurações do alto-falante" botão.

2 Abrir a caixa de diálogo "Configurações do alto-falante"

A caixa de diálogo Configurações do alto-falante será aberta.

3 Atribuir nomes

Você verá uma lista de alto-falantes detectados (Alto-falante 1, Alto-falante 2, etc.). Digite um nome real ao lado de cada um (por exemplo, “John”, “Sarah”). Clique "Adicionar linha" se precisar de mais.

Clique "Salvar". Todas as linhas de transcrição – passadas e futuras – agora usarão os nomes que você atribuiu.

Narração (conversão de texto em fala)

Faça com que as traduções sejam lidas em voz alta para que você possa ouvir em vez de ler. Este é o recurso "Narrar a voz deles". Pró

1 Abrir configurações de narração

Na barra de controle, clique no botão botão de narração (ícone de alto-falante). A caixa de diálogo Configurações de narração é aberta.

2 Escolha um provedor de TTS e voz

Selecione seu provedor de voz:

  • Google Tradutor (Gratuito) — Funciona sem nenhuma chave de API. Vozes de boa qualidade em mais de 60 idiomas.
  • OpenAI (Premium) — Vozes mais naturais e de maior qualidade. Requer uma chave de API OpenAI.

Selecione um voz no menu suspenso e clique "Teste" para visualizar como soa.

Ajustar velocidade da voz e volume se necessário.

3 Iniciar narração

Clique "Narrar a voz deles" na caixa de diálogo. Agora, sempre que aparecer uma tradução, ela será lida em voz alta para você.

Para parar, clique novamente no botão de narração e clique "Parar".

Narrar minha voz (modo intérprete)

Fale no seu idioma e faça com que sua voz traduzida seja reproduzida para outros participantes da reunião, como se tivesse um intérprete pessoal. Pró

Requisitos

Este recurso funciona em Google Meet, Zoom (Web), Microsoft Teams (Web), e WhatsApp Web. Você precisa do seu microfone ativado e uma sessão de tradução correndo.

1 Abrir configurações de narração

Clique no botão de narração na barra de controle para abrir a caixa de diálogo Configurações de narração.

2 Configurar "Narrar minha voz"

Na caixa de diálogo, encontre o "Narrar minha voz" seção. Escolha sua voz e provedor preferidos (Google ou OpenAI). Você pode definir um voz diferente para narração de "Minha voz" e "A voz deles".

Clique "Teste" para visualizar a voz selecionada.

3 Inicie "Narrar minha voz"

Clique "Comece a narrar minha voz". Quando você fala, suas palavras são:

  1. Transcrito em seu idioma
  2. Traduzido para outro idioma
  3. Falado em voz alta na reunião para que a outra pessoa ouça sua voz traduzida

Seu microfone é silenciado automaticamente enquanto o áudio traduzido é reproduzido para evitar eco.

4 Parar a narração

Clique no botão de narração e depois "Parar", ou pressione Fim para interromper a sessão inteira (o que também interrompe a narração).

Nota

Você só pode executar um modo de narração por vez — "Narrate My Voice" ou "Narrate Their Voice", não ambos simultaneamente. O ícone da barra de ferramentas muda para mostrar qual modo está ativo.

Enviar tradução via chat

Envie automaticamente seu discurso traduzido como mensagens de texto no chat da reunião. Pró

1 Ative "Enviar via chat"

Na barra de controle, procure o botão de envio de bate-papo (ao lado do ícone do microfone). Clique nele para ativar – o botão gira verde quando ativo.

2 Fale e ele envia automaticamente

Com o microfone ligado e uma sessão de tradução em execução, quando você fala, o texto traduzido é automaticamente inserido na entrada do chat da reunião e enviado.

Isso funciona em Google Meet, Zoom (Web), e Microsoft Teams (Web).

Dica

Isso é útil quando "Narrate My Voice" não está disponível para sua plataforma ou se você deseja narração de voz E tradução de chat de texto simultaneamente.

Modo Legenda

Mude para uma barra de legenda mínima, estilo filme, que mostra apenas a tradução mais recente na parte inferior da tela. Pró

1 Ativar modo de legenda

Na barra de título do painel do tradutor, clique no botão ícone de legenda (ícone de legendas).

O painel completo será substituído por uma fina barra de legendas na parte inferior da tela.

2 Personalize a barra de legendas

Clique no ícone de melodia (engrenagem de configurações) na barra de legenda para ajustar:

  • Opacidade — Deslize para tornar a barra mais ou menos transparente.
  • Tamanho da fonte — Escolha entre texto pequeno, médio ou grande.

Você também pode arrastar a barra de legenda para movê-la e redimensionar arrastando as bordas.

3 Sair do modo de legenda

Clique no botão fechar (✕) na barra de legenda para retornar ao painel completo do tradutor.

Assistente de IA inline

Obtenha explicações baseadas em IA e respostas geradas com base no contexto da conversa, diretamente da transcrição. Pró

Requisito

O Inline AI Assistant requer um Chave de API OpenAI a ser configurado nas configurações.

Explique uma linha

1 Passe o mouse sobre uma linha de transcrição

Quando você passa o mouse sobre qualquer linha da transcrição, um Ícone de IA (🤖) aparece.

2 Clique no ícone AI e escolha "Explicar"

O assistente de IA abre em uma janela separada. Ele usará o contexto completo da transcrição e a linha selecionada para fornecer uma explicação detalhada.

Gere uma resposta

1 Clique no ícone AI e escolha "Gerar resposta"

O assistente de IA gerará uma sugestão de resposta com base no que foi dito na conversa. Ele pode até usar a pesquisa na web para fornecer respostas mais precisas.

Abra o painel do Assistente

1 Clique no botão Assistente na barra de controle

Você também pode abrir o painel do assistente de IA diretamente do botão do assistente (ícone 🤖) na barra de controle. Isso abre uma janela de bate-papo arrastável onde você pode digitar qualquer pergunta. O assistente tem acesso à transcrição completa como contexto.

Configurações de contexto

Forneça contexto sobre o tópico da conversa para melhorar a precisão da tradução. Grátis

1 Abrir configurações de contexto

Clique no "Mais" Botão botão (⋯) na barra de controle e clique em "Contexto".

2 Preencha os campos de contexto

Você pode fornecer:

  • Domínio / Tópico — Sobre o que se trata a conversa (por exemplo, "Conferência médica", "Standup de engenharia de software").
  • Termos-chave — Termos técnicos, nomes próprios ou jargões que podem ser difíceis de reconhecer (por exemplo, "Kubernetes, gateway de API, microsserviços").
  • Informações do alto-falante — Nomes e funções dos participantes (por exemplo, "Dr. Smith — cardiologista, apresentando").

Clique "Aplicar" para enviar este contexto ao mecanismo de tradução para a sessão atual.

3 Salvar predefinições (opcional)

Se você tiver reuniões recorrentes com tópicos semelhantes, clique em "Salvar predefinição" para salvar o contexto atual. Da próxima vez, basta carregar a predefinição em vez de digitar tudo novamente.

Você também pode Importar/Exportar predefinições como arquivos JSON para compartilhá-los ou fazer backup deles.

Preenchimento automático do YouTube

Ao usar o Live Translator em YouTube, os campos de contexto podem ser preenchido automaticamente com o título e a descrição do vídeo. Isso ajuda o mecanismo de tradução a entender melhor o tema do vídeo.

Baixar transcrição

Salve suas transcrições traduzidas como arquivos para referência posterior ou uso em outras ferramentas. Pró

1 Clique no botão Baixar

Após uma sessão de tradução (ou durante uma), clique no botão botão de download (ícone ↓) na barra de controle. A caixa de diálogo Baixar transcrição é aberta.

2 Escolha um formato

Selecione seu formato preferido:

FormatoMelhor para
TXTArquivo de texto simples com carimbos de data e hora e nomes de locutores. Fácil de ler.
JSONFormato de dados estruturados. Bom para processamento com outras ferramentas ou scripts.
SRT (Original)Arquivo de legenda com o texto no idioma original. Use em software de edição de vídeo.
SRT (traduzido)Arquivo de legenda com o texto traduzido. Use em software de edição de vídeo.
3 Baixe o arquivo

Clique no botão de download do formato escolhido. O arquivo é gerado em seu navegador e salvo na pasta de download padrão.

Privacidade

Arquivos de transcrição são gerados totalmente no seu navegador. Nenhum dado de transcrição é enviado a nenhum servidor durante o processo de download.

Verificação de atualização automática

A extensão verifica automaticamente se há atualizações sempre que você a abre.

1 Procure o selo de atualização

Quando uma nova versão estiver disponível, você verá um "Atualização disponível" crachá na barra de título do painel do tradutor.

2 Clique para atualizar

Clique no emblema para ser levado à página de instalação, onde você pode baixar a versão mais recente. Siga o mesmo etapas de instalação para atualizar.

Solução de problemas

Problemas comuns e como corrigi-los.

Mensagem "Nenhuma chave de API Soniox"

! Solução

Você ainda não inseriu sua chave de API Soniox. Vá para Insira suas chaves de API para configurá-lo.

Nenhum áudio está sendo capturado

! Solução
  • Certifique-se de clicar "Permitir" quando o navegador solicitou permissão de captura de áudio.
  • Certifique-se de ter selecionado o guia correta para compartilhar quando solicitado.
  • Tente atualizar a página e abrir o tradutor novamente.
  • Verifique se a reunião/vídeo realmente tem áudio reproduzido.

A tradução parece imprecisa

! Solução
  • Certifique-se de que O menu suspenso "A linguagem deles" está definido para o idioma correto que está sendo falado.
  • Usar Configurações de contexto para fornecer informações sobre tópicos e termos-chave.
  • Certifique-se de que a qualidade do áudio seja nítida — o ruído de fundo pode afetar a precisão.

"Narrate My Voice" — outras pessoas não conseguem ouvir a tradução

! Solução
  • Confirme que você está em um plataforma suportada (Google Meet, Zoom Web, Teams Web, WhatsApp Web).
  • Certifique-se de que uma sessão de tradução seja correndo e seu microfone ligado.
  • Verifique se você tem um Licença Pro ativado.
  • Tente atualizar a página da reunião e começar novamente.

Recursos OpenAI não funcionam

! Solução
  • Verifique seu Chave de API OpenAI foi inserido corretamente em configurações.
  • Certifique-se de que sua conta OpenAI tenha configuração de faturamento com créditos suficientes.
  • Se você não precisa dos recursos OpenAI, use Vozes do Google Tradutor para narração (gratuito, sem necessidade de chave de API).

A extensão não aparece em um site

! Solução
  • O Live Translator só funciona em plataformas suportadas. Ele não aparecerá em sites não suportados.
  • Se o site for compatível, mas a extensão não aparecer, tente atualizando a página.
  • Certifique-se de que a extensão esteja ativado na página de gerenciamento de extensões do seu navegador.
