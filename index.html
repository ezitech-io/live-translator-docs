Narration Guide

How to Use Narration (TTS)

Turn translated captions into spoken audio in just a few clicks.

  1. Open Live Translator during a meeting or video and start translation.
  2. Click the Narration button, then enter your OpenAI API key (required for TTS), get your API key HERE.
  3. Choose a voice and adjust speed/volume settings to your preference.
  4. Use Test to preview the voice, then click Start to enable narration.
  5. Optionally enable auto-sync speed to keep narration close to live translation when conversations are fast.

Installation Guide

Install from ZIP in 3 Simple Steps

No technical skills needed. Follow these exact steps and you can install Live Translator successfully in just a few minutes.

1

Download the ZIP File

Click the download button below and get the latest extension ZIP from Google Drive.

Open download folder
2

Open Extensions & Enable Developer Mode

Open Chrome and go to chrome://extensions, then turn on Developer mode.

3

Drag ZIP into Extensions Page

Drag and drop the downloaded ZIP file into chrome://extensions. After install, open Google Meet, Teams, Zoom, or YouTube. A small bird icon appears at the bottom-right corner - click it to open Live Translator and start translating.

If something does not work

  • • If drag-and-drop does not work, unzip the package and install using Load unpacked.
  • • For Load unpacked, select the unzipped extension folder (not the ZIP file).
  • • If Chrome shows an error, remove the extension and install again.
  • • Keep Developer mode enabled while using this manually installed version.
  • • Verify your Soniox API key is valid and has enough usage quota.
