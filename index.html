Narration Guide

How to Use Narration (TTS)

Turn translated captions into spoken audio in just a few clicks.

Narrate Their Voice

Hear translations from other participants read aloud to you.

  1. Open Live Translator during a meeting and start translation.
  2. Click the Narration button and choose a voice from one unified list (OpenAI • ... or Google • ...).
  3. Use the voice search box to quickly find voices by provider, language, or name.
  4. If you select an OpenAI voice, enter your OpenAI API key (get key here). Google voices do not need an API key.
  5. Adjust voice speed and voice volume, then use Test to preview your selected voice.
  6. Click Start to enable narration.
  7. Narration automatically skips your own voice — only other participants' speech is read aloud.

Narrate My Voice

Speak your words into the meeting in the other person's language.

  1. Configure two-way translation: set My language and Their language.
  2. Open Narration settings, choose an OpenAI or Google voice, then toggle on Narrate my voice.
  3. If using OpenAI voice, make sure your OpenAI API key is set. Google voices work without OpenAI key.
  4. Click Start — your speech is translated and spoken into the meeting in real time.
  5. Voice speed and voice volume settings apply to both OpenAI and Google voices.
  6. Your original voice is muted while translated audio plays, avoiding overlap.
  7. Both "Narrate their voice" and "Narrate my voice" can be used at the same time.

Installation Guide

Install from ZIP in 3 Simple Steps

No technical skills needed. Follow these exact steps and you can install Live Translator successfully in just a few minutes.

Important before updating: remove your old Live Translator version from Chrome Extensions first, then install the new ZIP version to avoid conflicts.

1

Download the ZIP File

Click the download button below and get the latest extension ZIP.

Download Extension
2

Open Extensions & Enable Developer Mode

Open Chrome and go to chrome://extensions, then turn on Developer mode.

3

Drag ZIP into Extensions Page

Drag and drop the downloaded ZIP file into chrome://extensions. After install, open any supported platform (Google Meet, Zoom, Teams, WhatsApp Web, YouTube, Facebook, TikTok, Douyin, Dailymotion, Vimeo, or Udemy). A small bird icon appears at the bottom-right corner - click it to open Live Translator and start translating.

If something does not work

  • • If drag-and-drop does not work, unzip the package and install using Load unpacked.
  • • For Load unpacked, select the unzipped extension folder (not the ZIP file).
  • • If Chrome shows an error, remove the extension and install again.
  • • Keep Developer mode enabled while using this manually installed version.
  • • Verify your Soniox API key is valid and has enough usage quota.
